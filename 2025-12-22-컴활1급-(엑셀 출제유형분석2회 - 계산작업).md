## 1-1. 수식으로 조건에 따라 값 채우기
- 조건에 따라 함수를 작성하여 채우기 작업을 통해 빈값을 조건에 맞게 채워넣는 문제이다.
- 조건의 조건은 함수 마법사에서 해당 칸에 포인터를 가져다대고 함수 버튼을 누르면 된다.
- right는 원래 출력이 문자열 함수이지만 수학함수나 계산은 문자 + 숫자면 그냥 숫자로 계산한다. 따라서 이 문제의 함수는 문제가 없다.

```vba
=IF(MOD(RIGHT(E3,2),3)=0,"A반",IF(MOD(RIGHT(E3,2),3)=1,"B반","C반"))
```

## 1-2. vlookup, match 활용하여 값 채우기
- match나 vlookup에서 바뀌면 안되는 대상 배열은 f4로 절대화 해주어야 한다.
- 또한 완벽히 똑같은 것은 0으로 type을 설정한다.
- vlookup은 탐색을 첫번째 열부터 하므로 참조표에서 찾는 값이 첫번째 임을 확인하고 블럭설정한다.
- match로 구한 행의 값보다 vlookup에서의 행의 인덱스가 1이 큰 경우가 있어서 1을 더해주는 것을 잊으면 안된다. 아니면 애초에 match에서 crosstab의 1행 1열 빈값까지 포함해도 된다.

```vba
=VLOOKUP(B6,$B$31:$E$35,MATCH(F6,$B$31:$E$31,0),0)
```

## 1-3. 배열수식 활용
- 배열 수식은 아래와 같이 두가지 유형이 있다.
- =계산함수(if(조건, 계산범위))
- =계산함수((조건)*(계산범위)) -> and는 *, or는 +, 개수를 구할때는 계산범위를 1로 주기 
- 이때 조건 x 조건 x 계산범위 해도 된다. 1 x 1 x 계산범위로 두 조건이 맞을때만 계산범위의 값이 튀어나온다.
- 배열수식은 원래 한 셀끼리 비교하던 것을 떠나서 배열로 조건을 비교한다.
- 배열 수식을 다룰때는 절대화를 신경써서 해야한다.
- 아래의 배열 수식을 입력하고 ctrl + alt + enter를 입력하면 된다.
```vba
=MAX(IF(($C$3:$C$18=C$23)*($B$3:$B$18=$B24),$I$3:$I$18))
```
