# 📌 1. 양도소득세 기본 구조

## 1-1. 총결정세액 vs 신고납부세액
> 💡 계산은 내가, 확정은 세무서가 한다

### 1) 용어 구분
- 예정신고 납부세액: 양도 후 2개월 내 내가 직접 계산·신고·납부
- 확정신고 납부세액: 다음 해 5월, 내가 계산·납부 (예정신고분 차감)
- 총결정세액: 세무서가 신고 내용 검토 후 최종 확정한 세액

### 2) 전체 흐름
- 내가 양도 → 예정신고(2개월 내) → 확정신고(다음 해 5월) → 세무서 최종 확정

# 📌 2. 양도소득세 예정신고 의무

## 2-1. 왜 예정신고가 의무인가
> 💡 국가가 양도 사실을 자동으로 알 수 없어서 조기 신고를 강제한다

### 1) 예정신고 의무 이유
- 양도소득은 발생 시점이 불규칙해 국가가 즉시 파악 불가
- 고액 거래(수억~수십억)에서 세수 확보·탈세 방지 필요
- 납세자도 목돈 손에 있을 때 내는 게 부담이 적음

### 2) 안 내면 어떻게 되나
- 예정신고 의무 불이행 시 무신고 가산세 + 납부지연 가산세 부과
- 다음 해 5월 확정신고로 대신 납부는 가능하나 가산세는 피할 수 없음

# 📌 3. 외국납부세액 이중과세 조정

## 3-1. 세액공제 vs 필요경비 산입
> 💡 같은 소득에 두 나라가 세금 매기는 걸 방지하는 장치

### 1) 세액공제 방법
- 외국에서 낸 세금을 한국 세액에서 직접 차감
- 예) 한국 세액 3000만원 - 외국 납부 2000만원 = 실납부 1000만원
- 납세자에게 유리, 이중과세 해소 효과가 가장 큼

### 2) 필요경비 산입 방법
- 외국에서 낸 세금을 비용으로 처리해 과세 기준(양도차익) 자체를 낮춤
- 세액공제 한도 초과분에 대해 활용
- 절세 효과는 세액공제보다 작음

# 📌 4. 주거용 건물 임대업 과세

## 4-1. 종합소득 과세표준 공제의 의미
> 💡 공제는 세금 매기는 기준 금액을 낮춰 세율 구간 자체를 떨어뜨린다

### 1) 공제하면 뭐가 좋아?
- 과세표준이 낮아지면 적용 세율 구간이 내려갈 수 있음 (누진세 효과)
- 예) 5100만원(24%) → 공제 후 4900만원(15%)으로 세율 자체가 낮아짐
- 공제는 권리이므로 신청 안 하면 그냥 더 내는 것

### 2) 1주택자 비과세 원칙
- 기준시가 12억 이하 1주택자 월세 수입은 비과세
- 비과세 = 세금 안 내도 된다는 것, 임대 자체를 못 하는 게 아님
- 본격 임대업(공제 적용 구조)은 2주택 이상부터 해당

---

# 📌 5. 1주택자 월세 비과세 이유

## 5-1. 정책적 배려의 배경
> 💡 세금 매기면 임대 공급이 줄어 결국 세입자가 피해를 본다

### 1) 비과세 이유 3가지
- 1주택자는 투기 목적보다 생활 수단에 가깝고, 본인도 세입자인 경우 많음
- 과세 시 임대 매물 감소 → 전월세 가격 상승 → 서민 주거 불안정 초래
- 소득 규모가 작아 세수 대비 행정 비용이 더 커 징수 효율이 낮음

### 2) 12억 초과는 왜 과세?
- 고가 주택 보유자는 생활 수단보다 재산 증식 수단으로 판단
- 다주택자(과세)와 1주택자(보호)를 구분하는 형평성 기준

# 📌 6. 부동산 세금 납부 방법

## 6-1. 세금별 납부 시기와 방법
> 💡 고지서 오는 세금은 받고 내면 되고, 나머지는 내가 홈택스·위택스에서 직접 신고

### 1) 내가 직접 신고·납부 (신고납부)
- 취득세: 취득 후 60일 내, 위택스(wetax.go.kr)에서 신고·납부
- 등록면허세: 등기 전 위택스 납부 (실무상 법무사 대행)
- 양도소득세: 양도 후 2개월 내, 홈택스(hometax.go.kr)에서 신고·납부

### 2) 고지서 받고 납부 (부과징수)
- 재산세: 매년 7월(건물), 9월(토지) 고지서 자동 발송 → 위택스·ATM·편의점 납부
- 종합부동산세: 매년 12월 1~15일, 고지서 수령 후 납부

# 📌 7. 종합소득세 vs 지방소득세

## 7-1. 두 세금의 차이
> 💡 종합소득세는 나라에, 지방소득세는 내가 사는 지역에 내는 세금이다

### 1) 개념 차이
- 종합소득세: 국가(국세청)에 납부, 세율 6~45% 누진 적용
- 지방소득세: 지자체(시·군·구)에 납부, 종합소득세의 10%
- 예) 종합소득세 100만원이면 지방소득세 10만원 추가 → 총 110만원 납부

### 2) 신고 방법
- 매년 5월 동시에 신고
- 종합소득세: 홈택스 / 지방소득세: 위택스에서 별도 신고 필요
- 지방소득세는 지역 인프라(도로·복지·소방 등) 재원으로 활용됨
