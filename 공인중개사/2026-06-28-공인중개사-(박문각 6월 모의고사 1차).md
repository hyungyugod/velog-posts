# 📌 1. 부동산학개론

## 1-1. 시장실패와 정부의 개입
> 💡 외부효과·소득재분배는 '시장에 맡기면 안 되는 이유'를 설명한다

### 1) 부(-)의 외부효과와 과다생산
- 부의 외부효과는 사회적 비용을 고려하지 않으므로, 시장에 맡길 경우 최적 생산량보다 과다생산되는 문제가 발생한다.
```
-> 사회적 비용은 사적비용과 외부비용의 합인데 외부비용이 크고 사적비용이 작은 공장 같은 경우 사적비용만 생각해서 아무데나 짓게 되는데 그러면 사회적으로는 덜 지어졌어야하는 공장이 여기저기 많이 생기게 된다. -> 이는 시장실패의 예이고 이로써 용도지역 정책이 정당화된다.
```

### 2) 공공임대주택과 소득재분배
- 공공임대주택 공급은 실질적 주거비 보조 효과를 통해 고소득층 → 저소득층으로의 소득재분배 기능을 수행한다.
```
-> 고소득자에게 걷은 누진세로 저소득자의 주거비 보조를 지원하므로 소득재분배이다.
```

## 1-2. 토지·조세 정책
> 💡 비축토지·개발부담금·개별공시지가는 '누가·언제·무엇을' 구분이 핵심

### 1) 비축토지(공공토지비축)
- 비축토지는 도로·철도 같은 공공개발용과 시장 안정을 위한 수급조절용으로 나누어 운영되며, 더 잘 확보하기 위한 여러 정책적 유인이 있다.
- 토지비축업무는 한국토지주택공사(LH)가 담당하며, 지자체가 독립적으로 관리하는 것이 아니다.

### 2) 개발부담금과 공공토지비축의 구분
- 개발부담금: 개발이익을 환수하기 위해 시장·군수·구청장이 부과·징수하는 금액.
- 토지를 사전에 비축하여 장래 공익사업의 원활한 시행과 토지시장 안정에 기여하는 것은, 정확히는 '공공토지비축제도'라고 답하는 것이 더 옳다.

### 3) 개별공시지가 분할·합병
- 공시기준일 이후에 분할·합병이 발생하면 7월 1일 또는 1월 1일을 기준으로 개별공시지가를 다시 결정·공시한다.

## 1-3. 부동산금융 — 원금균등상환
> 💡 원금균등상환의 '해당기 이자'는 남은 기간으로 거꾸로 센다

### 1) 7기 이자 상환액 계산
- 원금균등상환에서 7기 원리금 상환액을 구할 때, 금리가 4%이고 대출기간이 20년이면 14 × 25(원금) × 0.04로 이자금액을 계산할 수 있다.
- 이자가 매번 (원금 × 이자율)씩 감소하므로 '남은 기간 × 원금 × 이자율'로 해당기 이자 상환액을 구할 수 있다. 이때 남은 기간은 현재가 7기면 7기까지 한 게 아니라 6기까지 상환한 상태이므로, 전체 - 7 + 1로 남은 기간을 구하여야 한다.

## 1-4. 부동산 경제론 — 수요·공급·탄력성
> 💡 직선 수요곡선의 탄력성, 자중손실, 균형량 변화는 모두 '그래프 확인'이 빠르다

### 1) 직선 수요곡선의 탄력성은 점마다 다르다
- 수요곡선이 직선인 경우, 어느 지점에서 측정하는가에 따라 수요의 가격탄력성은 일정하지 않고 달라진다.
```
E = (수량 변화율) / (가격 변화율)
  = (ΔQ/Q) / (ΔP/P)
  = (ΔQ/ΔP) × (P/Q)
       ↑조각   ↑조각2
       기울기 역수  점의 위치     

-> 조각 2는 점의 값에 관련된 부분이므로 같은 그래프에서 단위탄력적인 그래프 상 중점에서 위쪽은 가격이 더 크므로 탄력성을 키우고 아래쪽 점은 가격이 더 작으므로 탄력성을 낮춘다.
-> 이는 이렇게 이해할 수 있는데 직선 기울기는 같으므로 p의 가격이 변하면 q의 공급이 변하는 변하는 것은 어디나 똑같다. 위쪽점에서 가격을 조금 옮기면 작은 공급량의 상태에서 +q가 되는 것이고 아래쪽 점에서 가격을 p만큼 옮기면 같은 +q가 되지만 이미 거기는 공급량이 큰 시점이라 가지고 있던 것이 많으므로 +q의 가치가 전자에 비해 작아 비탄력적으로 보이는 것이다. 

-> 만약 점을 고정하고 그래프의 기울기를 비교하면 p/q가 고정인 꼴이므로 예전에 외웠던 것처럼 비탄력적일수록(가파를 수록) 기울기가 크고 이런 공식이 나온다.
```

### 2) 시장 구분과 수요량 변화 요인
- 시장의 수요량 변화를 일으키는 요인을 찾을 때는 해당 시장이 어떤 물건의 시장인지부터 파악해야 한다. 예를 들어 아파트 시장의 수요량 상승을 일으키는 요인에 오피스텔 가격 하락 같은 것은 부적합하다.

### 3) 세금과 경제적 순손실(자중손실)
- 세금부과에 따른 경제적 순손실은 공급이 탄력적일수록 더 커진다.
- 경제적 순손실(자중손실, deadweight loss)은 세금 때문에 거래가 줄어들어 사라진 사회적 이득을 의미한다.
```
-> 자중(스스로의 무게 -> 경제 구조에서 오는 그냥 짊어지는 부담)손실은 세금으로 거래자체가 사라지면서 사회적으로 매수인도 못사고 매도인도 못팔고 정부도 그 거래에서는 세금 못걷어 이득이 통째로 순수하게 사라지는 것이다. 아무일도 벌이지 않으면 아무 것도 얻어지지 않는다. 
-> 파이가 작아진다!

공급이 탄력적(완만) → 세금이 붙어 가격이 오르면 공급자가 수량을 크게 줄임 → 거래량 급감 → 순손실 큼
공급이 비탄력적(가파름) → 세금 붙어도 수량 거의 안 줄임 → 거래량 유지 → 순손실 작음
```

### 4) 수요 증가와 공급 탄력성
- 부동산 수요가 증가하면, 부동산 공급이 비탄력적일수록 시장균형량이 더 적게 증가한다.
```
-> 이전 수요곡선과의 균형점 똑같이 잡고 공급곡선 그래프 그려보면 확인할 수 있다.
-> 말 그대로 이해하면 수요가 증가하면 공급이 같이 증가해야하는데 비탄력적이니까 그 공급량은 더 적게 증가할 수밖에 없음 -> 그래프 확인이 빠르다.
```

## 1-5. 부동산 투자분석 — 소득승수
> 💡 'x승수 = 총투자액이 분모 소득의 몇 배인가'로 통일해서 본다

### 1) 순소득승수 vs 유효총소득승수
- 순소득승수는 지분투자액이 아닌 총투자금액을 순영업소득으로 나눈 금액이다.
- 유효총소득승수는 총투자액을 유효총소득으로 나누어서 정한다.
```
-> x승수이면 총투자액이 x의 몇배이냐 몇승을 한 수이냐를 묻는 것

가능총소득승수
= 총투자액 ÷ PGI(가능총소득)

유효총소득승수
= 총투자액 ÷ EGI(유효총소득)

순소득승수
= 총투자액 ÷ NOI(순소득)

세전현금흐름승수
= 지분투자액 ÷ BTCF(세전현금흐름)
```

# 📌 2. 민법 및 민사특별법

## 2-1. 조건과 기한
> 💡 단독행위 조건은 '원칙 불가'가 핵심, 기한은 소급효 없음

### 1) 불능 해제조건
- 조건이 법률행위 당시에 이미 불능인데 해제조건으로 법률행위를 한다면, 조건이 의미가 없으므로 그냥 조건 없는 법률행위가 된다.
```
대학가면 내가 준돈 다 돌려줘 -> 이미 대학졸업함 -> 그냥 돈 다 돌려줘
```

### 2) 단독행위와 조건
- 단독행위에는 원칙적으로 조건을 붙이지 못하나 예외가 있다. 원칙은 안 된다는 것이 중요하다.

### 3) 기한도래의 효력
- 기한도래의 효력은 도래 전으로 소급할 수 없고, 항상 장래를 향해 효력이 발생한다.

## 2-2. 권리변동
> 💡 상속은 원시취득이 아니다

### 1) 상속의 성질
- 상속은 포괄적 승계취득이지 원시취득이 아니다.

## 2-3. 의사표시 — 착오
> 💡 담보책임과 착오는 별개, 중과실 입증은 상대방이

### 1) 담보책임과 착오의 관계
- 담보책임과 중요부분 착오는 다른 이야기이므로 서로 영향이 없다.

### 2) 중대한 과실의 입증책임
- 착오자에게 중대한 과실이 있는지 여부는 취소를 저지하려는 상대방이 입증해야 한다. → 착오가 중대한 과실에 의한 거라면 착각이어도 주의를 안 한 것이므로 취소할 수 없다.

## 2-4. 점유·물권
> 💡 비용상환·부합·지역권 부종성

### 1) 무효 매매와 비용상환청구의 상대방
- 무효인 매매계약의 매수인이 점유목적물에 필요비 등을 지출한 후 매도인이 그 목적물을 제3자에게 양도한 경우, 점유자인 매수인은 양수인에게 비용상환을 청구할 수 있다.
```
-> 필요비의 이득을 보는자가 현재 양수인이므로
```

### 2) 지역권의 부종성과 분리처분
- 요역지소유권과 지역권이 함께 이전되지 않도록 하는 약정은 유효하며 이를 등기할 수 있다.
```
법적으로 따로 정할 수 있도록 해둠. 지역권을 소멸시켜도 됨 -> 보통 승역지 주인이 부담스러워서 이제 없애고 싶을때
근데 분리처분은 여전히 할 수 없다.
```

### 3) 증축부분의 부합과 경매
- 경매절차의 매수인이 증축부분의 소유권을 취득하기 위해서는, 증축부분이 기존건물에 대한 경매절차에서 목적물로 평가되지 않았어도 된다.
```
-> 기존 건물의 근저당권은 부합되는 순간 부합된 건물에도 미친다.
-> 위의 이유때문에 저당권 설정되어있었을때 건물 함부로 분할 못하고 못 붙이고 그런 문제가 생기는 것이다.
```

### 4) 농작물의 예외
- 적법한 권원 없이 타인의 토지에 경작한 성숙한 배추의 소유권은 경작자에게 속한다.
```
원칙(부합)대로라면 토지소유자 것이어야 하는데, 농작물만은 예외
```

## 2-5. 계약 일반
> 💡 분양광고는 청약의 유인, 계약금계약은 요물·종된 계약

### 1) 분양광고의 성질
- 아파트 분양광고는 청약의 성질이 아니라 청약의 유인의 성질을 갖는다.

### 2) 매도인의 담보책임 — 선의/악의 정리
- 수량지정매매에서 수량부족이나 일부멸실의 경우에 매수인이 악의이면 담보책임이 발생하지 않는다. → 그냥 부족한 대로 알고 샀네.

| 담보책임 (조문) | 선의 매수인 | 악의 매수인 |
|---|---|---|
| 전부 타인권리 (570) | 해제·손배 | 해제는 가능 (손배는 X) |
| 일부 타인권리 (572) | 감액·해제·손배 | 감액은 가능 |
| 수량부족·일부멸실 (574) | 감액·해제·손배 | 담보책임 X ← 이 명제 |
| 용익권 제한 (575) | 해제·손배 | 담보책임 X |
| 저당권·전세권 행사 (576) | 해제·상환·손배 | 선의·악의 불문 보호 |
| 물건의 하자 (580) | 해제·손배 (무과실 필요) | 담보책임 X |

### 3) 계약금계약
- 계약금 계약은 계약금 전부를 지급해야 성립한다. 일부만 지급받은 경우 배액상환하고 해제하지 못한다.
- 계약금계약은 종된 계약이다. 그러나 주된 계약과 동시에 체결해야 하는 것은 아니다.
```
10/1 — 주된 매매계약: 낙성계약(諾成契約) 이라 "사고팔자"는 합의(서면·말) 만으로 성립합니다. 돈이 안 오가도 이미 매매계약은 성립한 거예요. -> 해제하려면 합의해제
10/3 — 계약금계약: 요물계약(要物契約) 이라 계약금을 실제로 건네야 성립합니다. 10/1엔 돈을 안 줬으니 아직 계약금계약은 없는 상태고, 10/3 송금된 순간 비로소 성립합니다. -> 해제권도 이때 생김

-> 10/1에 해제하려면 일반적으로 합의해제를 하거나 상대방이 채무불이행을 했을때 법정해제정도만 할 수 있다.

-> 보통 계약금은 위약금의 형태로 많이 쓰인다. 다만 이는 추가적인 약정이 필요하다.
```

### 4) 예약완결권
- 예약완결권의 행사기간을 약정으로 정하는 경우에는 최장기 제한이 없고, 약정이 없으면 예약 성립 후 10년 이내에 행사해야 한다. → 예약을 완결 → 계약효과 실행.
- 예약완결권은 제척기간이므로 법원의 직권조사사항이다. 따라서 당사자의 주장 여부와 관계없이 법원이 이를 고려하여 판단해야 한다.
- 제척기간(除斥期間): 除斥 = "제거하여 물리침". 권리의 존속기간으로, 지나면 권리가 자동 소멸(소멸시효와 달리 중단·정지 없음). → 그냥 명확하게 몇 년 지나면 딱 끝나므로, 굳이 주장하지 않아도 법원이 명확히 판단할 수 있다.

### 5) 해제 — 정기행위·최고
- 정기행위의 경우에는 이행의 최고 없이 해제할 수 있다. → 정해진 기간에 꼭 해야 하는 계약(결혼식 등).
- 최고할 때는 꼭 언제까지 답을 주라고 할 필요 없고, 상당한 기간이 지나면 그냥 해제권이 생긴다.

## 2-6. 주택·상가 임대차
> 💡 환산보증금 기준을 넘으면 적용 규정이 제한된다

### 1) 환산보증금 감각
- 환산보증금을 구할 때 월차임 100만 원이면 (×100 하여) 억대인 것을 헷갈리면 안 된다.

### 2) 환산보증금 초과 상가임대차
- 환산보증금 이상의 상가임대차는 임차권등기명령을 신청할 수 없다. (표준계약서 3권대요만 가능)
- 표준계약서 3권대요: 상가임대차 표준계약서 권장, 3기 차임연체와 계약해지, 권리금보호규정, 대항력, 계약갱신요구권.

### 3) 보증금반환채권 가압류와 소유자 변경
- 일반적으로 보증금 반환채권이 가압류되고 주택 소유자가 바뀌면, 가압류 효력은 승계된 양수자에게만 주장할 수 있다.
```
일반적으로라고 붙어있으므로 승계를 거부하는 경우는 배제하고 생각한다. 이는 특수한 경우이므로 일반적인 관계에서는 위 명제가 맞다.
```

## 2-7. 집합건물
> 💡 공용부분 변경 = 구분소유자 2/3 + 의결권 2/3

### 1) 공용부분의 변경 정족수
- 집합건물 관리에서 공용부분의 변경은 구분소유자 2/3 이상 및 의결권의 3분의 2 이상의 결의가 있어야 한다.

# 💡 확인 필요

### 1) [민법] 불능 해제조건 — 본문 명제 ↔ 예시 결론 불일치
- 본문 명제("이미 **불능**인 조건을 **해제조건**으로 붙이면 → 조건 없는 법률행위")는 「민법」 제151조 제3항과 일치하며 **맞습니다**.
- 다만 코드블록 예시("대학 가면 돈 돌려줘 → 이미 대학 졸업 → 그냥 돈 다 돌려줘")의 **결론(돈 반환)**은, 오히려 제151조 제2항(이미 성취된 **기성** 해제조건 → **무효** → 부당이득 반환)에 들어맞는 결과로 읽힙니다.
  - 불능 해제조건(제3항) → 조건 없는 증여 = 증여 **확정** → 돈은 **돌려주지 않음**
  - 기성 해제조건(제2항) → 증여 **무효** → 돈을 **돌려줌**
- 즉 명제는 '불능(제3항)'을 말하는데 예시 결론은 '기성(제2항)' 쪽으로 보입니다. 예시가 어떤 조항을 의도한 것인지 한 번 점검해 보시길 권합니다. (제 해석이 틀렸을 수 있으니 조문으로 최종 확인 부탁드립니다.)

### 2) [참고] 집합건물 공용부분 변경 정족수 — 개정 이력
- 현재 「집합건물법」 제15조 기준 구분소유자 2/3 + 의결권 2/3가 맞습니다(과거 3/4 → 2/3로 개정). 개정 이력이 있는 조문이라 시험 직전 현행 조문 한 번 더 확인 권장.

### 3) [표현] 동일수급권 범위 — 정밀 표현 참고
- '인근지역을 포함한다'는 핵심은 맞습니다. 다만 엄밀히는 **동일수급권 ⊇ (인근지역 ∪ 유사지역 ∪ 주변용지)** 구조라, "유사지역 전부"라는 표현은 동일수급권과 유사지역을 동일시할 여지가 있어 시험 지문에서 한 번 더 확인해 두면 안전합니다. (원문 표현은 그대로 보존했습니다.)
