## 부동산 세법 등록면허세의 과세표준과 세율 ~ 재산세의 비과세
- 등록면허세의 과세표준은 원칙적으로 등록당시 가액으로 한다. 이는 신고가액이다. 하지만 신고가 없거나 신고가액이 시가표준액보다도 작으면 (시가인정액x) 그 금액을 시가표준액으로 한다. -> 등록면허세는 공부에 어떤 권리를 올렸는지가 얼마에 취득했냐보다 더 중요하므로 시가인정액 계산하고 그럴 실익이 없음

- 등록면허세 과세표준과 표준세율
| 구분            | 과세표준  | 표준세율            |
| ------------- | ----- | --------------- |
| **소유권 보존등기**  | 부동산가액 | 1천분의 8          |
| **이전등기 (유상)** 주택 | 부동산가액 | 취득세율 × 100분의 50 |
| **이전등기 (유상)** 주택 외   |       | 1천분의 20         |
| **이전등기 (무상)** 상속            |       | 1천분의 8          |
| **이전등기 (무상)** 상속 외          |       | 1천분의 15         |

- 말소등기 지목변경 등 외 아래 권리의 세율은 모두 1천분의 2 
| 권리           | 과세표준   |
| ------------ | ------ |
| 가등기          | 부동산가액, 채권금액  |
| 저당권, 경매신청, 가압류, 가처분| 채권금액   |
| 전세권          | 전세금액   |
| 임차권          | 월임대차금액 |
| 지역권          | 요역지가액  |
| 지상권          | 부동산가액  |
| 채권담보         | 채권금액   |
| 임차권 (중복 표시됨) | 월임대차금액 |
| 말소등기, 지목변경 등 | 건당 6000 |

- 갑이 을로부터 부동산을 취득 후 재산세 과세기준일까지 등기하지 않았다면 재산세와 관련해서 을은 부동산 소재지 관할 지방자치단체의 장에게 소유권 변동사실을 과세기준일로 15일 이내에 신고할 의무가 있다.
- 수탁자 명의로 등기등록된 신탁재산의 수탁자는 과세기준일부터 15일 이내에 지자체 장에 그 사실을 알 수 있는 증거자료를 가지고 신고해야한다.
- 건축물에서 허가등이나 사용승인을 받지 않으면 주거용 면적이 전체 면적이 50%이상이어도 주택으로 보지 않고 건축물과 토지로 각각 보고 과세하고 토지는 투기목적으로 이해되므로 종합합산과세대상이다.
- 토지의 과세대상에서 저율분리과세대상은 해당 토지가 생산목적이라고 생각하고 고율분리과세대상은 해당 토지가 사치성이라고 생각하고 종합합산과세대상은 투기목적으로 생각하고 별도합산과세대상은 사업목적으로 생각하고 과세대상을 정한 것이다.
- 공장용지는 군 / 읍면 / 산업단지 / 공업지역인 경우 생산(분리과세)에 관여하는 것으로 보고 그 외는 사업용지(별도 합산 과세)로 본다.
- 농지는 군 / 읍면 / 도시밖 / 개발제한구역, 녹지지역의 경우 생산에 관여하는 것으로 본다.
- 통제보호구역에 있어도 전, 답, 과수원 및 대지는 과세인 이유는 전, 답, 과수원은 수익이 발생하고 대지는 비싼 토지이기 때문에 통제보호구역에 있어도 과세한다.
