# 📌 1. 공인중개사법

## 1-1. 정책심의위원회와 공인중개사협회
> 💡 둘 다 국가의 감독을 받는 기구 — 위원은 국장이 임명·위촉, 협회는 총회 의결사항을 국장에게 보고

### 1) 공인중개사 정책심의위원회
- 공인중개사 제도 전반의 주요 정책을 심층 검토·의결하는 국토교통부 소속 합의체 기구
- 위원은 국토교통부장관이 임명하거나 위촉
- 위원장 1명을 포함하여 7명 이상 11명 이내로 구성

### 2) 공인중개사협회
- 국가가 법으로 존재를 인정하고 공적 역할을 부여한 법정단체 → 그만큼 국가 감독을 받음
- 법령에 맞게 잘 운영되는지 보기 위해, 총회 의결사항은 국토교통부장관에게 지체 없이 보고
- 단, 보고 의무일 뿐 승인까지 받을 사항은 아님

## 1-2. 중개대상물과 중개의 본질
> 💡 소유자가 없으면 거래 당사자가 없고 알선 자체가 불가능 → 무주의 부동산은 중개대상물 X

### 1) 무주(無主)의 부동산
- 무주의 부동산은 중개대상물이 될 수 없음
```
중개의 본질 = "거래 당사자 간 알선"

거래가 성립하려면?
  → 거래 당사자(파는 사람)가 존재해야 함
  → 파는 사람이 있으려면 소유권자가 있어야 함

무주의 부동산은?
  → 소유자 없음 = 거래 당사자 없음
  → 거래 당사자가 없으면 "알선" 자체가 불가능
  → 따라서 중개의 대상이 될 수 없음
```

## 1-3. 추정과 간주
> 💡 추정은 반증으로 뒤집을 수 있고, 간주는 못 뒤집음 — 중개보조원의 업무상 행위는 개공의 행위로 "본다(간주)"

### 1) 추정 vs 간주는 개념상 완전히 다름
- 중개보조원의 업무상 행위는 그를 고용한 개업공인중개사의 행위로 "추정"하는 것이 아니라 "본다"
```
추정한다 (推定)
  → "일단 A로 본다"
  → 하지만 반대 증거 제출하면 뒤집을 수 있음
  → 반증(反證) 허용 O

간주한다 / 본다 (看做)
  → "A로 확정한다"
  → 반대 증거 있어도 뒤집을 수 없음
  → 반증(反證) 허용 X
  → 법이 강제로 그렇게 처리해버림
```

## 1-4. 교육 제도 (실무·연수·직무·예방)
> 💡 실시기관은 모두 시·도지사, 비용 지원은 예방교육에만 (수혜자가 소비자·거래당사자이므로)

### 1) 실무교육 대상
- 대표자를 포함한 임원 또는 사원 전원이 실무교육을 받아야 함
- "대표자를 제외한다"는 설명은 틀림

### 2) 교육의 종류와 시간
```
✅ 실무교육 = 45시간, 등록 전
✅ 연수교육 = 12시간 이상, 2년마다
✅ 직무교육 = 3~4시간, 업무 개시 전
✅ 셋 다 실시기관 = 시·도지사
✅ 연수교육 미이수 → 과태료 (업무정지 아님)
✅ 실무교육 이수 후 개업 안 하면 → 교육 효력 유지
   (기간 제한 없음)
```

### 3) 부동산거래사고예방교육
- 시·도지사가 비용을 지원할 수 있는 것은 연수교육이 아니라 부동산거래사고예방교육
- 예방교육의 수혜자가 일반 소비자와 거래 당사자이기 때문

## 1-5. 사무소 명칭과 광고
> 💡 옥외광고물엔 성명만 표시하면 됨(책임자 특정 목적) — 명칭엔 "공인중개사사무소" 또는 "부동산중개" 문자 사용

### 1) 옥외광고물
- 개업공인중개사는 옥외광고물에 성명만 표시하면 되고, 중개사무소의 명칭까지 표시할 의무는 없음
- 책임자만 특정되면 된다는 취지

### 2) 명칭 사용 문자
- 명칭에 "공인중개사사무소" 또는 "부동산중개" 문자를 사용해야 함
- 부칙상 개업공인중개사는 "부동산중개" 문자만 사용 가능

## 1-6. 분사무소 폐업과 보증
> 💡 분사무소 폐업엔 설치신고서를 첨부, 보증은 만료 전에 다시 설정 (공백기 금지)

### 1) 분사무소 폐업신고
- 분사무소 폐업신고 시에는 중개사무소 등록증이 아닌 분사무소 설치신고서를 첨부
```
본사무소 개설 → 중개사무소 등록증 발급
                      ↓
분사무소 설치 → 분사무소 설치신고서 발급
                      ↓
폐업 시 각각 해당 서류를 반납
  - 본사무소 폐업 → 중개사무소 등록증 첨부(반납)
  - 분사무소 폐업 → 분사무소 설치신고서 첨부(반납)
```

### 2) 보증의 재설정
- 보증기간 만료로 다시 보증을 신청할 때는 만료 전에 다시 보증을 설정하고 등록관청에 신고
- 보증의 공백기가 존재하면 안 되므로

## 1-7. 관리비 설명의무

### 1) 관리비 설명의무

- 이때 인터넷·TV값 같은 것들을 비용의 항목, 즉 비목(費目)이라 부름

### 2) 1-1 해당: 양양이사오 판매명수 의비아아

| 코드 | 사유 |
|---|---|
| **양** | 자격증 양도 |
| **양** | 등록증 양도 |
| **이** | 이중 개설등록, 이중 소속 |

| **의** | 의뢰받지 않은 정보 공개, 다르게 공개 |
| **비** | 업무상 비밀 누설 |
| **아아** | 공인중개사가 아닌 자가 도용 |

- "아아"에는 개업공인중개사가 아닌 자가 그 사무소에 "공인중개사사무소"라는 명칭을 사용한 경우도 포함

- 따라서 진정명의회복에 기한 등기이전청구를 할 수 있고, 제3자도 보호할 수 있음

# 📌 2. 부동산공법

## 2-1. 용도구역의 지정권자
> 💡 개발제한구역은 광역·국가전략 사안이라 국토교통부장관만 지정 / 지구단위계획구역·도시자연공원구역은 목적이 달라 통합 불가

### 1) 개발제한구역
- 개발제한구역은 국토교통부장관만 지정할 수 있음
```
왜 시·도지사한테 못 맡기나?

개발제한구역의 특성:
① 광역적 효과
② 경계가 행정구역을 넘음
③ 국가 전체 토지 이용 전략과 연결

→ 한 시·도의 문제가 아님
→ 국가 차원에서 결정해야 함
```

### 2) 지구단위계획구역 vs 도시자연공원구역
```
지구단위계획구역 (시장군수구청장이 세밀하게 지정):
"개발은 하되, 이렇게 해라" → 개발의 질 관리

도시자연공원구역 (시도지사):
"여기는 손대지 마라" → 보전의 양 확보

둘은 목적 자체가 달라서 통합 불가
```

## 2-2. 도시·군관리계획의 실효
> 💡 주민 제안으로 결정된 지구단위계획은 결정 고시일부터 5년 내 미착수 시 실효 (직권 입안은 실효 없음)

### 1) 주민 입안제안에 따른 지구단위계획
- 주민의 입안제안에 따른 지구단위계획에 관한 도시·군관리계획은 결정 고시일부터 5년 이내에 착수해야 하며, 아니면 실효
```
도시·군관리계획:
→ 언제든 수시 입안 가능 (행정청 직권)
→ 주민도 제안 가능

주민이 제안해서 관리계획 결정됐는데:
→ 5년 안에 착수 안 하면 실효 
→ "네가 요청했으니 네가 책임져라"

행정청이 직권으로 입안한 것은:
→ 실효 없이 계획 유지
→ 공공 목적이니까

도시·군기본계획 재검토 주기 = 5년
도시·군관리계획 재검토 주기 = 5년
지구단위계획 실효 기간 = 5년
```

## 2-3. 주거지역과 시가화조정구역
> 💡 전용주거 = 양호한 환경, 일반주거 = 편리한 환경 / 시가화조정구역은 유보기간 만료 다음날 효력 상실

### 1) 주거지역 구분
- 양호한 주거환경 → 전용주거지역
- 편리한 주거환경 → 일반주거지역

### 2) 시가화조정구역
- 시가화조정구역의 지정에 관한 도시·군관리계획 결정은 시가화 유보기간이 끝난 다음 날부터 효력을 잃음
```
「국토계획법」상 유보기간:
= 5년 이상 20년 이내

"이 기간 동안은 개발 안 한다"고
법으로 정해둔 기간

예)
2024년 시가화조정구역 지정
유보기간 10년으로 설정
→ 2034년까지 개발 금지
→ 2034년 다음날부터 효력 상실
→ 그때부터 개발 가능

법령 전반에서:
"~기간 만료 다음날부터"
"~일이 경과한 날부터"

전부 같은 민법 기간 계산 원칙 적용
```

## 2-4. 도시·군계획시설사업의 시행
> 💡 행정청 사업시행자는 허가 없이 7일 전 통지로 타인 토지 출입 가능 / 시설은 둘 이상으로 분할 시행 금지

### 1) 타인 토지 출입
- 사업시행자가 행정청인 경우 허가를 받지 않고 7일 전에 일시와 장소를 알리면 타인의 토지에 출입할 수 있음
- 애초에 행정청은 누군가에게 허가를 받지 않기 때문

### 2) 의견청취 주체
- 시·도지사의 의견을 들어 사업을 진행하는 경우, 그 의견을 들을 수 있는 자는 국토교통부장관뿐

### 3) 분할 시행 금지
- 사업시행자는 도시·군계획시설사업 대상시설을 둘 이상으로 분할하여 시행해서는 안 됨
- 도로를 절반만 지을 수는 없으므로 (+ 규제 회피 방지)

## 2-5. 도시개발법 — 환지와 청산금
> 💡 청산금 징수 위탁 수수료는 100분의 4 / 환지 제외 토지는 결정과 교부가 동시

### 1) 청산금 징수 위탁 수수료

## 2-7. 도시정비법 — 조합의 구조
> 💡 조합은 회사와 같은 구조 / 임원은 견제받는 자라 대의원 불가, 조합장만 의장으로서 예외

### 1) 조합임원과 대의원
- 조합장이 아닌 조합임원은 대의원이 될 수 없음
```
총회
  ↓ 권한 위임
대의원회 ← 임원 견제·감시
  ↓ 
조합임원 (집행)

→ 견제하는 자 = 대의원
→ 견제받는 자 = 임원
→ 같은 사람이면 견제 불가

조합장만 예외:
→ 대의원회 의장으로서
   행정적 역할 필요
→ 의결권은 없고 진행만 함
```

### 2) 조합 = 회사 구조
```
조합원 = 주주
→ 사업의 주인
→ 최종 의사결정권자

총회 = 주주총회
→ 조합원 전원이 모이는 최고 의결기관
→ 중요한 건 여기서 결정

대의원회 = 이사회
→ 총회 열기 어려울 때
→ 총회 권한 일부 대행

조합임원 = 경영진
→ 실제 사업 집행
→ 조합장(대표이사) + 이사 + 감사
```

### 3) 벽·기둥 구획 없는 건축물 (예: 주유소)
- 벽·기둥의 구획이 없는 건축물(주유소 연상, 기둥의 "1" 연상) → 지붕 끝부분으로부터 수평거리 1미터를 후퇴한 선으로 둘러싸인 수평투영면적을 바닥면적으로 함

### 4) 노대(발코니)의 바닥면적
- 건축물 노대 등의 바닥은 난간 등의 설치 여부와 관계없이, 노대 등이 접한 가장 긴 외벽에 접한 길이에 1.5미터를 곱한 값을 뺀 면적을 바닥면적에 산입
```
[건물 외벽]
    |
    |← 1.5m →|←── 나머지 ──→|
    |         |               |
    | 산입 제외|    산입 포함   |
    |         |               |
    [외벽]  [1.5m선]      [끝부분]

→ 외벽에서 1.5m까지는 제외
→ 1.5m 넘어가는 부분만 산입
```

## 3-1. 지적공부의 등록사항
> 💡 "공대x" — 공유지연명부·대지권등록부엔 도면번호 X / 고유번호는 도면·지상경계점등록부 빼고 전부 O

- 그럼에도 토지의 고유번호는 넣어야 함 — 고유번호는 도면과 지상경계점등록부를 빼고 전부 들어가는 사항이기 때문
```
지상경계점등록부:
→ 현장 말뚝·표지 중심
→ 소재·지번으로 토지 특정 가능
→ 전산 연계 필요성 낮음

경계점좌표등록부:
→ 수치 좌표 중심
→ 정밀 측량 데이터
→ 전산 시스템으로 관리

   전산 연계 필수
→ 고유번호로 시스템 연결
→ 고유번호 필수
```

### 2) 소유권 지분과 대지권 비율
- 공유지연명부와 대지권등록부 모두 소유권의 지분은 공통으로 등록하는 사항
- 대지권의 비율은 대지권등록부에만 기입
```
공유지연명부:
갑 1/3, 을 1/3, 병 1/3
→ 이 토지에서 각자 얼마만큼
   소유하고 있는지

대지권등록부:
101호 100/10000, 102호 95/10000...
→ 이 토지에서 각 세대가
   얼마만큼 소유하고 있는지

→ 둘 다 결국
   "이 토지를 얼마나 갖고 있는가"
   = 소유권 지분
→ 공통으로 기록
```

## 3-2. 등기할 수 없는 사항
> 💡 말소는 등기하는 권리가 아니므로 이를 보존하기 위한 가등기는 불가 (제29조 제2호)

- 소유권이전등기말소청구권을 보존하기 위한 가등기는, 말소 자체가 등기하는 권리가 아니므로 제29조 제2호(등기할 수 없는 사항)에 해당
- 공유자 한 사람만의 본등기 가처분등기 신청도 마찬가지

### 2) 점유물분주농지하천상속 (암기코드)
- "점유물분주농지하천상속등기"가 의미하는 것: 점유권, 유치권, 물권적 청구권, 분묘기지권, 주위토지통행권, 농지에 대한 전세권, 하천에 대한 용익권, 자기 지분만의 상속등기, 가등기에 의한 본등기 금지 가처분등기
- 하천에 대한 용익권 설정등기는 등기할 수 없지만 저당권 설정은 가능

## 3-3. 지역권 등기
> 💡 승역지에 지역권 설정등기를 하면, 등기관이 직권으로 요역지 을구에 기록

### 1) 지역권 설정등기의 직권 기록
- 승역지의 등기기록에 지역권 설정등기를 하면, 등기관은 직권으로 요역지 등기기록 을구에 지역권설정의 목적·범위·승역지의 표시를 기록
- 승역지 위에 잡히는 물권이므로 승역지에 신청

본문·코드블록은 원본 그대로 보존했고, 아래 두 가지만 별도로 짚어둡니다.

### 1) [참고 — 오류 아님] 실무교육 45시간 (1-4)
- 원문의 "실무교육 = 45시간"은 **2026.1.1 시행 개정 「공인중개사법 시행령」 제28조 기준으로 정확**합니다.

### 2) [분류 점검 권장] 세대구분형 공동주택 (2-9)
- 원문 **수치(전체 1/10 + 동 1/3)** 는 「공동주택관리법」 제35조에 따라 **기존 공동주택을 행위허가·신고로 전환**하는 경우의 요건입니다.
- 반면 코드블록 설명("처음부터 설계 단계에서 구분을 전제로 지은 구조 / 건축 허가 단계에서 인정")은 「주택법」 **사업계획승인을 받아 처음부터 건설**하는 경우에 해당합니다.
- 사업계획승인(처음부터 건설) 케이스의 요건은 **전체 세대수의 1/3 + 전체 주거전용면적 합계의 1/3**로 수치가 다릅니다.
- 즉 한 항목 안에 '행위허가(전환)'의 수치와 '사업계획승인(신축)'의 설명이 섞여 있으니, 어느 케이스를 정리할지 교재로 한 번 더 확인 권장합니다.
