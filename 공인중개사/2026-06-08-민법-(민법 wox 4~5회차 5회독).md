# 📌 1. 무권대리와 표현대리

## 1-1. 무권대리

> 💡 본인이 추인해야 효력이 생기고, 무권대리인은 무과실책임을 진다

### 1) 철회와 추인의 기본 구조
- 무권대리에서 철회는 상대방이 선의인 경우에만 할 수 있다.
- 본인이 추인하지 않으면 그 행위는 본인에 대하여 효력이 없다.
- 무권대리의 추인은 소급하여 효과를 발생시킨다.

### 2) 무권대리인의 무과실책임
- 무권대리인은 무과실책임을 진다. 따라서 무권대리인의 과실 없이 제3자의 기망 등 위법행위로 무권대리 행위가 행해졌더라도, 무권대리인은 상대방에게 책임을 진다.
```
무권대리인 입장에서:
본인을 대리할 권한이 있는지 확인할 의무 → 본인에게 있음
그 확인 없이 나선 자가 무권대리인

상대방 입장에서:
내가 어떻게 기망 경위를 알 수 있나?
나는 오로지 무권대리인의 말만 믿고 계약했을 뿐
```

## 1-2. 표현대리

> 💡 외관을 믿은 선의·무과실 상대방을 보호하기 위한 제도

### 1) 표현대리의 기본 구조
```
대리권 없음(또는 범위 초과)
+ 대리권 있는 것처럼 보이는 외관
+ 상대방의 선의·무과실
= 본인이 책임
```

### 2) 유권대리 주장과 표현대리 주장의 구별
- 유권대리 주장 속에 표현대리 주장이 당연히 포함되는 것은 아니다.
```
유권대리 주장
= "을에게 진짜 대리권이 있었다"
= 본인(갑)이 을에게 대리권을 수여했다는 사실 주장

표현대리 주장
= "진짜 대리권은 없지만,
   외관을 갑이 만들어냈고,
   나(병)는 그걸 선의·무과실로 믿었다"
= 갑의 귀책사유 + 상대방 선의·무과실을 별도로 주장
```

### 3) 기본대리권의 종류는 불문
- 법정대리권을 기본대리권으로 하는 표현대리가 성립할 수 있다.
```
핵심 = "상대방 보호"

기본대리권이 임의대리인지 법정대리인지는
상대방 입장에서 알 수 없음

상대방은 그냥
"이 사람이 대리권 있구나" 하고 믿은 것

→ 대리권의 종류를 불문하고
   외관을 믿은 상대방을 보호하는 게 맞다
```
- 사자(使者)도 표현대리의 기본대리권으로 인정될 수 있다.
```
사자로 활동하던 을이
권한을 넘어 대리인처럼 행동한 경우

→ 상대방 병 입장에서는
  "이 사람이 대리권 있는 대리인인가,
   아니면 전달만 하는 사자인가"
   구별하기 어려움

→ 외관을 신뢰한 상대방 보호 필요
→ §126 표현대리 성립 가능
```
- 권한을 넘은 표현대리의 기본대리권은 대리행위와 같은 종류의 행위에 관한 것일 필요가 없다.
```
기본대리권: 갑이 을에게
            "내 도장 맡아서 주민등록 서류 발급받아와"
            (행정적 심부름 권한)

실제 행위: 을이 그 도장으로
           병에게 갑 소유 부동산 매매계약 체결
           (재산처분 행위)
```

### 4) 현명주의와 표현대리
- 본인을 위한 것임을 현명하지 않은 경우에는 표현대리가 성립하지 않는다.
```
을이 병과 계약할 때
"나는 갑의 대리인으로서 계약합니다" → 현명 O
"나는 그냥 나로서 계약합니다"      → 현명 X
```

### 5) 과실상계의 부적용
- 표현대리에는 과실상계의 법리가 적용되지 않는다. 표현대리가 성립한 경우, 상대방에게 과실이 있다고 해서 본인의 책임을 감경할 수 없다.
```
과실상계는 손해배상액을 조정하는 법리인데, 표현대리에서 상대방이 원하는 건 계약 이행 그 자체이기 때문
```

### 6) 표현대리 주장권자
- 표현대리는 직접 상대방만 주장할 수 있다. 본인이나 전득자는 주장할 수 없다.
```
표현대리 주장권
= 상대방(병)을 보호하기 위해
  병에게 부여된 권리

= 병의 일신전속적 권리
```

### 7) 표현대리가 성립하지 않는 경우
- 기본대리권이 처음부터 존재하지 않는 경우에는 표현대리가 성립할 수 없다.
- 대리행위가 강행규정을 위반하여 무효인 경우에는 표현대리가 성립할 수 없다.

# 📌 2. 토지거래허가구역과 유동적 무효

## 2-1. 유동적 무효의 법률관계

> 💡 허가 전이라도 구속력 있는 법률관계가 존재한다

### 1) 취소권 행사 가능
- 토지거래허가구역 내에서 계약이 유동적 무효인 경우에도 취소할 수 있다. 따라서 매매계약이 사기에 의해 체결된 경우, 토지거래허가를 신청하기 전에 사기를 이유로 계약을 취소하여 허가신청절차의 협력의무를 면할 수 있다.
```
유동적 무효 상태라도
계약은 당사자 사이에 존재하고 있음

= 허가 신청 협력 의무 등
  일정한 법적 구속력은 발생한 상태

구속력이 있는 법률관계가 존재하는 이상
그 법률관계에 취소·해제 등을
행사할 수 있어야 함
```

### 2) 중간생략등기의 무효
- 토지거래허가구역에서 중간생략등기는 무효이다.
```
갑→병 직접 등기
= 을→병 거래에 대한 허가를
  사실상 건너뜀

②번 허가를 우회하는 수단으로 악용
```

### 3) 이행청구의 범위
- 허가 전에는 허가신청협력의무 이행청구만 가능하다. 이전등기청구는 허가 후에 발생하는 권리이므로, 토지거래허가가 있을 것을 조건으로 하여 소유권이전등기절차의 이행을 청구할 수 없다.

### 4) 해제 가능 여부
- 유동적 무효 상태에서는 채무불이행 해제는 불가능하고, 해약금 해제는 가능하다.
```
채무불이행 해제가 안 되는 이유:
허가 전 = 이행청구권 미발생
= 채무불이행 자체가 성립 안 함
= 해제 원인 없음

해약금 해제가 되는 이유:
허가·이행청구권과 무관하게
계약금만 포기/배액상환하면
그냥 나올 수 있는 구조
```

# 📌 3. 무효·취소·추인

## 3-1. 무효행위의 추인

> 💡 무효 원인이 소멸한 뒤에야 추인할 수 있다

### 1) 추인의 시기
- 무효행위의 추인은 그 무효의 원인이 소멸한 후에 해야 한다.
```
무효 원인이 아직 존재하는 상태에서
추인을 해도

→ 그 추인 행위 자체가
   동일한 무효 원인에 걸려버림
→ 추인도 무효
→ 추인의 의미 없음
```

### 2) 무권리자 처분행위의 추인
- 무권리자의 행위는 유동적 무효이다. 따라서 일반 무효와 달리 추인 시 소급하여 유효이다.
```
무효행위 추인 (§139)
= 행위 자체에 흠이 있는 무효
  예) 불공정 법률행위, 강행법규 위반 등

무권리자 처분
= 행위 자체는 완전하고 흠이 없음
  단지 "처분 권한이 없는 사람"이 한 것
```

### 3) 추인할 수 없는 무효
- 강행규정 위반, 사회질서 위반, 불공정한 법률행위는 추인해도 무효이다.

### 4) 추인할 수 있는 무효·취소사유
- 질서위반이 아닌 통정허위표시, 취소사유들은 당사자가 무효임을 알고 추인하면 새로운 법률행위로서 무효행위 추인이 된다.
```
강박 취소행위 추인 (§143):
"강박으로 한 계약인 거 알지?
 그래도 유지하겠어" → 추인

통정허위표시 추인 (§139 단서):
"우리 짜고 한 허위계약인 거 알지?
 이제 진짜로 하자" → 추인

공통점:
둘 다 "흠을 인식한 상태에서" 추인
```

### 5) 일부무효·일부취소
- 일부무효와 일부취소는 원칙적으로 전부 무효이지만, 분할가능성과 가상적 의사가 있다면 나머지는 유효로 할 수 있다. 이때 일부취소도 추인되면 소급효가 발생한다.

## 3-2. 취소의 효과와 성질

> 💡 취소는 형성권이고, 그 상대방은 한정된다

### 1) 주장 상대방의 차이
- 취소의 상대방은 무조건 그 상대방인 반면, 무효는 누구에게나 주장할 수 있다.

### 2) 미성년자 취소와 현존이익
- 미성년자가 매도하고 이후 법정대리인이 취소했는데, 미성년자가 대금을 모두 생활비로 썼으면 대금 전체를 반환해야 한다.
```
갑의 재산 상태 비교:

대금 없었다면:
생활비 100만원
→ 갑의 재산에서 지출
→ 재산 100만원 감소

대금 받아서 생활비로 씀:
대금 100만원으로 생활비 충당
→ 갑의 재산에서 지출 안 함
→ 재산 100만원 보존

결론:
대금을 생활비로 썼어도
갑의 재산은 100만원만큼
이익이 현존함
```

### 3) 취소권의 성질
- 취소권은 형성권이다.
```
형성권:
일방 의사표시만으로
법률관계 변동
→ 상대방 동의 불필요
예) 취소권·해제권·상계권
```

# 📌 4. 조건과 기한

## 4-1. 기한과 조건의 작동

> 💡 시기·종기의 도래 시점과 조건의 효력을 구별한다

### 1) 기한이익상실 특약
- 기한이익상실의 특약은 정지조건부가 아니라 형성권적으로 추정한다. 즉 채권자가 사용해야 바로 효력이 발생한다.

### 2) 불확정 사실과 이행기간
- 불확정한 사실의 발생시기를 이행기간으로 정한 경우, 그 사실의 발생이 불가능하게 된 순간 이행기간은 도래한 것으로 본다.

### 3) 상계와 시기
- 상계는 즉각적 확정성이 생명이므로 시기를 붙일 수 없다.

### 4) 기성조건의 어원
- 旣成條件(기성조건) = 旣 이미 기 + 成 이룰 성 + 條件 조건 = "이미 성취됐거나 성취 불가능한 것을 조건으로 붙인 것"

### 5) 불법조건의 효과
- 사회질서에 반한 조건이 해제조건이라고 해서 조건 없는 법률행위가 되는 것이 아니라, 그냥 조건은 있는데 무효인 것이다.

# 📌 5. 물권적 청구권

## 5-1. 물권적 청구권의 종류와 한계

> 💡 점유 여부에 따라 인정되는 청구권이 다르고, 비용청구는 불가하다

### 1) 점유하지 않는 권리
- 지역권 및 저당권은 점유하는 권리가 아니므로 반환청구권이 인정되지 않는다. 방해예방청구권과 방해제거청구권만 가능하다.

### 2) 사기에 의한 인도
- 사기에 의해 물건을 인도한 자는 침탈당한 것이 아니라 자의로 물건을 넘긴 것이므로, 점유물반환청구권을 행사할 수 없다. 물권적 청구권으로 해결하는 것이 아니라, 사기로 인하여 계약을 취소하고 부당이득반환청구를 해야 한다.

### 3) 임차인의 대위행사
- 임차인은 임대인의 소유권에 기한 물권적 청구권을 대위행사할 수 있다.

### 4) 예방·담보 청구
- 소유자는 자신의 소유권을 방해할 염려 있는 행위를 하는 자에 대하여 그 예방이나 손해배상의 담보를 청구할 수 있다.

### 5) 무단 건물의 처리
- 무단으로 나의 토지에 건물을 지은 자에게, 그 건물은 내 것이 아니므로 건물에서 나가라고는 할 수 없다. 다만 건물을 철거하라고 할 수 있을 뿐이다.

### 6) 허무인 명의 등기 말소
- 소유자의 방해배제청구로 인한 등기의 말소청구는 허무인 명의로 해두든 뭘 하든 간에 무조건 가능하다.

### 7) 비용청구의 불가
- 소유자는 물권적 청구권에 의하여 방해제거비용 또는 방해예방비용을 청구할 수 없다.
```
물권적청구권의 내용:

"네가 직접 방해를 제거해라"
"네가 직접 예방조치를 해라"

= 방해자의 행위를 요구하는 것

비용 청구:
"내가 방해를 제거할 테니
 비용을 내놔라"

= 금전 지급을 요구하는 것
= 물권적청구권의 내용이 아님
```

### 8) 방해 원인 vs 방해 결과
- 소유권에 기한 방해제거청구권은 현재 계속되고 있는 방해의 원인을 제거하는 것이지, 방해 결과의 제거를 목적으로 하는 것은 아니다.
```
을이 갑 소유 토지에
무단으로 건물 지음

방해 결과:
건물이 토지 위에 서 있는 상태
→ 방해제거청구권으로 철거 요구 ✅

방해 원인:
건물을 지을 수 있는 자재·장비
→ 이건 방해제거청구권 대상 ❌
→ 방해예방청구권 대상 ✅
```

### 9) 미등기 무허가건물
- 미등기 무허가건물은 등기 자체가 불가능한 건물로 소유권 공시 수단이 없어, 이 건물의 양수인은 소유권에 기한 방해배제청구권을 행사할 수 없다.

# 📌 6. 중간생략등기와 등기청구권

## 6-1. 중간생략등기

> 💡 합의가 있으면 직접 청구·대위 청구 모두 가능하다

### 1) 합의된 중간생략등기(미등기전매)
- 중간생략등기에 합의했다는 것은 유효인 중간생략등기이고, 매도해서 인도하고 대금 받고 등기는 아직 안 해준 사례가 출제되므로 미등기전매를 했다는 의미이다. 갑→을→병 순으로 매매를 했으면, 병은 갑에게 직접 이전등기 청구도 되고(합의했으므로), 을을 대위해서 등기를 넘기라고 할 수도 있고, 을도 갑에 대해 등기를 넘겨달라고 할 수 있다. 더해 갑의 소유물 반환청구는 누구에게도 불가능하다.

### 2) 단속규정으로서의 성질
- 중간생략등기는 단속규정이어서 합의했다면 바로 등기해주는 것이 유효하다.
- 중간생략등기가 이미 경료된 경우, 합의가 없어도 실체관계에 부합하면 유효이다.

### 3) 대금 인상과 등기 거부
- 갑·을 사이에 매매대금을 인상했는데 을이 안 주면, 갑은 병에게도 등기를 해주지 않을 수 있다.

## 6-2. 등기청구권

> 💡 발생 원인에 따라 양도성과 시효 진행이 달라진다

### 1) 등기청구권의 양도
- 매매와 같은 법률행위로 인한 매수인의 등기청구권은 채권양도의 법리에 따라 양도할 수 없고, 양도하려면 매도인의 동의가 필요하다. (점유취득시효 완성으로 인한 청구권은 통상 채권양도의 법리에 따라 양도 가능)

### 2) 점유 상실과 소멸시효
- 점유취득시효를 완성한 자가 점유를 상실해도 바로 등기청구권이 사라지는 것이 아니라 소멸시효가 진행되는 것이다.
- 매수인이 매도인에게 물건을 인도받은 후 제3자에게 점유를 승계한 경우, 매도인에 대한 등기청구권 시효는 진행되지 않는다. → 이 경우 간접점유를 인정해 줌

### 3) 등기청구권과 등기신청권
- 등기청구권은 사법상 권리, 등기신청권은 공법상 권리다.
```
등기청구권 (사법상 권리)
≠
등기신청권 (공법상 권리)

등기신청권:
등기권리자·의무자가
함께 국가(등기소)에
등기를 신청하는 권리
= 이건 공법상 권리 맞음

등기청구권:
등기권리자가
등기의무자에게
등기에 협력하라고 요구하는 권리
= 사법상 채권
```

# 📌 7. 물권변동과 등기

## 7-1. 등기를 요하는 물권변동과 요하지 않는 물권변동

> 💡 제186조(등기 필요) vs 제187조(등기 불요)를 구별한다

### 1) 등기 필요 / 불요의 구별
- 매매와 같은 법률행위, 점유시효취득 완성(등기청구권만 가진 것)은 등기를 요하는 부동산 물권변동이다. 건물 신축이나 판결은 바로 물권취득의 효력이 있다.

### 2) 분묘기지권
- 관습법 자체가 "등기 없이도 효력 있다"는 내용을 포함하고 있으므로, 분묘기지권은 등기 없이도 물권변동의 효력이 발생한다.

### 3) 이행판결
- 이행판결은 이행하라는 판결이지 바로 물권을 변동시키지 않는 판결이므로, 이후에 등기를 해야 물권이 변동한다.

### 4) 합유
- 합유자는 조합체가 존속하는 한 합유물의 분할을 청구하지 못하며, 합유자의 권리는 합유물 전부에 미친다. 또한 합유는 조합체의 해산으로 인하여 종료한다.

### 5) 관습법상 물권의 성립과 양도
- 강제경매로 인해 성립한 관습법상 법정지상권을 법률행위에 의해 양도하기 위해서는 등기가 필요하다.
```
분묘기지권도:
관습법으로 성립 → 등기 불요 ✅
법률행위로 양도 → 등기 필요 ✅

모든 관습법상 물권
성립과 양도는 별개의 문제
```

### 6) 경매와 소유권 취득시기
- 경매로 인한 부동산 소유권의 취득시기는 매각대금을 완납한 때이다.

### 7) 합의해제와 말소청구권
- 계약을 합의해제 후 원상복구를 위한 등기말소청구권은 명의를 회복하는 것이므로, 채권적 청구권이 아니라 물권적 청구권이다.
```
합의해제
= 계약 소급 소멸
= 처음부터 계약 없었던 것

소유권 이전 효과도 소급 소멸
= 소유권이 원래 소유자에게
  복귀한 것으로 봄
```

# 📌 8. 가등기와 등기의 추정력

## 8-1. 가등기

> 💡 순위는 소급하나 물권변동 시기는 본등기 시점이다

### 1) 본등기와 소급효
- 청구권 보전을 위한 가등기에 기한 본등기에 의한 물권변동 시기는 소급효가 인정되지 않는다.
```
순위소급 ✅
= 본등기 순위가 가등기 시점으로 올라감
= 가등기 후 본등기 전 사이
  중간등기들을 밀어낼 수 있음

물권변동 소급 ❌
= 소유권 취득 시점은
  어디까지나 본등기 시점
```

### 2) 정지조건부 청구권 보전
- 장래 권리의 순위를 미리 확보하기 위해 정지조건부 청구권을 보전하기 위해서도 가등기를 할 수 있다.

### 3) 본등기 절차에 의하지 않은 별도 본등기
- 가등기에 기한 본등기 절차에 의하지 않고 별도의 본등기를 경료받은 경우, 제3자 명의로 중간처분의 등기가 있어도 가등기에 기한 본등기 절차의 이행을 구할 수 있다.
```
가등기하고 거기에 본등기를 안하고 가등기-3자등기-따로 본등기 이렇게 해도 가등기에 본등기를 할 수 있다. 왜냐면 따로 본등기는 3자등기를 밀어낼 수 없기때문에 의도대로면 무용지물이기 때문이다. -> 그래서 가등기에 본등기 하는 것을 허락하는 것
```

## 8-2. 무효등기의 유용과 추정력

> 💡 부동산이 동일하면 권리관계 교체(유용)가 가능하다

### 1) 전용 약정
- 전용 약정은 무효인 가등기를 유효한 가등기로 "다시 쓰기로 합의"하는 것을 의미한다. 이 때문에 전용하기로 약정하면 당사자가 무효임을 알고 추인한 경우로서 새로운 법률행위로 본다. 즉 소급효가 발생하지 않는다.

### 2) 무효등기의 유용 합의
- 무효등기의 유용은 무효인 등기를 새로운 법률관계에 맞게 재활용하기로 합의하는 것으로, 반드시 명시적으로 이루어져야 하는 것은 아니다.

### 3) 사항란의 의미와 유용 가능 여부
- 사항란은 갑구, 을구에 권리 내용이 기재되는 칸을 의미한다.
```
보존등기 유용 시도:

[A건물 등기부]          [B건물 실체]
A건물 을 소유 →  유용  → B건물 을 소유?
          ↑
     부동산 자체가 달라
     공시 기능 붕괴
     유용 불가 ❌

사항란 등기 유용:

[갑 소유 토지 등기부]
갑→을 이전등기(무효)  →  새 매매계약으로 유용
          ↑
     부동산은 동일
     권리관계만 교체
     유용 가능 ✅
```

### 4) 각 등기의 추정력
- 가등기는 추정력이 없고, 소유권이전등기는 승계취득한 사실을, 보존등기는 원시취득한 사실이 추정된다.
- 사망 전에 등기원인이 이미 존재했다면 추정력이 있지만, 사망자 명의로 신청한 등기는 추정력이 없다.
- 등기의 추정력 = 등기가 있으면 그 등기대로 권리관계가 존재한다고 일단 추정.
