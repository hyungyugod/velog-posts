# 📌 1. 민법 법률행위·의사표시 (2~3회차 5회독)

## 1-1. 법률행위의 종류·요건
> 💡 단독행위의 '상대방 유무'와 '효력요건 vs 등기요건'을 구분한다

### 1) 상대방 없는 단독행위
- 상대방 없는 단독행위(유유재소): 유언, 유증, 재단법인설립행위, 소유권의 포기
- '상대방이 없다'는 것은 특정인의 수령 없이도 효력이 발생한다는 의미다
```
유언자가 유언을 남김
→ 효력 발생 시점 = 유언자 사망 시 (「민법」 제1073조)
→ 이때 유언자는 이미 사망 → 의사표시를 누군가에게 "도달"시킬 수 없음
→ 수증자(상속받는 사람)가 수령했는지 여부와 무관하게 효력 발생
→ 따라서 수령을 요하는 "상대방"이 존재하지 않음
```
- 상대방은 수인일 수 있다 → ex) 공유지분의 포기에서 다른 공유자 전원

### 2) 효력요건 vs 등기요건
- 토지거래허가구역 내 토지거래허가는 법률행위의 효력을 발생시키는 효력요건이나, 농지거래계약에서 농지취득자격증명은 효력요건이 아니다
- 농지취득자격증명은 등기요건이다
```
농지 매매계약 체결
→ 계약 자체는 즉시 유효 (농지취득자격증명 불필요)
→ 소유권이전등기 신청 시
   → 농지취득자격증명 첨부 필요 (「농지법」 제8조)
→ 증명서 없으면 등기 신청 자체가 불가
→ 즉, 계약 효력 ≠ 등기 가능 여부
```

## 1-2. 강행규정·단속규정·효력규정
> 💡 위반의 목적이 '거래 무효'면 효력규정, '행위자 제재'면 단속규정

### 1) 단속규정 사례
- 무허가, 중간생략등기, 전매약정, 개업공인중개사가 중개의뢰인과 직접 거래 → 모두 단속규정이다
```
법이 어떤 행위를 금지함
→ 위반했을 때 법이 노리는 목적이 뭔가?

① 그 거래 자체를 아예 없던 일로 만들려는 것
   → 효력규정 → 계약 무효

② 그 행동을 한 사람을 처벌/제재하려는 것
   (거래 자체는 인정하되 행위자만 규제)
   → 단속규정 → 계약 유효, 행위자만 제재
```

### 2) 강행규정과 추인
- 임차인의 비용상환청구권은 임차인을 보호해야 하므로 강행규정이다
- 강행규정을 위반하여 무효인 법률행위는 추인해도 유효가 될 수 없다 (싹 다 무효)
```
추인이란?
→ "이 행위, 내가 인정할게"라고 확정하는 것
→ 전제: 추인하기 전까지는 "일단 유효하지만 흔들리는 상태"여야 함
   (= 취소할 수 있는 상태)

강행규정 위반 행위는?
→ 처음부터 아무 효력이 없는 상태 (= 완전 무효)
→ "흔들리는 유효"가 아니라 "존재 자체가 없는 것"
→ 없는 것을 인정한다고 해서 생겨나지 않음

결론:
→ 추인 = 있는 것을 확정하는 행위
→ 강행규정 위반 = 없는 것
→ 없는 것에 추인을 갖다 대도 여전히 없음
```

### 3) 위약벌·초과 중개수수료 (일부무효)
- 위약벌(違約罰) = 違(어길 위) + 約(약속 약) + 罰(벌할 벌) "약속을 어겼을 때 부과하는 벌"
- 과도한 위약벌은 상대방을 경제적 파멸로 이르게 하기 위한 수단이 될 수 있으므로 무효이다
- 초과 중개수수료 약정은 한도 초과 부분만 무효이다
```
강행규정 위반 = 원칙적으로 무효 (맞음)
그런데 「민법」 제137조 일부무효 법리 적용
→ "법률행위의 일부분이 무효인 때에는
    그 나머지 부분도 무효로 한다"
→ 단, 나머지 부분만으로도 유효하게 성립할 수 있으면
    나머지 부분은 유효

중개수수료 약정의 경우
→ 한도 내 부분 = 적법한 수수료 약정 → 독립적으로 유효 성립 가능
→ 초과 부분만 잘라내도 계약의 본질은 살아있음
→ 따라서 초과 부분만 무효, 나머지는 유효
```

## 1-3. 반사회적 법률행위 — 부동산 이중매매
> 💡 제3자 적극가담 시 제2매매는 「민법」 제103조 위반으로 무효, 전득자도 보호받지 못한다

### 1) 적극가담과 무효의 효과
- 부동산 이중매매에서 제3자가 적극가담한 경우 이는 사회질서위반으로 무효이므로 전득한 자도 소유권을 획득할 수 없다. 또한 원래 매수인의 채권자취소권도 행사할 수 없다. 손해배상을 청구하거나 유효인 제1매매계약에 대한 이전등기를 청구하는 수밖에 없다
```
이중매매에서 제3자가 적극 가담
→ 제2매매계약은 반사회적 법률행위로 무효 (「민법」 제103조)
→ 무효인 행위는 처음부터 아무 효력이 없음

채권자취소권의 대상
→ "채무자의 법률행위"가 있어야 함
→ 그런데 제2매매는 처음부터 무효
→ 법률상 아무 효력 없는 행위 = 법률행위로서 존재하지 않는 것
→ 취소할 법률행위 자체가 없음

추가로:
→ 채권자취소권은 유효하게 성립한 법률행위를 취소하는 것
→ 처음부터 무효인 행위를 다시 취소할 이유가 없음
   (이미 무효인 걸 취소해서 얻을 게 없음)
→ 취소권 행사의 실익도 없음
```

### 2) 적용 확장과 물권의 절대성
- 부동산 이중매매의 법리는 이중으로 부동산임대차계약이 체결되는 경우에도 적용될 수 있다
- → "먼저 계약한 사람의 채권을 제3자가 적극 가담해서 침해했다"는 본질이 동일하기 때문
- 이중매매에서 일반적으로 물권은 절대적으로 누구에게나 주장 가능하기 때문에 먼저 등기한 매수인이 목적부동산의 소유권을 취득한다. 이때 예외적으로 제2매수자가 적극가담했을 경우에는 소유권을 취득할 수 없다

## 1-4. 의사표시의 해석·착오
> 💡 합의는 채권행위, 등기는 물권행위 — 둘은 별개로 효력을 따진다

### 1) 지번 착오 토지매매 (계약 유효, 등기 무효)
- 지번을 착오한 토지매매는 계약은 유효지만 등기는 무효이다. 등기는 합의의 반영이기 때문이다
```
채권행위 (매매계약)
→ 갑·을이 A토지를 거래하기로 한 계약
→ 이 계약 자체는 유효 (A토지에 대한 합의 있음)

물권행위 (등기)
→ B토지로 소유권이전등기 완료
→ B토지에 대한 물권적 합의? → 없음
→ 당사자가 합의한 건 A토지인데 B토지로 등기됨
→ 물권적 합의 없는 등기 = 무효
```
- 지번을 착오한 토지매매에서 착오를 이유로 계약을 취소할 수는 없는데, 이는 착오가 "계약 내용"에 있는 게 아니라 "등기 서류 작성"에 있기 때문이다

### 2) 해석방법과 착오의 요건
- 자연적 해석방법은 내심의 효과의사이고, 규범적 해석방법은 표시상의 효과의사를 말한다
- 중요부분에 대한 착오만 취소할 수 있는데, 경제적 불이익이 없으면 중요부분의 착오로 보지 않는다
- 착오취소는 임의규정이다

## 1-5. 비진의표시·통정허위표시
> 💡 둘 다 무효사유지만, 통정허위표시는 선의의 제3자에게 대항할 수 없다

### 1) 비진의표시
- 비진의표시에서 진의는 진정으로 마음속에서 원하는 사항이 아니라, 특정한 내용의 의사표시를 하고자 하는 표의자의 생각이다
- 비진의표시는 상대방이 무과실이어야 유효이다 → 비진의표시의 상대방이 처음부터 무효임을 알았을 경우 이는 처음부터 무효이므로 표의자가 취소할 수도 없다

### 2) 통정허위표시(가장매매)와 제3자
- 가장매매는 통정허위표시로 계약이 무효이지만 사회질서위반으로 무효는 아니므로 선의의 제3자에게 무효로 대항할 수 없다. 또한 제3자는 항상 선의이기만 하면 된다
- 만약 제3자가 악의인 경우는 저당권이나 소유권 등기의 말소를 청구할 수 있다. 이는 소유권에 기한 물권적 청구권 중 방해배제청구권에 해당한다
- 통정허위표시는 채권자취소권의 대상이 될 수 있다
```
원칙: 무효인 행위 → 취소 대상 아님 (강행규정 위반 때 배운 것)

통정허위표시 예외가 되는 이유
→ 채권자는 당사자가 아니므로
   무효를 당연히 원용할 수 있는 위치가 아님
→ 채권자 입장에서 이 행위는
   "무효인 행위"가 아니라 "사해행위"로 포착됨
   사해행위는 채권자를 해치는 행위지만 일단 유효고 취소할 수 있는 것
→ 따라서 채권자취소권 행사 가능
```
- 통정허위표시가 성립하려면 단순히 아는 것이 아니라 짜고 치는 수준까지 가야 한다

### 3) 제3자에 해당하는지 여부
- 파산관재인 → 파산채권자 전원이 악의가 아니라면 선의의 제3자
- 본인은 대리인의 통정허위표시의 "제3자"가 아니다. 대리행위의 효과는 직접 본인에게 귀속되므로 본인은 그 법률관계의 당사자이다
- 채권의 가장양도에서 변제 전 채무자는 새롭게 신뢰하고 거래에 뛰어든 자가 아니므로 제3자가 아니다

## 1-6. 취소사유와 소송행위
> 💡 취소사유는 착오·사기·강박·제한능력 4개뿐, 소송행위는 취소 불가

### 1) 취소사유 4가지
- 비정상적 의사표시에서 비진의표시와 통정허위표시는 무효사유이고, 착오·사기·강박·제한능력자의 법률행위는 취소사유이다. 취소사유는 이 4개뿐이라는 것이 엄청 중요하다

### 2) 소송행위와 제한능력자 수령
- 착오·사기·강박에 의한 소송행위는 원칙적으로 취소할 수 없다
```
민법 109조·110조 취소권의 전제
→ 사적 자치 영역에서의 법률행위
→ 개인 간 거래 = 당사자 의사 보호가 핵심

소송행위의 성격
→ 사적 자치 영역 ❌
→ 국가의 사법(司法) 절차 영역 ✅
→ 법원·상대방·제3자 모두 관련된 공적 절차

만약 소송행위를 취소할 수 있다면?
→ "그 자백은 착오로 한 거라 취소"
→ "그 소 취하는 사기당해서 취소"
→ 확정된 소송 결과가 뒤집어짐
→ 소송절차 전체가 불안정해짐
→ 법적 안정성 붕괴
```
- 의사표시의 상대방이 의사표시를 받은 때에 제한능력자인 경우, 표의자는 원칙적으로 그 의사표시로써 대항할 수 없다 = 제한능력자는 의사표시를 받아도 그 내용을 제대로 이해하고 판단할 능력이 없으므로, 표의자가 "나 의사표시 했어"라고 주장할 수 없다. 즉 의사표시의 효력을 주장할 수 없다

## 1-7. 사기·강박과 대리
> 💡 대리인의 행위 효과는 본인에게 귀속 — 대리인 사기 = 무조건 취소

### 1) 사기의 성립
- 부작위나 침묵은 사기지만, 시가나 다소 과장은 사기가 아니다

### 2) 대리권과 배임적 대리행위
- 대리인의 기망행위로 계약을 체결한 상대방은 무조건 계약을 취소할 수 있다
```
대리인의 기망행위는?
→ 대리인의 행위 효과 = 본인에게 귀속 (「민법」 제114조)
→ 대리인이 사기 = 본인이 사기한 것과 동일
→ 무조건 취소 가능
```
- 대리권은 절대 상속되지 않는다 → 사람이 달라지는데 그 대리인이어야만 의미가 있는 것이다
- 대리인이 배임적 대리행위를 하고 매수인도 이를 안 경우 본인에게 법률의 효력이 미치지 않는다 → 비진의표시를 유추적용하여 무효
```
「민법」 제107조 제1항 단서
→ 비진의표시에서
→ 상대방이 표의자의 진의 아님을 알았거나
   알 수 있었을 경우 → 무효

유추적용 이유
→ 배임적 대리행위 구조
   = 대리인은 겉으로 정상적 대리행위처럼 표시
   = 실제로는 본인 이익에 반하는 의도
   = 상대방이 이를 알고 있음

→ 비진의표시와 구조 유사
   (표시와 진의가 다르고, 상대방이 이를 앎)
→ 107조 단서 유추적용 가능
→ 본인에게 효력 불귀속
```

## 1-8. 명의신탁
> 💡 원칙 무효, 배우자·종중 예외 / 실명법은 선의·악의 불문 제3자 보호

### 1) 원칙과 예외
- 명의신탁은 원칙적으로 무효이나 배우자나 종중에게 한 명의신탁은 법령상 제한 회피 목적이 아니면 유효하다. 이때 신탁자가 제3자에게 수탁자에게 이전했던 토지를 매도해도 이는 아직 실질적 소유자가 신탁자기 때문에 타인권리매매라고까지 볼 건 아니다

### 2) 수탁자의 재취득·재처분
- 무효 명의신탁에서 수탁자가 적법하게 건물을 제3자에게 양도했다가 다시 소유권을 취득한 경우, 신탁자가 소유물반환을 청구할 수 없다. 이는 수탁자가 소유권을 다시 취득한 것은 명의신탁과 무관한 새로운 원인에 의한 취득이기 때문에 그렇다
- 위의 경우에서 수탁자가 건물을 제3자에게 매도하고 등기를 해줬다면, 그건 수탁자가 신탁자에게 불법행위책임을 진다
- 계약명의신탁에서 등기를 이전받은 수탁자가 자의로 신탁자에게 이전등기를 하였으면 신탁자는 소유권을 취득한다 → 약정에 기한 이행이 아닌 새로운 법률행위를 한 것이기 때문

### 3) 실명법의 제3자 보호
- 명의신탁약정의 무효만 가지고서는 악의의 제3자에게 대항할 수 없다 → 실명법은 민법 108조(통정허위표시)와 달리 선의·악의를 불문하고 제3자를 보호하기 때문이다
```
민법 108조의 취지
→ 당사자가 짜고 친 행위
→ 선의로 믿고 거래한 자만 보호
→ 알고 거래한 자(악의)는 보호 불필요

실명법의 취지
→ 부동산 거래 안전 보호가 1차 목적
→ 등기를 믿고 거래한 자 보호
→ 명의신탁 사실을 알았더라도
   등기 공시를 신뢰한 거래는 보호

이유
→ 명의신탁 사실을 아는 것 자체가
   우리 사회에서 매우 흔한 일
→ 악의라고 모두 거래 효력 부정하면
→ 부동산 거래 안전 심각하게 훼손
→ 따라서 선의·악의 불문 보호
```

## 1-9. 집합건물(구분소유)
> 💡 공용부분은 '사용 = 용도, 비용 = 지분', 관리인과 관리위원회는 견제관계

### 1) 공용부분의 사용·비용·물권변동
- 공용부분의 사용은 전유부분의 지분비율에 따르는 것이 아니라 용도에 따라 사용한다
```
사용 = 용도 기준 (모두 동등하게)
비용 = 지분비율 기준 (많이 가진 자가 더 부담)
```
- 구분소유건물의 공용부분에 관한 물권의 득실변경은 등기가 필요하지 않다 → 공용부분은 전유부분에 종속되어 있어서 전유부분 등기만으로 공용부분 물권변동도 자동으로 처리되기 때문

### 2) 관리인과 관리위원회
- 구분소유자가 10명 이상이면 반드시 관리인을 선임해야 한다. 이해관계자가 많아지기 때문
- 관리인은 관리위원회의 위원이 될 수 없다 → 관리인과 관리위원회는 견제·감독 관계이기 때문이다
```
집합건물 관리
→ 집행: 관리인
→ 감독: 관리위원회
```
