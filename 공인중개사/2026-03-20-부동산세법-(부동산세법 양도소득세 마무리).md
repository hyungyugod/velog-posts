# 📌 1. 부동산세법 양도소득세 마무리

## 1-1. 세금의 전체 사이클

> 💡 자산은 취득 → 보유 → 처분 3단계를 거치며, 각 단계마다 세금이 붙는다

### 1) 취득 단계
- 취득세, 등록면허세, 인지세 → "얻기 위해 필수로 낸 돈" → 취득가액 포함
- 상속세, 증여세 → "받았기 때문에 내는 세금" → 취득가액 포함 X

### 2) 보유 단계
- 재산세, 종합부동산세(종부세) → 들고 있는 것 자체에 부과
- 지방교육세 → 메인 세금(취득세/재산세)에 종속된 추가세, 해당 단계 따라감

### 3) 처분 단계
- 양도소득세 → 팔아서 생긴 이익에 과세
- 상속/증여로 넘기면 다시 취득 단계로 순환

---

## 1-2. 취득가액 포함 여부 기준

> 💡 "이 돈이 없으면 자산을 못 얻나?" YES → 포함 / NO → 제외

### 1) 포함되는 항목
- 취득세 → 취득하려면 반드시 내야 하는 비용 → 원가
- 지방교육세(취득세에 붙은 것) → 취득세와 함께 납부 → 원가
- 부가가치세(개인) → 환급 불가능 → 원가

### 2) 포함 안 되는 항목
- 상속세, 증여세 → 받은 후 내는 세금, 취득 비용 아님
- 부가가치세(사업자) → 매입세액 공제로 환급 가능 → 비용 아님

# 📌 2. 양도소득세 계산 구조

## 2-1. 3단계 계산 흐름

> 💡 차익(원초) → 소득(공인) → 과세표준(기준) 순으로 점점 정제된다

### 1) 양도차익
- 양도가액 − 취득가액 − 필요경비 = 양도차익
- "차익(差益)" = 차이에서 생긴 이익, 아직 아무것도 안 빠진 원초적 이익

### 2) 양도소득금액
- 양도차익 − 장기보유특별공제 = 양도소득금액
- "소득(所得)" = 정책적 공제까지 반영해 세법상 소득으로 확정된 금액

### 3) 양도소득 과세표준
- 양도소득금액 − 기본공제(250만원) = 과세표준
- "과세표준(課稅標準)" = 여기에 세율 곱하면 세금 나오는 최종 기준값

---

## 2-2. 과세표준 세금별 차이

> 💡 과세표준은 항상 "그 세금이 보고 싶은 기준"이 다르다

### 1) 세금별 과세표준 비교
- 취득세 → 자산 가격 기준 ("얼마짜리 얻었냐?")
- 양도소득세 → 이익 기준 ("얼마 벌었냐?")
- 재산세 → 보유 가치 기준 ("얼마짜리 들고 있냐?")

# 📌 3. 장기보유특별공제

## 3-1. 공제율 계산 공식

> 💡 일반 자산은 보유연수×2%, 1세대 1주택은 보유×4% + 거주×4%

### 1) 일반 자산
- 공식: 보유연수 × 2% = 공제율
- 3년 이상 6%, 4년 이상 8%, 15년 이상 최대 30%
- 최대 30% 한도

### 2) 1세대 1주택 (조건: 보유 3년 이상 + 거주 2년 이상)
- 보유기간 공제(100분의 12~40) + 거주기간 공제(100분의 8~40) 따로 계산 후 합산
- 3년 보유 + 2년 거주 시 12% + 8% = 20%, 최대 80%까지 가능
- 일반 자산보다 훨씬 높은 공제율 → 실거주 장기보유 정책적 우대

### 3) 적용 배제
- 미등기 자산(법령이 정하는 자산은 제외)
- 국외 자산

# 📌 4. 양도차손 공제 구조

## 4-1. 왜 양도소득금액에서 공제하냐

> 💡 장기보유공제까지 반영해 확정된 후에 합산해야 정확한 정산이 된다

### 1) 단계별 흐름
- 이익 난 자산 → 장기보유공제 적용 → 양도소득금액 확정
- 손실 난 자산 → 장기보유공제 없이 → 차손으로 확정
- 이 둘을 양도소득금액 단계에서 합산해 정산

### 2) 왜 양도차익 단계에서 공제 안 하냐
- 장기보유공제 전에 섞으면 계산 순서가 꼬임
- 각자 계산이 끝난 후 같은 선상에서 플러스/마이너스 하는 구조

### 3) 왜 각 호의 소득별로 공제하냐
- 부동산 손실은 부동산 이익에서만, 주식 손실은 주식 이익에서만 공제
- 다른 종류 자산끼리 손익 통산 불가 → 과세 형평성 유지
