# 📌 부동산공시법 — 부동산등기법 (WOX 6회차 5회독)

## 1. 저당권·근저당권

> 💡 채권액은 필수, 변제기는 옵션 / 채권최고액은 언제나 하나

### 1) 기록사항
- 저당권등기 시 **변제기는 임의적 기록사항**이다.
- 근저당권에서 **존속기간의 약정이 있는 경우**에는 그 약정도 표시해야 한다.

### 2) 채권최고액의 단일성
- 근저당권자의 채권자가 여러 명이라고 채권최고액을 여러 개 기록할 수는 없다. **채권최고액은 단일해야 한다.**

### 3) 채권최고액 감액 변경등기
- 근저당권의 채권최고액을 감액하는 변경등기는 **근저당권설정자와 근저당권자가 공동신청**하여야 한다.

---

## 2. 용익물권 (지상권·지역권)

> 💡 토지 일부엔 지적도, 범위는 전부라도 무조건 기록

### 1) 지역권 시효취득
- 시효완성을 이유로 통행지역권을 취득하기 위해서는 **그 등기가 되어야 한다.**

### 2) 지상권 — 토지 일부 설정
- 1필 토지의 일부에 대해 지상권설정등기를 신청하는 경우, **그 일부를 표시한 지적도를 첨부정보**로서 등기소에 제공하여야 한다.

### 3) 지상권 — 범위 기록
- 지상권설정등기를 하는 경우, 지상권 설정의 범위를 기록하여야 하므로 **1필 토지 전부에 지상권을 설정하더라도 무조건 범위를 기록**해야 한다.

## 3. 임차권·전세권

> 💡 임차권등기명령 = 잔존 권리 박제용 → 이전·전대 불가

### 1) 임차권등기명령 (★ 확인 필요 항목 — 하단 참조)
- 주택임차권등기명령에 따라 임차권등기가 된 경우 그 등기에 기초한 임차권이전등기를 할 수 있다.

```
임차권등기명령 = 임대차가 "이미 끝난" 상태에서
                보증금 못 받은 임차인을 위한 '대피용 등기'
   → 목적은 단 하나: 임차인이 이사 나가도
     대항력·우선변제권을 등기부에 박제해 두는 것
   → 즉 "보증금 회수만 남은 잔존 권리"
   → 살아있는 임대관계가 아니므로
     여기 기초해 임차권을 넘기거나(이전) 다시 빌려줄(전대) 대상이 없다
   → ∴ 이전등기·전대등기 불가
```

### 2) 임차권 전대 → 부기등기
- 임차권설정등기가 마쳐진 후 임대차 기간 중 임대인의 동의를 얻어 임차물을 전대하는 경우, 그 전대등기는 **부기등기**의 방법으로 한다.
  - → 소유권 외의 권리(임차권 등)의 이전등기는 부기등기로 한다는 게 부동산등기법의 원칙이다. 전대는 "임차권을 깔고 그 위에 새 권리를 얹는" 구조라 같은 논리로 부기등기가 된다.

### 3) 공동전세목록
- 2개의 목적물에 하나의 전세권설정계약으로 전세권설정등기를 하는 경우, **공동전세목록을 작성하지 않는다.**
  - → 공동전세목록은 목적물이 "5개 이상"일 때만 작성하기 때문이다.

```
공동전세목록 = 여러 부동산을 하나의 전세권으로 묶을 때
              그 목록을 '별지'로 따로 빼는 장치
   → 부동산이 적으면(4개 이하): 각 등기기록에 그냥 다 나열해도 안 번잡
   → 부동산이 많으면(5개 이상): 일일이 적으면 등기기록이 비대해짐
   → ∴ 5개 이상일 때만 별지 목록으로 묶는다
   → 2개? → 그냥 각 등기기록에 적으면 끝 → 목록 작성 X
```

## 4. 등기 신청·실행 일반

> 💡 권리변경은 승낙 있으면 부기·없으면 주등기 / 협의분할 상속도 원인일자는 사망일

### 1) 변경등기의 부기/주등기 구분
- 권리의 이전이 아닌 권리의 변경등기는 **이해관계 제3자의 승낙이 있으면 부기등기**로 해도 되지만, 없다면 **주등기**로 한다.

### 2) 상속재산분할협의에 따른 상속등기
- 상속등기 경료 전의 상속재산분할협의에 따라 상속등기를 신청하는 경우, **등기원인일자는 피상속인의 사망일**로 한다.

### 3) 판결에 의한 단독신청
- 채권자가 채무자가 등기절차에 협력하지 않음을 이유로 소송을 해 확정판결을 받으면 **단독신청도 당연히 가능**하다.

### 4) 합병 제한 — 건물 일부 저당권
- 건물도 일부에 저당권이 설정되어 있으면 합병할 수 없다. → **"도면으로 쪼갤 수 없다."**

```
"A건물에만 저당권 → 나중에 A부분만 경매하면 되지"
   ← 이건 '합병 전' 얘기다.
   합병하면 A + B = 등기기록 1개 = 법적으로 '하나의 건물'
   → 옛 A부분은 더 이상 독립한 등기 단위가 아니다
   → 경매는 '등기기록(1부동산) 단위'로 진행된다
   → ∴ "A부분만 떼어 경매"할 등기상 토대가 없어진다
```

### 5) 등기필정보 통지 생략
- 등기관이 새로운 권리의 등기를 마친 경우에, **등기필정보의 통지를 원하지 않은 등기권리자**에게는 등기필정보를 통지하지 않아도 된다.

## ⚠️ 확인 필요

### 1) 메모 3-1) — 임차권등기명령 기초 이전등기 (★ 메모 내부 모순)
- **메모 본문**: "임차권이전등기를 **할 수 있다**"
- **메모 코드블록 결론**: "이전등기·전대등기 **불가**"
- 두 진술이 정면으로 충돌함. 법리상으로는 **코드블록(불가)이 맞다.**
  - 임차권등기명령에 의한 임차권등기는 종료된 임대차의 보증금 회수를 위한 '잔존 권리 보전' 목적이라, 이를 기초로 한 임차권이전등기·임차물전대등기는 할 수 없다는 것이 통설·실무.
  - 본문 문장은 **틀린 지문(OX의 X)** 으로 옮겨 적은 것으로 보임 → 정답 판단 시 "할 수 없다"가 옳음. 원문(WOX 지문) 확인 필요.

### 2) 공동전세목록 "5개 이상" 기준 조문
- "5개 이상일 때만 작성" 기준 자체는 맞으나, 근거 조문 번호(부동산등기규칙 해당 조항)는 원문 확인 권장.

### 3) 등기필정보 통지 생략 근거 조문
- "통지를 원하지 않은 자에게는 통지하지 않아도 된다"는 결론은 맞으나, 정확한 규칙 조문 번호는 원문 확인 권장.

> 법령 개정 가능성: 부동산등기규칙은 개정 빈도가 있으니 시행 중 기준 재확인 권장.
