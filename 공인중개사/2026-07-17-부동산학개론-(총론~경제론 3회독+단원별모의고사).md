# 📌 1. 부동산학개론 총론·경제론 3회독 정리

## 1-1. 주택법과 주택의 종류
> 💡 주택법 = 주택사업을 규정하는 법 / 주택 = 장기 거주용 구조

### 1) 주택법의 의의
- 주택법은 '주택사업'을 규정한다는 데에서 의미가 있다.

### 2) 주택의 정의
- 주택 = 사람이 장기간 거주할 수 있는 구조로 되어 있는 것.

### 3) 다중주택
- 하숙집 같은 형태. '독립된 주거의 형태'가 아니어야 한다.
- 적어도 취사시설이 따로 있으면 안 된다.

## 1-2. 토지의 개별성과 불완전경쟁시장
> 💡 개별성 → 상품 이질성 → 불완전경쟁시장

### 1) 개별성이 시장에 미치는 영향
- 토지의 개별성은 부동산시장을 불완전경쟁시장으로 만드는 요인이다.

### 2) 원리 (원문 메모 보존)
```
완전경쟁시장의 요건 중 하나는 상품의 동질성인데 이는 재화가 표준화되어 서로 완전 대체 가능하다는 것을 의미한다. 따라서 토지의 개별성은 서로가 완전대체가 안되게 되어 완정경쟁시장이 되지 못하도록 만드는 요건이다.
```

## 1-3. 시장수요곡선의 도출
> 💡 시장수요곡선 = 개별수요곡선의 '수평' 합산

### 1) 도출 방법
- 부동산시장수요곡선은 개별수요곡선을 수평으로 합하여 도출한다.

### 2) 계산 논리 (원문 메모 보존)
```
같은 가격 P에서
  → 개인 A의 수요량 QA + 개인 B의 수요량 QB + 개인 C의 수요량 QC + ...
  → 이걸 다 더한 게 그 가격에서의 시장 전체 수요량
  → 여러 가격 수준에서 이 작업을 반복 → 점들을 이으면 시장수요곡선
  = 수평 합산 (가격 고정, 수량을 더함)
```

## 1-4. 부동산의 측면 구분과 준부동산
> 💡 유형적 = 물리적 측면 / 준부동산 = 법률적 측면

### 1) 유형적(물리적) 측면
- 유형적 측면 = 외형을 의미 → 법률적 측면이 아니라 물리적 측면이다.

### 2) 준부동산 (의제부동산)
- 자동차, 광업재단, 어업권, 광업권 등 = 부동산처럼 취급되는 동산·권리.
- 즉 준부동산은 '법률적 측면'의 구분 개념이다.

## 1-5. 정착물이 아닌 것 (동산으로 보는 것)
> 💡 임시로 놓거나 심은 것 = 동산

### 1) 동산 취급 사례
- 가식(임시로 심어놓음) 중인 수목 → 동산에 속한다.
- 정원석(정원 위에 올려둔 장식용 돌) → 그냥 동산으로 본다.

## 1-6. 토지의 용도 전환 — 이행지 vs 후보지
> 💡 소분류 내부 이행 = 이행지 / 대분류 상호 전환 = 후보지

### 1) 이행지
- 공업지가 주거지로 전환되는 것 = 택지 내부의 소분류 변화 → 중간단계 토지는 이행지이다.

### 2) 후보지
- 임지지역·농지지역·택지지역 등 용도지역 상호 간의 용도가 변화되고 있는 지역 내의 토지이다.

## 1-7. 수요량과 소비량
### 1) 개념 관계
- (원문) 수요량 감소와 소비량 감소는 같은 말이다.

---

## 📌 확인 필요
- **수요량 = 소비량 (1-7):** 엄밀히는 수요량(사전적·'구매하고자 하는' 의도 개념)과 소비량/거래량(사후적·'실제 소비한' 실현 개념)은 구분된다. 시장 균형에서는 결과가 일치하지만, "같은 말"로 단정하면 *부동산 수요 = 사전적·유효수요* 성격을 묻는 함정 지문에 걸릴 수 있음 → 사용 교재 표현 기준 재확인 권장.
- **정원석 → 동산 (1-5):** 통설로 통용되나 교재에 따라 정착물/동산 예시 목록이 갈릴 수 있으니, 시험 대비용으로는 사용 중인 교재의 정착물 판단 예시를 기준으로 재확인.
