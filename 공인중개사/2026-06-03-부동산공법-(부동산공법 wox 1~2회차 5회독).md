# 📌 1. 부동산공법 — WOX 1~2회차 5회독 정리

---

## 1-1. 광역도시계획

> 💡 광역계획권 지정권자는 국장·도지사만, 수립 절차는 입안→공청회→협의→심의→승인 순이다.

### 1) 공청회 주체
- 국장, 시도지사, 시장, 군수는 광역도시계획 수립 전 미리 공청회를 열어 주민·관계전문가 의견을 들어야 한다.

### 2) 광역계획권 지정권자와 수립 주체
- 광역계획권이 **둘 이상의 도**에 걸쳐 있는 경우 → 해당 **도지사들이 공동으로** 광역계획권을 지정
- 광역계획권이 **한 도의 관할범위 안**에 있는 경우 → 관할 **시장·군수가 공동으로** 광역도시계획 수립
- 지정권자는 **국장·도지사뿐** → 광역시장은 지정 권한 없음

### 3) 도지사의 단독 수립 가능 여부
- 도지사가 시장 또는 군수의 요청으로 **관할 시장·군수와 공동으로** 수립하는 경우 → **국장의 승인 불요**
- 이유: 도지사는 원래 시장·군수 입안의 **승인권자**이므로, 승인권자 본인은 별도 승인을 받지 않아도 됨

### 4) 국장이 수립 후 공고하는 방법
- 국장이 수립하였을 때 → **직접 공고하지 않고** 시도지사에게 관련 서류를 송부 → **시도지사가** 내용을 공고

### 5) 변경 절차
- 변경은 수립과 **동일한 절차** 적용
- 변경 시 관계 시도군 의회, 관계 시장·군수 의견을 미리 들어야 함

## 1-2. 도시군기본계획

> 💡 심의·협의는 승인권자(도지사)의 절차이고, 시장·군수는 신청만 한다.

### 1) 승인 절차 분리 원칙
- 시장·군수는 도시군기본계획 변경 시 도지사에게 **신청**만 함
- 관계기관 협의, 지방도시계획위원회 심의는 **도지사**가 받음
- 지방도시계획위원회 심의는 **승인권자의 절차**임을 기억

### 2) 인접 지역 포함 수립
- 시장·군수는 지역여건상 필요하면 **인접한 시 또는 군의 관할구역 전부 또는 일부를 포함**하여 도시군기본계획을 수립할 수 있음

## 1-3. 도시군관리계획

> 💡 결정 효력은 지형도면 고시일 당일 발생, 성장관리·개발밀도·기반시설부담구역은 관리계획 결정 대상 아님.

### 1) 결정 효력 발생 시점
- 지형도면을 **고시한 날**부터 효력 발생 (다음날 ❌)
```
"다음날 효력" = 투기 창구를 하루 열어주는 것으로 볼 수 있음
→ 당일 효력 발생이 원칙
```

### 2) 관리계획 결정 대상에서 제외되는 구역
- 성장관리계획구역, 개발밀도관리구역, 기반시설부담구역 → **도시군관리계획으로 결정하지 않음** (이용파트 소속)

### 3) 주민 입안 제안 시 토지소유자 동의율
```
내 땅이 이득 보는 제안 → 2/3 (낮은 문턱)
남의 땅을 끌어들이는 제안 → 4/5 (높은 문턱)
도로·공원·학교 등이 내 땅 위에 들어옴,
토지소유자의 재산권 행사 방식 자체가 복잡하게 제한되고,
다른 소유자와 권리관계가 얽히는 구조가 됨
```
- **기반시설·입체복합구역만 4/5**, 나머지는 2/3

### 4) 수산자원보호구역
- 수산자원보호구역 지정에 관한 도시군관리계획 결정권자 → **해양수산부장관(해장)** (국토교통부장관 ❌)

### 5) 주민 의견청취 생략 가능 경우
- **도시지역의 축소**에 따른 용도지역 변경을 내용으로 하는 관리계획 입안 → 의견청취 생략 가능
```
도시지역 축소
= 이미 도시 기능을 잃어가는 땅
= 현실을 반영한 행정적 정리
        ↓
새롭게 규제를 추가하거나
개발을 제한하는 게 아님
        ↓
현상 추인(현실을 그대로 인정)에 가까운 조치
→ 의견청취 실익이 낮음
```

## 1-4. 용도지역

> 💡 도시지역 미지정 시 보전녹지지역 용적률 적용, 일반주거지역은 저·중·중고층으로 구분.

### 1) 일반주거지역 층수 구분
- 1종 일반주거지역 → **저층**
- 2종 일반주거지역 → **중층**
- 3종 일반주거지역 → **중고층**

### 2) 도시지역 세부 미지정 시 용적률 적용 기준
- 세부 용도지역으로 **지정되지 않은 경우** → **보전녹지지역**에 관한 용적률 규정 적용
```
도시지역
├── 주거지역 (1종전용 / 2종전용 / 1종일반 / 2종일반 / 3종일반 / 준주거)
├── 상업지역 (근린 / 유통 / 일반 / 중심)
├── 공업지역 (전용 / 일반 / 준)
└── 녹지지역 (보전 / 생산 / 자연)

→ 난개발 방지를 위해 가장 규제가 센 보전녹지지역으로 일단 적용
```

## 1-5. 건폐율 특별규정

> 💡 취락지구 60%, 농공단지 70%(기반시설 충분 시 80%), 공업지역 산업단지 80%.

| 구분 | 건폐율 상한 |

| 개발진흥지구 (도시지역 외) | 40% 이하 |
| 개발진흥지구 (계획관리지역 산업·유통) | 60% 이하 |
| 개발진흥지구 (자연녹지지역) | 30% 이하 |
| 수산자원보호구역 | 40% 이하 |
| 자연공원 | 60% 이하 |
| 농공단지 | 70% (기반시설 충분 시 80%) |
| 공업지역 산업단지 | 80% |

## 1-6. 용도지구

> 💡 복합용도지구는 추가 허용(일일계), 특정용도제한지구는 추가 제한으로 반대 개념이다.

### 1) 복합용도지구
- 지정 가능 용도지역: **일반주거지역, 일반공업지역, 계획관리지역** (암기코드: **일일계**)
```
복합용도지구 = "섞어도 되는데, 아직 덜 섞인 곳"

너무 순수하게 보호해야 하는 곳 → ❌ (전용주거·전용공업·보전관리)
이미 다 섞여있는 곳 → ❌ (상업지역·준주거)
딱 중간 지점 → ✅ (일반주거·일반공업·계획관리)
```
- 일반주거지역 복합용도지구 안 건축 불가 항목: **마관동 공장물** (준주거지역 허용 건축물 중 예외)
- 일반주거지역에 지정된 복합용도지구 안에서는 **장례시설 건축 불가**

### 2) 복합용도지구 vs 특정용도제한지구 비교

| 구분 | 복합용도지구 | 특정용도제한지구 |
|---|---|---|
| 기능 | 용도 **추가 허용** | 용도 **추가 제한** |
| 이유 | 용도지역 규제 완화 필요 | 지역 특성상 원래 허용 용도도 차단 필요 |

### 3) 방재지구·경관지구 세분

| 지구 | 세분 |
|---|---|
| 방재지구 | 자연방재지구, 시가지방재지구 (2종) |
| 경관지구 | 자연경관지구, 시가지경관지구, 특화경관지구 (암기: **자시특**, 3종) |

### 4) 용도지구 지정권자
```
원칙: 용도지구 지정권자 = 특별시장·광역시장·특별자치시장·
                          특별자치도지사·시장·군수
        ↓
대도시 시장(인구 50만 이상)은
도지사 승인 없이 직접 지정 가능
```

## 1-7. 시가화조정구역

> 💡 유보기간 종료 다음날 실효, 시행은 중앙행정기관장 요청 + 국장의 지장 없음 확인이 전제.

### 1) 실효 시점
- 유보기간이 끝난 **다음날** 실효 (고시한 다음날 ❌)

### 2) 구역 내 사업 시행 요건
- 주민 요청 ❌ → **중앙행정기관장이 요청**하고, **국장이 지장이 없다고 인정**한 경우에만 가능

### 3) 허용 행위
- 입목의 벌채, 조림, 육림 행위 → **허가를 받아** 할 수 있음

### 4) 도시군계획사업 시행
- 대통령령으로 정하는 사업만 시행 가능
- 그 외 행위는 법령이 허용하는 행위에 한하여 **6명(국토부장관)의 허가**를 받아 시행

## 1-8. 도시혁신구역 · 공동구 · 도시군계획시설

> 💡 도시혁신구역은 공주미·학교·공원·주택을 따로 정할 수 있고, 공동구는 가스관·하수도관만 심의 필요.

### 1) 도시혁신구역
- 다른 법률에도 불구하고 **공주미(공개공지, 부설주차장, 미술작품 설치), 학교, 공원, 주택**에 관해 따로 정할 수 있음

### 2) 공동구 수용 심의 대상
- 공동구 설치 시 **가스관과 하수도관만** 공동구협의회의 심의를 거쳐야 수용 가능

### 3) 도시군계획시설 결정 실효
- 고시일부터 **20년이 지날 때까지** 시설사업이 시행되지 않으면
- 20년이 되는 날 ❌ → **20년이 되는 날의 다음날**에 효력을 잃음
