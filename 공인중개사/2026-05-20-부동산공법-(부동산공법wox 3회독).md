# 📌 1. 부동산공법 3회독 핵심 정리

---

## 1-1. 용도지역·지구 구분

> 💡 상업지역 4종 + 취락지구 3종 분류를 혼동 없이 구분할 것

### 1) 상업지역 4종 (중일유근)

| 지역 | 목적 |
|------|------|
| 중심상업지역 | 도심·부도심의 상업기능 및 업무기능 확충 |
| 일반상업지역 | 일반적인 상업기능 및 업무기능 담당 |
| 유통상업지역 | 도시 내·외 유통기능 증진 |
| 근린상업지역 | 근린지역의 일용품 공급 |

- **도심·부도심 상업·업무기능 확충 = 중심상업지역** (일반상업지역 아님)

### 2) 보호지구 내 취락지구 3종 비교

| 구분 | 특징 |
|------|------|
| 자연취락지구 | 녹지·관리·농림·자연환경보전지역 내 마을 정비 |
| 보호취락지구 | 자연취락지구보다 녹지·관리·농지 내 주거기능 강화 형태 |
| 집단취락지구 | 개발제한구역 내 집단주거지역 대상 |

### 3) 생태계보호지구

- 야생동물서식처 등 생태적으로 보존가치가 큰 지역의 보호와 존존을 위해 지정

### 4) 자연취락지구 건축 제한

- 4층 이하 건축 가능
- 노래연습장 건축 가능 (노래는 부를 수 있다)
- **건축 불가**: 휴게시설, 제과점, 일반음식점, 단란주점, 안마시술소 (암기: **휴제일단안**), 정신병원

|------|--------|

## 1-3. 복합용도지구 내 금지 용도

> 💡 일반주거 = 마관동공장물 / 일반공업 = 노란파마

### 1) 일반주거지역 내 복합용도지구 금지 (암기: **마관동공장물**)

- 안마시술소, 관람장, 동물 및 식물관련시설, 공장, 장례시설, 위험물저장 및 처리시설

### 2) 일반공업지역 내 복합용도지구 금지 (암기: **노란파마**)

- 노유자시설, 단란주점, 아파트, 안마시술소

## 1-4. 시가화조정구역

> 💡 자가관리용 건축물 증축 시 기존 면적 '포함'하여 10평 이하

- 자가관리용 건축물을 증축하려면 기존 건축물 면적을 **포함**하여 10평 이하여야 함

## 1-5. 도시·군계획시설

> 💡 계획시설은 토지에 결정되는 것 — 건물 소유자 해제 신청 불가

### 1) 해제 입안 신청 자격

- 시설결정 고시일부터 10년 이내 사업 미시행 시:
  - **지목이 대(垈)인 토지 소유자만** 해제 입안 신청 가능
  - 건물만 소유한 자는 신청 불가 (계획시설은 토지에 결정)

### 2) 도시·군계획시설 결정 없이 설치 가능 (암기: **주사장이 종일 비맞으면 서있다**)

- 주차장, 사회복지시설, 장사시설, 종합의료시설, 빗물저장 및 이용시설, 폐차장

### 3) 계획시설 실효 고시 의무자

| 구분 | 내용 |
|------|------|
| 결정권자 | 국장, 시도지사, 대도시시장 |
| 실효 고시 의무자 | 결정권자와 동일 (국장·시도지사·대도시시장) |
| 시장·군수 | 수립 권한은 있으나 인가·고시 의무자 아님 → 실효 고시 의무 없음 |

- 도정법 등으로 도시·군관리계획 의제 시 → **2년 이내** 단계별 집행계획 수립 의무

## 1-6. 지구단위계획구역

> 💡 의무 지정 vs 재량 지정을 반드시 구분

### 1) 의무적 지정 대상

| 요건 | 내용 |
|------|------|
| 정비구역·택지개발지구 사업 완료 후 10년 경과 | 암기: 정택이는 10년 지난 친구 |
| 녹지지역 → 주·상·공업지역 30만㎡ 이상 변경 | — |

### 2) 재량적 지정 대상

- 개발제한구역, 도시자연공원구역, 시가화조정구역, 공원에서 **해제되는 구역** (암기: 공개도시 + 해제)
- 일반주거지역, 준주거지역, 상업지역, 준공업지역 (암기: **일준이와 상준이**)

## 1-7. 개발행위 제한·밀도관리·기반시설부담

> 💡 기반시설부담구역 기간 = 기의 ㅣ (1년) / 비용 부과 = 기2반시설(2개월)

### 1) 개발행위 제한 기간

- 시도지사: 기반시설부담구역 지정 지역에 대해 **최대 5년** 개발행위허가 제한 가능

### 2) 개발밀도관리구역

- 변경고시: 해당 지방자치단체 공보 게재 방식
- **농림지역은 지정 대상 제외**

### 3) 기반시설부담구역 주요 기간

| 구분 | 기간 |
|------|------|
| 구역 지정 후 부담계획 미수립 시 해제 간주 | 지정고시일부터 **1년** (암기: 기의 ㅣ) |
| 건축허가일로부터 비용 부과 기한 | **2개월 이내** (암기: 기2반시설) |
| 납부 시기 | 사용승인 신청 전 납부 후 사용승인 수령 |

### 4) 기반시설설치비용 부과 대상

- 연면적 **200㎡ 초과** 건축물 신축·증축
- 철거 후 신축: 기존 연면적 **초과분**에 한해 부과
- 부담구역 내 개발행위허가자 전원 부과 아님

### 5) 기반시설유발계수 해당 시설 (암기: **위관이가 종일숙캐면 병원간다는 것을 방송으로 알려라**)

- 위락시설, 관광휴게시설, 제2종 근린생활시설, 종교시설, 제1종 근린생활시설, 숙박시설, 의료시설, 방송통신시설

## 1-8. 도시개발법 주요 내용

> 💡 토지상환채권 발행계획에 보증 관련 사항 포함 여부 주의

### 1) 토지상환채권

| 항목 | 내용 |
|------|------|
| 포함 사항 | 토지가격 추산방법, 상환대상지역·토지 용도 |
| 미포함 사항 | 한국토지주택공사(LH) 발행 시 보증기관·보증내용 (공공기관은 보증 불필요) |
| 담보 | 질권의 목적으로 할 수 있음 |

### 2) 환지계획 평가액 기준

- 토지·건축물 평가액: **최초 환지계획 인가 시** 기준 적용 → 신청 시 기준 아님

### 3) 환지처분 관련

- 환지를 정하지 않은 종전 토지의 권리: **환지처분 공고일이 끝나는 때** 소멸
- 등기는 공고일 이후 사후적으로 진행

### 4) 구역 지정 후 개발계획 수립 가능 경우 (암기: **세순임밖**)

- 세부목록, 순환개발, 임대주택건설계획 등 세입자 주거·생활 안정 대책, 구역 밖 기반시설 설치
- 주거·상업·공업지역 면적 합계가 전체 면적의 **30% 이하**인 경우

### 5) 환지방식 시행 시 등기 통지

- 등기소에 **사업의 명칭과 목적만** 통지 (도시·군관리계획 결정 내용 불필요)

### 6) 타 시도 비용 부담

- 지정권자가 시행자일 때, 이득 얻는 다른 시도: 최대 **1/2 범위**에서 비용 부담 가능 (전부 부담 불가)

## 1-9. 도시 및 주거환경정비법 (도정법)

> 💡 도정법 타당성검토 = 기본계획 수립권자 (국장 아님)

### 1) 단계별 정비사업추진계획 경미한 변경

- 도지사 승인 불필요, 대도시 시장도 불필요

### 2) 기본계획 타당성 검토

- 검토 주체: **기본계획 수립권자** (국토교통부장관 아님 — 도정법 특수사항 주의)

### 3) 재건축진단

- 실시 요청 동의 요건: 소유자 **1/10 이상**
  - 사유: 재건축진단은 "사업 추진" 결정이 아닌 "건물 노후도 검사" 수준이므로 1/10으로 충분

### 4) 조합 관련

| 항목 | 내용 |
|------|------|
| 이사 | 조합에만 존재 |
| 임원 임기 | 3년, 연임 가능 |

### 5) 국민주택규모 주택 인수 불가 시

- 시도지사는 LH가 아닌 **국토교통부장관**에게 인수자 지정 요청

### 6) 관리처분계획인가 타당성 검증

- 조합원 **1/5 이상**이 인가 신청일부터 **15일 이내** 요청 시
- 시장·군수는 공공기관에 타당성 검증 요청 의무 발생

## 1-10. 주택법 관련

> 💡 주택상환사채 = 국장 승인 + 자본금 5억 원 이상 법인 등록사업자

### 1) 등록 의무

- 연간 **20호 이상** 단독주택 건설사업 시행자
- 연간 **1만㎡ 이상** 대지조성사업 시행자
- → 국토교통부장관에게 등록 필요

### 2) 조정대상지역·투기과열지구 해제 요청 비교

| 구분 | 해제 요청 가능 여부 |
|------|---------------------|
| 조정대상지역 | 시장·군수·구청장이 국장에게 해제 요청 가능 |
| 투기과열지구 | 시장·군수·구청장의 해제 요청 불가 |

### 3) 건축 관련 기준

- 건축물 높이 **3m 이하** 범위에서 증축: 신고 대상
- 용도변경 바닥면적 합계 **100㎡ 이상**: 사용승인 규정 준용 신고 필요
