# 📌 1. 취득세

## 1-1. 과세대상

> 💡 열거주의 — 목록에 없으면 과세대상 아님

### 1) 과세대상 포함 여부

- 광업권·어업권·양식업권·골프회원권·콘도미니엄·승마회원권 등 → 취득세 과세대상 O
- 등기된 임차권 → 채권(부동산 자체를 지배하는 권리 아님)이므로 취득세 과세대상 X

### 2) 공동주택 개수 비과세 특례

- 취득 당시 시가표준액 9억원 이하 공동주택 개수 → 비과세
- 대수선은 금액 무관 무조건 과세

---

## 1-2. 취득세 표준세율

> 💡 일단삼 농지도삼 외사 / 상속받은 38선을 외팔오 / 원시팔을 분할해서 삼

### 1) 유상취득

| 대상 | 표준세율 |
|------|--------|
| 주택 (6억 이하) | 1/100 |
| 주택 (6억 초과 ~ 9억 이하) | (취득가액 × 2/3억 − 3) × 1/100 |
| 주택 (9억 초과) | 3/100 |
| 농지 | 3/100 |
| 그 외 부동산 | 4/100 |

### 2) 무상취득

| 구분 | 대상 | 표준세율 |
|------|------|--------|
| 상속 | 농지 | 2.3% |
| 상속 | 농지 외 부동산 | 2.8% |
| 상속 외 (증여 등) | 비영리사업자 | 2.8% |
| 상속 외 (증여 등) | 비영리사업자 외 | 3.5% |

### 3) 원시취득 / 기타

| 구분 | 표준세율 |
|------|--------|
| 원시취득 (건축 등) — 비영리사업자 | 2.8% |
| 공유물·합유물·총유물 분할 (본인 지분 이내) | 2.3% |

---

## 1-3. 과세표준 관련 특이사항

> 💡 할인받은 금액·채권 매각차손의 포함 여부를 구분할 것

### 1) 사실상취득가격 제외·포함 항목

- 취득대금 일시금 지급 할인금액 → 사실상취득가격에 포함하지 않음
- 국민주택채권 매각차손 → 취득을 위해 어쩔 수 없이 지출한 비용이므로 과세표준에 포함

---

## 1-4. 세율 특례 (표준세율-중과기준세율 vs 중과기준세율)

> 💡 환상속에서 합병, 분할, 이전해서 이혼으로 입목을 등기한다 / 개수하고 지목변경한 과점주주가 묘지를 임시로 보존·이전해서 정원을 조성·설치했다

| 구분 | 세부 내용 | 표준세율-중과기준세율 | 중과기준세율 |
|------|---------|:-------:|:---------:|
| 환매등기 | | O | |
| 상속 | 1가구 1주택 및 부속토지 | O | |
| 상속 | 감면대상 농지 | O | |
| 합병 | | O | |
| 공유물 분할 | 본인 지분 이내 | O | |
| 건축물 이전 | 종전 가액 이내 | O | |
| 이혼 | 재산분할 | O | |
| 입목 | | O | |
| 개수 | 종전 면적 이내 | O | |
| 지목변경 | | | O |
| 과점주주 | | | O |
| 토지 (묘지) | | | O |
| 임시건축물 | 1년 초과 | | O |
| 등록면허세 | 보존·이전등기 외 | | O |
| 정원 등 조성·설치 | | | O |

- 대도시 법인 설립 등에 따른 중과세: 영리법인 + 비영리법인 모두 포함

---

## 1-5. 납세의무 성립시기

> 💡 취득세는 원칙적으로 '가장 빠른 날'에 성립

### 1) 유상승계취득

- 사실상 잔금지급일 vs. 등기·등록일 중 빠른 날
- ❗ "사실상의 잔금지급일이 납세의무 성립시기" → 비교 없이 단독으로 쓰면 틀림

### 2) 재건축조합이 취득하는 토지 (조합원 미귀속분)

- 소유권 고시일 다음날 성립
- 납세의무자: 재건축 조합
- 주택조합은 사용검사를 받은 날 / 재건축은 소유권이전고시 다음날 — 구분 주의

### 3) 지목변경

- 사실상 변경된 날 vs. 공부상 변경된 날 중 빠른 날
- 지목변경일 이전 사용 부분 → 사실상 사용일에 성립

---

## 1-6. 납세 절차 및 가산세

> 💡 등기접수일까지 납부 → 영수증 지참 → 등기

### 1) 신고납부 기한 (무상취득·상속)

- 무상·부담부 증여: 취득일이 속하는 달의 말일부터 3개월 이내 (증말삼)
- 상속: 상속개시일이 속하는 달의 말일부터 6개월 이내 / 외국 9개월 이내 (상말육)

### 2) 미등기 전매

- 신고 없이 매각(미등기 전매) 시 → 80% 세율 부과
- 지목변경은 미등기 전매 중과 사유에서 제외

### 3) 국가의 과세물건 매각

- 국가가 매각 시 → 30일 이내 지자체장에게 신고

### 4) 법인 장부 불이행 가산세

- 법인의 장부 작성·보존 의무 불이행 시 → 취득당시가액의 10% 가산세

### 5) 취득세와 등록면허세 관계

- 취득세 납부 시 등록면허세는 별도 납부 불필요

# 📌 3. 재산세

## 3-1. 과세표준 계산 방식

> 💡 공정시장가액비율은 법령에서 이미 수치로 정해져 있으므로, 선지에 수치만 나와도 동일하게 판단한다.

### 1) 공정시장가액비율 적용 구조

- 토지·건축물: 시가표준액 × 100분의 70
- 주택: 시가표준액 × 100분의 60 (단독주택, 공동주택 불문)
- 과세표준 상한 특례는 주택만 해당 — 토지·건축물에는 적용 안 됨

---

## 3-2. 재산세 표준세율표

> 💡 암기 코드: 농지 "행운을 0.7", 공장 "2차생산", 사치성 "죽을4(40)", 종이오(2~5), 별이사(2~4), 주택은 일단 사고봐 (1~4) 선주 성은 오씨(50)

### 1) 토지

| 구분 | 세율 |
|---|---|
| 농지(전·답·과수원), 목장용지, 임야 | 1천분의 0.7 |
| 공장용지 등 | 1천분의 2 |
| 회원제 골프장·고급오락장용 토지 | 1천분의 40 |
| 종합합산 | 1천분의 2 ~ 1천분의 5 |
| 별도합산 | 1천분의 2 ~ 1천분의 4 |

### 2) 건축물

| 구분 | 세율 |
|---|---|
| 회원제 골프장·고급오락장용 건축물 | 1천분의 40 |
| 시(읍·면지역 제외) 주거지역 등 공장용 건축물 | 1천분의 5 |
| 그 밖의 건축물 | 1천분의 2.5 |

### 3) 주택

| 구분 | 세율 |
|---|---|
| 9억원 이하 + 1세대 1주택 | 1천분의 0.5 ~ 1천분의 3.5 |
| 그 외 주택 | 1천분의 1 ~ 1천분의 4 |

### 4) 선박·항공기

| 구분 | 세율 |
|---|---|
| 고급선박 | 1천분의 50 |
| 그 밖의 선박·항공기 | 1천분의 3 |

---

## 3-3. 재산세 과세대상 토지 분류

> 💡 저율분리=생산 / 고율분리=사치 / 종합합산=투기 / 별도합산=사업

### 1) 분류 기준 정리

| 과세유형 | 이미지 | 해당 예시 |
|---|---|---|
| 저율분리과세 | 생산목적 | 농지(군·읍면·도시밖·개발제한·녹지), 공장용지(군·읍면·산단·공업지역), 임야 |
| 고율분리과세 | 사치성 | 회원제 골프장·고급오락장 |
| 종합합산과세 | 투기목적 | 허가·사용승인 없는 주거용 건물 부속토지, 나대지 등 |
| 별도합산과세 | 사업목적 | 공장용지(그 외), 상가 부속토지 등 |

### 2) 특수 사례

- 허가·사용승인 없는 주택: 주택 아님 → 건축물+토지 각각 과세, 토지는 종합합산과세
- 통제보호구역 내 전·답·과수원·대지: 수익 발생 또는 고가 토지이므로 과세

---

## 3-4. 재산세 특례 규정

> 💡 납부특례 3가지(20만원 이하 합산납부 / 징수면제 2천원 미만 / 납부유예·물납)는 주택과 기타 과세대상을 구분해서 암기

### 1) 20만원 이하 합산납부

- 주택만 해당 — 세액 20만원 이하 시 조례로 납기를 7월 16일~7월 31일로 하여 한꺼번에 부과·징수 가능

### 2) 소액 징수면제

- 고지서 1장당 재산세 징수세액 2천원 미만이면 징수하지 않음

### 3) 납부유예 요건 (3가지 동시 충족)

- 1세대 1주택 (시가표준액 9억 이상 포함) + 납부기한 만료 3일 전까지 신청 + 담보 제공

### 4) 물납 (재산세만 가능)

- 납부세액 1천만원 초과 + 관할구역 부동산으로만 납부
- 신청·변경: 납부기한 10일 전 / 불허가 통지받은 날부터 10일 이내 재신청 가능
- 부동산 평가액 기준: 과세기준일 현재 시가

### 5) 재산세 신고 의무

- 소유권 변동·과세대상 변동 사유 발생 후 등기등록 미완료 → 과세기준일로부터 30일 이내 지자체장에게 신고
- 신탁재산 수탁자 → 과세기준일부터 15일 이내 신고

# 📌 4. 종합부동산세

## 4-1. 종합부동산세 과세표준 계산 구조

> 💡 공시가격에서 일정 금액을 차감 후 공정시장가액비율을 곱함 — "일정 금액 이상에만 부과"하겠다는 취지

### 1) 주택

| 구분 | 공제금액 | 공정시장가액비율 | 세부담 상한 |
|---|---|---|---|
| 개인 일반 | 9억원 | 100분의 60 | 100분의 150 |
| 1세대 1주택 (단독소유·거주자) | 12억원 | 100분의 60 | 100분의 150 |
| 법인·단체 (비례세율 적용) | 0원 | 100분의 60 | 없음 |

### 2) 토지

| 구분 | 공제금액 | 공정시장가액비율 | 세부담 상한 |
|---|---|---|---|
| 종합합산 (5억원 초과분) | 5억원 | 100분의 100 | 100분의 150 |
| 별도합산 (80억원 초과분) | 80억원 | 100분의 100 | 100분의 150 |

- 건축물은 종부세 과세대상 아님

---

## 4-2. 종합부동산세 세율표

> 💡 암기: 개인 주택 오이칠(527) 오오(550) ~ 명품을 하루 종일 삼(13) 별도로 옻칠(57)함 / 법인은 최고세율 비례세

### 1) 주택

| 구분 | 세율 |
|---|---|
| 개인 2주택 이하 | 1천분의 5 ~ 1천분의 27 (7단계 초과누진세율) |
| 개인 3주택 이상 | 1천분의 5 ~ 1천분의 50 (7단계 초과누진세율) |
| 법인 2주택 이하 | 1천분의 27 (단일 비례세율) |
| 법인 3주택 이상 | 1천분의 50 (단일 비례세율) |

### 2) 토지

| 구분 | 세율 |
|---|---|
| 종합합산 | 1천분의 10 ~ 1천분의 30 (3단계 초과누진세율) |
| 별도합산 | 1천분의 5 ~ 1천분의 7 (3단계 초과누진세율) |

---

## 4-3. 납세지 · 분납 · 납부유예

> 💡 종합부동산세는 법인이 아닌 단체면 개인으로 보아 거주지 기준 납세지 적용

### 1) 납세지 결정

- 원칙: 개인 → 주소지, 법인 → 법인 소재지
- 비거주자 개인: ① 국내사업장 소재지 → ② 국내원천소득 발생 장소 → ③ 주택·토지 소재지 순
- 공익법인과 법인으로 보지 않는 단체는 개인을 준용

### 2) 분납

- 납부세액 400만원 초과 시 분납 가능
- 최대 150만원까지 (500만원 이하일 때는 해당 세액에서 250만원 차감한 금액까지) 납부기한 경과 후 6개월 이내 분납

### 3) 납부유예

- 1세대 1주택 + 납부기한 만료 전까지 신청 + 담보 제공
- 시가표준액 9억원 초과 주택도 포함

### 4) 1세대 1주택 판정 제외 사례

- 다른 주택의 부속토지만 소유 / 일시적 2주택 / 상속주택 / 지방저가주택
- 위 항목에서 1세대 1주택자 판정받으려면 9월 16일~9월 30일 관할세무서장에게 신청 필요

# 📌 5. 양도소득세

## 5-1. 양도·취득시기 판정

> 💡 취득세 vs 양도소득세, 시기 판정 기준이 다르다

### 1) 취득·양도시기 암기법

- 취득세: "상상으로 증여계약을 점등한다" → 상속·상환·증여·계약·점유·등기
- 양도소득세: "상상 증증 점점 무당이 되어간다" → 상속·상환·증증(증여2)·점점(점유2)·무효·당사자약정
- 무효판결이 있는 경우 → 당초 취득일을 취득시기로 봄

---

## 5-2. 양도차익 계산

> 💡 양도차익 = 양도가액 − 필요경비 (취득가액 + 자본적지출액 + 양도비)

### 1) 환산취득가액 적용 시 필요경비

- 취득가액 불명 시 환산취득가액으로 대체
- 필요경비 = 다음 두 값 중 더 큰 금액 선택
  - ① 환산취득가액 + 필요경비개산공제액 (등기: 취득당시 기준시가 × 3/100, 미등기: × 3/1000)
  - ② 자본적지출액 + 양도비

### 2) 국민주택규모 초과 건물 취득가액으로 감정가액 사용 시

- 5년 이내 양도 → 감정가액(또는 취득가액)의 100분의 5를 가산세로 양도소득 결정세액에 가산

---

## 5-3. 장기보유특별공제

> 💡 보유기간이 길수록 공제율 ↑, 단 적용 제외 대상 주의

### 1) 적용 대상·제외

- 토지·건물 외 조합원입주권도 보유기간 3년 이상이면 적용
- 미등기 건물 → 절대 불가
- 조정대상지역 내 1세대 2주택 이상 → 적용 불가

---

## 5-4. 1세대 1주택 비과세 기본 요건

> 💡 양도일 현재 국내 1주택 + 2년 보유 (조정대상지역: 2년 거주 추가)

### 1) 비과세 기본 요건 (「소득세법」 제89조 제1항 제3호)

- 1세대 + 양도일 현재 국내 1주택 + 2년 이상 보유
- 조정대상지역 소재 주택: 2년 이상 거주 요건 추가
- 국외주택은 주택 수 계산에 영향 없음

### 2) 겸용주택·고가주택 처리

- 겸용주택(일반): 주택 연면적 > 주택 외 연면적이면 전부 주택으로 인정
- 겸용주택(고가주택): 조건 무관, 주택 부분만 주택으로 인정
- 고가주택 판정 기준: 기준시가 ✗ → 양도 당시 실거래가 기준 (「소득세법 시행령」 제160조)
- 고가주택(양도가액 12억 초과) → 12억 초과 비율만큼 양도차익 안분하여 과세

---

## 5-5. 1세대 1주택 비과세 — 보유·거주 제한 배제 특례

> 💡 해외이주·이사·임차거주 등 불가피한 경우 2년 요건을 면제한다

### 1) 해외이주 특례 (「소득세법 시행령」 제154조 제1항 제2호 가목)

- 출국일로부터 2년 이내 주택 양도 시 보유·거주기간 제한 없이 비과세

### 2) 타 시군 이전 특례 (「소득세법 시행령」 제154조 제1항 제2호 나목)

- 취학·근무·질병요양으로 다른 시·군으로 이전 시 해당 주택에서 1년 이상 거주 입증 시 비과세

### 3) 건설임대주택 임차 특례 (「소득세법 시행령」 제154조 제1항 제3호)

- 건설임대주택 임차일~양도일까지 거주기간 5년 이상이면 보유·거주 제한 배제

---

## 5-6. 다주택 비과세 특례 (일시적 2주택)

> 💡 비자발적 취득(상속·혼인·귀농)은 원래 주택의 비과세를 보호한다

### 1) 상속주택 특례 (「소득세법 시행령」 제155조 제2항)

- 상속 당시 보유한 일반주택을 양도 → 비과세 적용
- 상속받은 주택을 양도 → 1세대 1주택 비과세 미적용

### 2) 혼인 특례 (「소득세법 시행령」 제155조 제5항)

- 혼인한 날부터 10년 이내에 먼저 양도하는 주택 → 비과세

### 3) 귀농주택 특례 (「소득세법 시행령」 제155조 제7항)

- 귀농주택 취득일부터 5년 이내 일반주택 양도 → 비과세
- 부득이한 사유로 취득한 수도권 밖 주택 + 일반주택 → 3년 이내 일반주택 양도 시 비과세

---

## 5-7. 거주·보유기간 통산

> 💡 멸실 후 재건축하거나 비거주자가 거주자로 전환해도 기간을 합산한다

### 1) 멸실 후 재건축 (「소득세법 시행령」 제154조 제5항)

- 멸실 전 주택 + 재건축 주택의 거주·보유기간 통산

### 2) 비거주자 → 거주자 전환 (「소득세법 시행령」 제154조 제6항)

- 비거주자로서 해당 주택 3년 이상 보유·거주 후 거주자로 전환 시 해당 기간 통산

## 5-8. 양도소득세 세율 구조

> 💡 자산 유형 + 보유기간에 따라 세율이 완전히 달라진다

### 1) 세율 요약

| 자산 유형 | 보유기간 | 세율 |
|---|---|---|
| 미등기양도자산 | 무관 | 70% |
| 주택·조합원입주권 | 1년 미만 | 70% |
| 주택·조합원입주권 | 1년 이상~2년 미만 | 60% |
| 주택·조합원입주권 | 2년 이상 | 6~45% 초과누진 |
| 그 외 자산 | 1년 미만 | 50% |
| 그 외 자산 | 1년 이상~2년 미만 | 40% |
| 그 외 자산 | 2년 이상 | 6~45% 초과누진 |

- 초과누진세율 8단계: 6%~45% / 1단계: 과세표준 1,400만원 이하 → 6%

## 5-9. 비사업용토지

> 💡 비사업용토지 = 투기 목적 토지, 중과세율 적용 대상이다

### 1) 개념 및 예외 (「소득세법」 제104조의3)

- 비사업용토지: 사업에 직접 사용하지 않는 투기 목적의 토지

## 5-11. 거주자 판단 기준 및 공동소유

> 💡 183일 이상 거소 = 거주자 / 공동소유는 각자 별도 납세

### 1) 거주자 판단 (「소득세법」 제1조의2 제1항)

- 1과세기간(1년) 중 국내에 183일 이상 거소를 두면 거주자
- 183일은 연속 거주 불요 — 누계 합산으로 판단

### 2) 공동소유 자산의 납세의무 (「소득세법」 제2조의3)

- 연대납부 의무 ✗ → 각자 자기 지분에 대해 별도 납세의무 부담

## 6-5. 지방세 가산세 종류와 계산

> 💡 무신고·과소신고는 세액 기준 %, 납부지연은 일수·세액 기반 계산

### 1) 가산세 유형별 세율

| 유형 | 가산세율 |
|---|---|
| 일반 무신고 | 무신고납부세액 × 20% |
| 부정 무신고 | 무신고납부세액 × 40% |
| 일반 과소신고 | 과소신고납부세액 × 10% |
| 부정 과소신고 | 과소신고납부세액 × 40% + 일반분 × 10% |

### 2) 납부지연 가산세

- 신고납부 방식 (취득세 등): 미납세액 × (22/100,000) × 미납일수, 한도 75%
- 납세고지서 방식 (재산세 등): 미납세액 × 3% (즉시 부과) + 미납세액 × (66/10,000) × 경과 개월수 / 단, 45만원 이하 미적용
