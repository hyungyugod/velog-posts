# 📌 1. 무권대리 ~ 취소

## 1-1. 무권대리의 구조와 효력

> 💡 무권대리는 '유동적 무효' 상태 — 추인 전까지 상대방은 철회 가능

### 1) 무권대리의 기본 구조
- 대리권 없는 을이 갑을 대리하여 선의의 병과 매매계약을 체결 → 계약은 유동적 무효 상태
- 병은 갑 또는 을 어느 쪽에 철회를 주장해도 매도계약을 소급소멸시킬 수 있음
- 단, 갑이 추인하기 전에(추인한 사실을 알게 되기 전까지) 철회해야 함 → 추인으로 계약이 유효해지면 철회 불가

### 2) 추인거절권과 확정적 무효
- 갑의 추인거절권 = 갑이 추인권을 행사하지 않겠다는 의사표시
- 추인거절 시 유동적 무효 → 확정적 무효로 전환
- 무권대리는 계약(채권행위) 단계의 문제 — 만약 등기(물권행위)까지 넘어갔다면 무효인 등기의 말소 문제로 전환되어 전혀 다른 법리가 적용됨

### 3) 추인의 소급효와 제3자 보호
- 추인은 다른 의사표시가 없을 때 계약 시에 소급하여 효력 발생 (「민법」 제133조)
- 단, 제3자의 권리를 해하지 못함 → 추인으로 계약은 유효해지나, 제3자의 소유권을 빼앗아 이행할 수는 없음

### 4) 악의 상대방과 무권대리인의 책임
- 악의의 상대방은 무권대리인에게 계약 이행 청구 불가 → 을의 책임은 "병을 속인 것"에 대한 제재이므로 알면서 뛰어든 자는 보호 불가
- 제한능력자는 무권대리인의 이행·손해배상 책임 없음 (「민법」 제135조 제2항)
- 병이 철회 후 을에게 지급한 계약금 → 을에게 부당이득반환 청구해야 함 (갑에게 청구 불가 — 갑과 을 사이에 유효한 대리권 수여 없었으므로)

## 1-2. 표현대리

> 💡 표현대리 = 무권대리를 유효하게 만드는 제도 — 거래 상대방의 신뢰 보호 목적

### 1) 표현대리의 목적과 주장 자격
- 표현대리 제도의 목적 = 거래 상대방의 신뢰 보호
- 표현대리 성립은 상대방만 주장 가능

### 2) 기본대리권의 범위
- 사자(심부름꾼)도 기본대리권으로 인정 → 웬만한 대리권은 권한을 넘은 표현대리의 기본대리권이 될 수 있음 (「민법」 제126조)
- 예: 갑이 을에게 저당권설정 대리를 맡겼는데 을이 월권하여 병에게 토지를 매도 → 표현대리 성립 안 하면 무권대리 → 갑이 추인 안 하면 무효 → 이전등기 청구 불가

### 3) 표현대리의 한계
- 계약 자체가 이미 확정 무효라면 표현대리 법리 적용 불가 — 유효하게 만들 대상 자체가 없기 때문

## 1-3. 무효·취소·철회·해제 비교

> 💡 대상·시점·주체·원인이 각각 다른 4개의 법적 수단

| 구분 | 무효 | 취소 | 철회 | 해제 |
|---|---|---|---|---|
| **대상** | 법률행위 | 법률행위 | 의사표시 | 계약 |
| **시점** | 처음부터 | 소급하여 | 장래를 향해 | 소급하여 |
| **누가** | 법이 자동 | 취소권자 | 철회권자 | 해제권자 |
| **원인** | 강행규정 위반 등 | 착오·사기·강박·제한능력 | 도달 전 의사표시 / 무권대리 특칙 | 채무불이행 등 |
| **원상회복** | 부당이득반환 | 부당이득반환 | 부당이득반환 | 원상회복의무 (제548조) |
| **추인 가능** | 원칙 불가 | O | X | X |
| **주장 기간** | 제한 없음 | 3년/10년 | 추인 전까지 | 제한 없음 |

## 1-4. 취소

> 💡 취소는 상대방에게 하는 것 — 전득자에게 직접 주장 불가 (단, 미성년자 보호 우선)

### 1) 취소의 상대방과 제3자 효과
- 취소는 상대방에게 하는 것이지 전득자에게 주장하는 것이 아님
- 미성년자 케이스: 미성년자 보호가 제3자 보호보다 우선 → 취소 시 전득자도 소유권 취득 불가

### 2) 취소권 행사 기간
- 추인할 수 있는 날로부터 3년 / 법률행위한 날로부터 10년 이내 행사 (「민법」 제146조)

### 3) 법정추인 사유 (「민법」 제145조)

| 호 | 사유 |
|---|---|
| 1호 | 이행 |
| 2호 | 이행의 청구 |
| 3호 | 경개 (기존 채무 소멸 + 새 채무 성립 계약) |
| 4호 | 담보의 제공 |
| 5호 | 취소할 수 있는 행위로 취득한 권리의 전부·일부 양도 |
| 6호 | 강제집행 |

## 1-5. 일부무효와 해약금

> 💡 일부무효 판단 기준 = 당사자의 가정적 의사

### 1) 일부무효의 판단 기준
- 행위의 일부가 무효일 때 나머지 부분의 유효성은 당사자의 가정적 의사를 고려하여 판단
- 가정적 의사 = "무효 부분이 없었다는 걸 처음부터 알았다면 나머지만으로도 계약했을까"라는 가상의 의도

### 2) 해약금과 이행의 착수
- 해약금으로서 계약금 지급 시: 매수인은 계약금 포기, 매도인은 배액 상환으로 해제 가능 — 단 당사자 일방이 이행에 착수한 때에는 불가 (「민법」 제565조)
- 토지거래허가구역에서 허가 신청 = 계약상 채무를 직접 실현하는 이행의 착수가 아님 → 해약금 해제 여전히 가능
