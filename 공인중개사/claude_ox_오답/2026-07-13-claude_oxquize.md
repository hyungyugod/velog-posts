## 🔑 복습 큐 핵심 암기 포인트 5장

*(지난주 신규 오답이 0이라 이번 5장은 복습 큐 개념의 원장 샘플에서 뽑았다 — 숫자·바꿔치기 함정 중심)*

1. **개발행위허가 제한 = 3년 + 1회 2년 = 최장 5년.** 연장 시엔 도시계획위원회 심의 X. 연장 가능 지역 = 관리계획 수립 중·지구단위계획구역·기반시설부담구역. **개발밀도관리구역은 빠진다**(규제 강화 장치라 OUT).
2. **공간재구조화계획 결정권자 = 시·도지사 + 국토교통부장관.** 일반 도시·군관리계획과 달리 **대도시 시장은 OUT, 국장 IN**(국가급 큰 그림이라).
3. **거미집이론:** 수렴 = 수요탄력성 > 공급탄력성 / 발산 = 수요탄력성 < 공급탄력성(공급이 더 탄력적일 때 진동↑). 수렴 조건을 그대로 둔 보기가 함정.
4. **하향여과 = 저가주택 수요가 증가할 때 발생**(저소득층이 노후·저가화된 옛 고소득주택을 사용). 주거분리는 정(+)의 외부효과를 좇는 고소득가구가 주도.
5. **등록사항정정 근거서류:** 토지표시(경계·면적) = 측량성과도·인접소유자 승낙서·확정판결서 / **소유자표시 = 등기필증·등기사항증명서.** 둘의 근거가 다르다는 게 핵심.

## 💡 콘텐츠 소재 후보 3

새 오답은 없지만, 밀린 복습 큐에서 카드뉴스 각이 서는 개념 3개. 셋 다 **그림 한 장으로 설명되는 시각형**이라 미감 무기 쓰기 좋다.

1. **개발행위허가 제한 (상습·2회 틀림)** — 훅: *"개발을 막을 수 있는 기간, 최대 몇 년일까?"* → 3년 + 2년 = 5년 타임라인 한 장 + '개발밀도관리구역 vs 기반시설부담구역' 이름 함정.
2. **거미집이론 수렴 vs 발산** — 훅: *"부동산 가격은 왜 균형으로 안 가고 점점 요동칠까?"* → 거미집 다이어그램 + 탄력성 대소 비교(수렴/발산 두 컷).
3. **하향여과·주거분리** — 훅: *"낡은 부잣집은 결국 누구에게 흘러갈까?"* → 화살표 한 방향으로 흐르는 여과 도식 + 정(+)의 외부효과.
