## 🔑 복습 큐 핵심 암기카드 5장

*(지난주 신규 오답이 0이라 이번 5장도 복습 큐 개념의 원장 샘플에서 뽑았다 — 3주째 같은 5장인 이유는 재도전이 안 돌아 큐가 안 바뀌기 때문. 이 5개는 지금 확실히 외워두고 재도전 때 맞히면 바로 큐에서 빠진다.)*

1. **개발행위허가 제한 = 3년 + 1회 2년 = 최장 5년.** 연장 시엔 도시계획위원회 심의 **X**. 연장 가능 지역 = 관리계획 수립 중·지구단위계획구역·기반시설부담구역. **개발밀도관리구역은 빠진다**(규제 강화 장치라 OUT). ↔ 이름 비슷한 '기반시설부담구역(비용 부과)'과 바꿔치기가 단골 함정.
2. **공간재구조화계획 결정권자 = 시·도지사 + 국토교통부장관.** 일반 도시·군관리계획과 달리 **대도시 시장은 OUT, 국장 IN**(국가급 큰 그림이라).
3. **거미집이론:** 수렴 = 수요탄력성 **>** 공급탄력성 / 발산 = 수요탄력성 **<** 공급탄력성(공급이 더 탄력적일수록 진동↑). 수렴 조건을 슬쩍 뒤집은 보기가 함정.
4. **하향여과 = 저가주택 수요가 증가할 때 발생**(저소득층이 노후·저가화된 옛 고소득주택으로 이동). 주거분리는 정(+)의 외부효과를 좇는 고소득가구가 주도.
5. **등록사항정정 근거서류:** 토지표시(경계·면적) = 측량성과도·인접소유자 승낙서·확정판결서 / **소유자표시 = 등기필증·등기사항증명서.** 두 갈래의 근거서류가 다르다는 게 핵심.

## 💡 콘텐츠 소재 후보 3

새 오답은 없지만, 밀린 복습 큐에서 카드뉴스 각이 서는 개념 3개. 셋 다 **그림 한 장으로 설명되는 시각형**이라 미감 무기 쓰기 좋다. 데이터가 비는 이번 주야말로 밀린 개념으로 **발행을 밀어내기 좋은 타이밍** — 학습을 콘텐츠 자산으로 바꾸는 변환 사이클을 여기서 돌려라.

1. **개발행위허가 제한 (상습·2회 틀림)** — 훅: *"개발을 막을 수 있는 기간, 최대 몇 년일까?"* → 3년 + 2년 = 5년 타임라인 한 장 + '개발밀도관리구역 vs 기반시설부담구역' 이름 함정 대비표.
2. **거미집이론 수렴 vs 발산** — 훅: *"부동산 가격은 왜 균형으로 안 가고 점점 요동칠까?"* → 거미집 다이어그램 + 탄력성 대소 비교(수렴/발산 두 컷).
3. **등록사항정정 근거서류** — 훅: *"지적도 면적이 틀렸을 때, 뭘 들고 가야 고쳐줄까?"* → 토지표시 vs 소유자표시 2갈래 근거서류 분기도(측량성과도 라인 / 등기필증 라인).
