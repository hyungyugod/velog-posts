## 🟦 1회차 오답 (5문제 / 20문제 중)

### 1. 용도지역·지구·구역의 중첩 구조

**문제** — 용도지역은 하나의 토지에 단 하나만 지정되며, 용도지구나 용도구역과 중첩될 수 없다.

**정답** — X

**해설**
- **용도지역끼리**는 중복 불가 ⭕ (예: 같은 필지에 주거지역+상업지역 동시 지정 X)
- **용도지역 ↔ 용도지구 ↔ 용도구역**은 같은 토지에 셋이 동시 적용 가능
- 구조: 용도지역(기본 성격) → 용도지구(추가 규제) → 용도구역(정책 브레이크) 층층이 쌓임

**함정 키워드** — "중첩될 수 없다" 단정 표현이 함정 신호

### 2. 복합용도지구 지정 지역 (일일계)

**문제** — 복합용도지구는 일반주거지역, 일반공업지역, 계획관리지역에만 지정할 수 있다.

**정답** — O

**해설**
- **암기코드: 일일계** = 일반주거 · 일반공업 · 계획관리
- 단일 용도만 쓰기 아까운 땅에 혼합 허용
  - 역세권 일반주거 → 1층 카페·사무실 + 위층 주거
  - 외곽 계획관리 → 물류창고 + 근생 + 소규모 주거 혼합

**함정 키워드** — "~에만"이라는 한정 표현이 들어와도 진짜 한정인 경우가 있음. 일일계는 진짜 3곳만.

### 3. 그린벨트 안 취락지구 (집단 ≠ 자연)

**문제** — 그린벨트(개발제한구역) 안의 기존 마을에는 자연취락지구가 지정된다.

**정답** — X

**해설**
- **집단취락지구** → 개발제한구역(그린벨트)
- **자연취락지구·보호취락지구** → 녹지지역
- 이름이 비슷해서 헷갈리는 전형적 함정

**함정 키워드** — "집단 = 그린벨트 안 모여 사는 마을"

### 9. 개발행위허가 제한 기간

**문제** — 개발행위허가 제한 기간은 최대 5년(원칙 3년 + 1회 2년 연장)이다.

**정답** — O

**해설**
- **최대 5년 = 3년(원칙) + 2년(1회 연장)**
- 처음 3년만 도시계획위원회 심의 거침, 연장 시 추가 심의 불요
- 추가로 기반시설부담구역은 한 번 더 연장 가능

**함정 키워드** — 숫자 정답인데 "함정인가" 의심하다 X로 가는 의심병

### 11. 도시개발조합 동의 요건 (숫자 뒤집기)

**문제** — 도시개발조합 설립인가의 동의 요건은 토지면적 1/2 이상 소유자 + 토지소유자 수 2/3 이상 동의이다.

**정답** — X

**해설**
- 정답: **토지면적 2/3 이상 + 토지소유자 수 1/2 이상**
- 노트 그대로 숫자가 거꾸로 들어간 함정

**암기 팁** — **"땅이 더 무겁다 → 면적 2/3, 사람 1/2"**

### 5. 복합용도지구 — "일반상업" 함정

**문제** — 복합용도지구는 일반주거지역·일반상업지역·계획관리지역에 지정할 수 있다.

**정답** — X

**해설**
- 정답은 **일일계 = 일반주거 · 일반공업 · 계획관리**
- "일반상업"으로 살짝 바꾼 전형적 함정
- 상업지역은 이미 다용도 활용 가능 → 굳이 복합용도지구로 묶을 필요 X
- 일반공업지역은 외곽 산업지대라 주거·근생 혼합 필요

**함정 키워드** — "일일계에 일반상업 끼면 무조건 X"

### 6. 도시 외 개발진흥지구 건폐율

**문제** — 도시지역 외 지역에 지정된 개발진흥지구의 건폐율 상한은 40% 이하이다.

**정답** — O

**해설**
- **개발진흥지구 건폐율 3단**
  - 도시 내: **30%**
  - 도시 외: **40%**
  - 산업·유통: **60%**
- "도시 안 빡빡(30) → 밖 좀 풀어줌(40) → 산업유통은 더 풀어줌(60)"

**함정 키워드** — 정답인데 의심병으로 X 가는 패턴

### 7. 산업·유통 개발진흥지구 건폐율 (80% 함정)

**문제** — 산업·유통 개발진흥지구의 건폐율 상한은 80% 이하이다.

**정답** — X

**해설**
- 정답은 **60% 이하**
- 80%는 **국가·일반 산업단지** 등의 건폐율
- 비슷한 이름·숫자로 헷갈리게 만든 함정

| 구분 | 건폐율 |
|---|---|
| 개발진흥(도시 내) | 30% |
| 개발진흥(도시 외) | 40% |
| 산업·유통 개발진흥 | **60%** |
| 수산자원보호구역 | 40% |
| 자연공원 | 60% |
| 농공단지 | 70% |
| 국가·일반 산업단지 | **80%** |

**함정 키워드** — "산업진흥지구 ≠ 산업단지" 칼같이 분리

### 8. 공간재구조화계획 결정권자 (대도시시장 OUT)

**문제** — 공간재구조화계획(도시혁신구역·복합용도구역)의 결정권자는 시·도지사와 대도시시장이다.

**정답** — X

**해설**
- **공간재구조화계획 결정권자: 시·도지사 + 국토교통부장관**
- 일반 도시·군관리계획은 "시·도지사 + 대도시시장"인데, **공간재구조화는 대도시시장 빠지고 국장 들어옴**
- 도시혁신구역·복합용도구역이 국가급 큰 그림이라 대도시시장 손에 안 맡김

**암기** — **"공간재구조화 = 대도시시장 OUT, 국장 IN"**

### 10. 개발행위허가 제한 연장 가능 지역

**문제** — 개발행위허가 제한 연장이 가능한 지역은 도시·군관리계획 수립 중인 지역, 지구단위계획구역, 개발밀도관리구역이다.

**정답** — X

**해설**
- 정답: **관리계획 수립 중 + 지구단위계획구역 + 기반시설부담구역**
- "개발밀도관리구역"이 아니라 "기반시설부담구역"
- 둘 다 "구역" 붙어서 헷갈리는데 성격 정반대
  - **개발밀도관리구역**: 도로 곤란 → 건폐율·용적률 **강화(제한)** 장치
  - **기반시설부담구역**: 도로 새로 깔 곳 → 비용 **부과** 장치
- 연장 사유는 "곧 새 인프라 깔 곳"이라 **기반시설부담구역**이 들어가야 논리상 맞음

**함정 키워드** — 비슷한 이름 쌍 함정

### 14. 재개발 조합 동의 요건 (4321)

**문제** — 재개발사업의 조합설립인가 동의 요건은 토지등소유자 3/4 이상 + 토지면적 1/2 이상의 동의이다.

**정답** — O

**해설**
- **암기코드: 4321 (재개발 = 사람 3/4, 땅 1/2)**
- 조합 동의 요건 비교표

| 사업 | 사람(소유자 수) | 땅(토지면적) |
|---|---|---|
| 도시개발조합 | 1/2 | 2/3 |
| 재개발 조합 | **3/4** | **1/2** |
| 재건축 주택단지 | 동별 과반수 + 전체 **3/4** | **3/4** |
| 재건축 주택단지 외 | **3/4** | **2/3** |

**함정 키워드** — 1회차 11번 트라우마로 자동 X 가는 의심병

### 15. 주택단지 재건축 조합 (전원 동의 함정)

**문제** — 주택단지에서 시행하는 재건축사업의 조합설립인가는 토지등소유자 전원의 동의가 필요하다.

**정답** — X

**해설**
- **전원 동의는 어디에도 없음** — 알박기 한 명에 막히면 안 되니까
- 정답: **각 동별 구분소유자 과반수 + 전체 3/4 이상 + 토지면적 3/4 이상**
- 암기: "재건축 주택단지 = 770(전체 3/4 + 면적 3/4) + 동별 과반수"

**함정 키워드** — "전원 동의" 같은 극단 표현은 십중팔구 X

### 18. 매수 거부 시 설치 가능 시설

**문제** — 매수청구를 받은 매수의무자가 매수하지 않기로 결정한 경우, 토지소유자는 단독주택·제1·2종 근린생활시설(일부 제외) 및 공작물을 설치할 수 있다.

**정답** — O

**해설**
- 노트 그대로: "매수 안 한다 결정 → **단독주택, 1근린, 2근린(놀자놀자 빼고), 공작물 설치 가능**"
- 논리: 매수의무자가 안 사겠다면 토지소유자 입장 너무 손해 → 최소한 작은 건물은 짓게 해줘야 형평
- 단, **유흥성 시설(놀자놀자 = 단란주점·노래연습장 등)은 제외**

**함정 키워드** — "안 사주면 작은 건물은 짓게 해준다" 직관

## 📊 통합 패턴 분석 (1·2회차)

| 약점 패턴 | 1회차 틀린 수 | 2회차 틀린 수 | 누적 |
|---|---|---|---|
| 숫자 함정 (스왑·근접 숫자) | 1개 | 2개 | 3개 |
| 의심병(정답인데 X로) | 1개 | 3개 | 4개 ⚠️ |
| 권한자/주체 혼동 | 0개 | 2개 | 2개 |
| 단정적 표현 함정 | 1개 | 1개 | 2개 |
| 비슷한 이름 함정 | 1개 | 1개 | 2개 |

**가장 큰 적신호: 의심병 (4건)** — 기본기 진술인데 "함정인가" 의심하다 X로 뒤집는 패턴

**핵심 대응**
1. 기본기 자신 있는 진술은 그대로 답 — 의심은 숫자·이름이 어색할 때만
2. 3개 한 묶음 암기 강화 (일일계, 4321, 30·40·60, 시도지사+국장)
3. 비슷한 이름 쌍 비교 카드 만들기 (개발밀도 vs 기반시설부담 / 산업진흥 vs 산업단지 / 자연취락 vs 집단취락)

## 🔑 핵심 암기카드 5장

1. **일일계** — 복합용도지구 = 일반주거 · 일반공업 · 계획관리
2. **4321** — 재개발 조합 = 사람 3/4, 땅 1/2
3. **30·40·60** — 개발진흥지구 건폐율 = 도시 내 / 도시 외 / 산업·유통
4. **시도지사 + 국장** — 공간재구조화 결정권자 (대도시시장 OUT)
5. **3·2 = 5** — 개발행위허가 제한 = 3년 + 2년 연장 = 최대 5년
