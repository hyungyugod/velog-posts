# 📌 0. 엑셀 함수 총정리
## 0-1. 통계함수

| 함수                                  | 설명                       |
| ----------------------------------- | ------------------------ |
| **MAX(인수1, 인수2, …)**                | 인수 중 최대값(논리값/텍스트 제외)     |
| **MAXA(인수, …)**                     | 인수 중 최대값(논리값/텍스트 포함)     |
| **MIN(인수, …)**                      | 인수 중 최소값(논리값/텍스트 제외)     |
| **MINA(인수, …)**                     | 인수 중 최소값(논리값/텍스트 포함)     |
| **LARGE(범위, n번째)**                  | 범위의 n번째로 큰 값             |
| **SMALL(범위, n번째)**                  | 범위의 n번째로 작은 값            |
| **MEDIAN(인수, …)**                   | 중간값                      |
| **MODE.SNGL(인수…)**                  | 가장 자주 발생하는 값(최빈수)        |
| **VAR.S(인수…)**                      | 표본 분산                    |
| **STDEV.S(인수…)**                    | 표본 표준편차                  |
| **RANK.EQ(인수, 범위, 정렬 방식)**          | 순위 구함 (0=내림차순 / 1=오름차순)  |
| **AVERAGE(인수…)**                    | 평균                       |
| **AVERAGEA(인수…)**                   | 텍스트/논리값 포함 평균            |
| **AVERAGEIF(범위, 조건, 실제 계산 범위)**     | 조건 충족 값들의 평균             |
| **AVERAGEIFS(범위, 조건 범위1, 조건1, …)**  | 여러 조건 충족 값들의 평균          |
| **COUNT(범위)**                       | 숫자 개수                    |
| **COUNTA(범위)**                      | 비어있지 않은 셀 개수             |
| **COUNTIF(범위, 조건)**                 | 조건에 맞는 셀 개수              |
| **COUNTIFS(범위1, 조건1, 범위2, 조건2, …)** | 여러 조건 충족 개수              |
| **COUNTBLANK(범위)**                  | 빈 셀 개수                   |
| **FREQUENCY(배열1, 배열2)**             | 배열1 값의 구간별 빈도를 수직 배열로 반환 |
| **PERCENTILE.INC(범위, 인수)**          | 0~1 사이 백분위수 반환           |

## 0-7. 정보함수

| 함수              | 의미                       | 반환값                                        |
| --------------- | ------------------------ | ------------------------------------------ |
| CELL(정보유형, 셀주소) | 셀의 서식, 위치, 내용에 대한 정보를 조회 | 다양한 text 정보(row, format, contents, type 등) |
| TYPE(값)         | 값의 데이터 유형을 숫자로 반환        | 숫자 1, 텍스트 2, 논리 4, 오류 16                   |
| ISERR(값)        | **#N/A 제외** 오류면 TRUE     | TRUE/FALSE                                 |
| ISERROR(값)      | 모든 오류면 TRUE              | TRUE/FALSE                                 |
| ISEVEN(값)       | 짝수면 TRUE                 | TRUE/FALSE                                 |
| ISODD(값)        | 홀수면 TRUE                 | TRUE/FALSE                                 |
| ISTEXT(값)       | 텍스트면 TRUE                | TRUE/FALSE                                 |
| ISNONTEXT(값)    | 텍스트가 아니면 TRUE            | TRUE/FALSE                                 |
| ISNUMBER(값)     | 숫자면 TRUE                 | TRUE/FALSE                                 |
| ISLOGICAL(값)    | 논리값이면 TRUE               | TRUE/FALSE                                 |

## 0-8. 재무함수

| 함수                            | 인수                        | 의미                                                 |
| ----------------------------- | ------------------------- | -------------------------------------------------- |
| FV(이자율, 기간, 금액, 현재가치, 납입시점)   | rate, nper, pmt, pv, type | 일정 금액을 일정 이율로 적립할 때 **미래 가치(Future Value)** 계산     |
| PV(이자율, 기간, 금액, 미래가치, 납입시점)   | rate, nper, pmt, fv, type | 미래의 지불·수령 금액에 기반한 **현재 가치(Present Value)** 계산      |
| NPV(할인율, 금액1, 금액2 …)          | rate, value1…             | 할인율과 미래 현금흐름을 사용하여 **순현재가치(Net Present Value)** 계산 |
| PMT(이자율, 기간, 현재가치/미래가치, 납입시점) | rate, nper, pv/fv, type   | 일정 이율이 적용되는 대출의 **매월 상환액** 계산                      |

## 0-9. 데이터베이스 함수 (D- 함수)

| 함수                    | 인수                  | 의미                           |
| --------------------- | ------------------- | ---------------------------- |
| DMAX(범위, 열위치, 조건)     | DB, field, criteria | 조건에 맞는 값 중 **최대값**           |
| DMIN(범위, 열위치, 조건)     | DB, field, criteria | 조건에 맞는 값 중 **최소값**           |
| DSUM(범위, 열위치, 조건)     | DB, field, criteria | 조건에 맞는 값들의 **합계**            |
| DCOUNT(범위, 열위치, 조건)   | DB, field, criteria | 조건에 맞는 숫자 데이터 개수             |
| DCOUNTA(범위, 열위치, 조건)  | DB, field, criteria | 조건에 맞는 **비어 있지 않은** 데이터 개수   |
| DAVERAGE(범위, 열위치, 조건) | DB, field, criteria | 조건에 맞는 값들의 **평균**            |
| DVAR(범위, 열위치, 조건)     | DB, field, criteria | 조건에 맞는 값들의 **표본분산**          |
| DSTDEV(범위, 열위치, 조건)   | DB, field, criteria | 조건에 맞는 값들의 **표본표준편차**        |
| DGET(범위, 열위치, 조건)     | DB, field, criteria | 조건에 맞는 값이 **단 하나이면 그 값을 반환** |
| DPRODUCT(범위, 열위치, 조건) | DB, field, criteria | 조건에 맞는 값들의 **곱**             |
