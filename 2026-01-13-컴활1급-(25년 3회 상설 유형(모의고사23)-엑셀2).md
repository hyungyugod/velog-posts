## 1-1. 외부데이터 가져오기 csv
> 데이터 모델에 이 데이터 추가라는 말이 없으면 csv파일은 피벗테이블 메뉴에서 가져오는게 아니라 데이터 탭에서 외부데이터 원본으로 가져와야한다. -> 데이터 모델에 데이터 추가는 말이 없으면 하지 않는다.
> 이때는 데이터 -> 파일에서를 활용하여 데이터를 가져온다. 이때 특정 열을 삭제해야하는 경우 데이터 변환을 사용해야한다.

## 1-2. 피벗테이블에서 부분합을 2개 설정하는법
> 우선 부분합을 하나 만든뒤에 우클릭하여 필드설정에 들어가 사용자 지정-> 2개 부분 선택 이렇게 진행해야한다.

## 1-3. 필드명이 분할되어있는 경우 정렬, 부분합
> 어떤 필드명은 그대로고 어떤 필드명은 위아래로 쪼개져있으면 전체를 선택한 후 정렬을 할 수 없다. 따라서 일반 데이터들만 선택하여 정렬을 진행하고 내 데이터에 머리글 표시 옵션은 체크해제해야한다.
> 이후 그룹합을 눌러도 위에서와 같은 경고문구가 나오는데 이때는 그냥 진행하면 된다.
> 개요를 표시하지 않는 방법은 그룹해제-개요지우기를 활용한다.

## 1-4. 데이터 통합을 고급필터처럼 사용하기

## 1-6. 조건부서식 vs 사용자 지정 서식
> 조건부서식과 사용자 지정 서식을 구분해야한다.
> 사용자 지정 서식은 [조건]서식;[조건]서식;그외 숫자서식;문자일 때 와 같은 버전과 양수서식; 음수서식; 0서식; 문자일때 서식 이렇게 있다. 이는 셀 서식의 사용자 지정을 사용한다.
> [>=15][자홍]*★#;[>=10]★ #;◎ 이런식으로 조건을 입력하는데 스페이스바를 유의해야한다. (필수로 0이나오려면 0을 입력한다.)
> 두 영역에 한번 지정할때는 ctrl로 두 영역을 선택하고 사용자 지정 서식을 입력하면 된다.
> 사용자 지정 서식과 메크로가 같이 나오는 문제의 경우 범위를 옆에 복사하여 한번 작성해보고 메크로를 작성해보는 것이 좋다.

## 1-7. 프로시저 문제 관련 여러가지 사항
```Private Sub Cmd추가_Click()

If Txt수량 = "" Then
    MsgBox "수량을 넣으시오"

ElseIf Cmb직책.ListIndex = -1 Then
    MsgBox "직책을 선택하시오"
Else
    입력행 = Range("b4").CurrentRegion.Rows.Count + 4
    Cells(입력행, 2) = Txt사번
    Cells(입력행, 3) = Cmb직책
    Cells(입력행, 4) = Txt수량.Value
    Cells(입력행, 5) = Txt접수일자
    Cells(입력행, 6) = CDate(Txt접수일자) + 10

    If Opt남 Then
        Cells(입력행, 7) = "남"
    Else
        Cells(입력행, 7) = "여"
    End If

End If

End Sub
```
> 수량이나 직책을 선택하지 않았을때 입력을 진행하지 않으려면 위처럼 else에 나머지 과정들을 위치시켜서 실행되지 않도록 해야한다.
> 그림 개체는 글자위에 연결되어 있어도 currentregion에 포함되지 않는다.
> 콤보상자 선택값은 그냥 Cmb직책 이렇게만 해도 입력이 된다. 참조행을 사용하는건 목록을 선택했을때만 이다.
> 날짜+10이렇게하면 날짜가 10 더해지는 연산이 수행된다.
