# 📌 1. 부동산학개론 부동산의 개념과 분류
## 1-1. 부동산의 개념
### 1) 복합개념의 분류
- 부동산은 토지와 그 정착물, 동산은 부동산 외의 것들을 의미한다. 이는 법률적 개념임
- 부동산에서 물리적 개념 -> 외형적인
- 물리적 개념 -> 공치환자 - 공간, 위치, 환경, 자연
- 경제적 개념은 소비재와 생산재(생산할 수 있는 요소) -> 토지는 관광상품일때 소비재가 된다.
- 준부동산도 광의 개념에서 부동산
- 토지와 토지 위의 정착물이 결합 -> 복합 부동산

### 2) 정착물의 구분
- 종속되지 않은 독립 정착물에는 건물(등기부가 따로 있음), 임목, 수목 -> 둘다 등록됨, 권원(지상권, 임차권)이 있는 농작물이 있다. -> 따로 거래된다.
- 다년생 식물(계속 자람), 구거(시골물길), 교량, 담장 -> 종속 정착물
- 가식 중 수목(임시), 경작된 수확물, 정원석 -> 동산

### 3) 토지용어
- 택지, 농지, 임지 간의 용도 전환 -> 대분류끼리 이므로 후보지 (대후보)
- 세분된 용도지역 간의 전환 -> 소분류끼리이므로 이행지 (소이행)
- 나지 - 나체땅, 건물 등 정착물이 없고, 사법상의(민법등) 제한을 받지 않는 땅 (공법은 국가의 규칙이라 배제할 수 없다.)
- 부지 - 바닥부
- 건부지 - 건축물의 바닥토지로 이용 중인 토지 -> 이용 중인 것에만 붙이는 용어
- 필지 - 등록단위
- 획지 - 이용의 단위 (가격수준 유사)
- 법지 - 안전상 이유로 법으로 심하게 규제 -> 경사로
- **빈지 - 바닷가 토지**
- 휴한지는 정당하게 쉬고있는 토지이고 유휴지는 바람직하지 못하게 쉬고있는 것이다.
- 포락지 -> 물에 의해 무너져 내림
- 맹지 - 도로가 없어서 맹인 
- 선하지 - 고압선 아래땅
- 소지 - 자연상태 그대로, 만두 소 그대로
- 공지 - 빈공터
- 일단지 - 용도상 묶여있는 토지

## 1-2. 부동산의 분류
### 1) 공동주택과 단독주택
- 주택이란 여러 사람이 다같이 장기간 거주할 수 있는 구조를 의미한다.
- 공동주택은 주거용이 4개층이상
- 연초다이
- 아파트(주거용 5개층 이상) -> 연립주택(660초과) -> 다세대주택(660이하)
- 단독주택은 주거용이 3개층 이하
- 다가구 -> 주택을 쪼개서 임대 (660이하)
- 다중주택 -> 하숙집

# 📌 2. 부동산학개론 부동산의 특성과 수요공급
## 2-1. 부동산의 특성
- 부부영개 용병사
- 부동산의 자연적 특성은 아래와 같다.
- 부동성(부동산은 움직이지 않는다.) - 찌부등 -> 지역화(지역자체가 시장이 된다. 시장에 가져다 팔 수 없기 때문), 외부효과의 근거(가만히 있기때문에 외부 영향을 잘 받는다.)
- 부증성(부동산은 확장되지 않는다.) - ㅈㅈㅊ -> 집약적 토지이용, 최유효이용의 근거 -> 희소자원이기 때문에 알뜰살뜰하게 효율적으로 사용해야 한다.
- 영속성(부동산은 영속한다.) - 감자가수 -> 물리적 감가상각부정(외형변화가 없다.), 자본이득의 근거(가격상승이득), 가치정의(장래기대이익을 가치로 환산한 값), 수익환원법
- 개별성(대체성 유사성의 부정)
- 인문적 특성은 아래와 같다.
- 용도의 다양성, 병합, 분할 가능성, 사회적, 경제적, 행정적 위치의 가변성

## 2-2. 수요공급의 특성
### 1) 수요, 공급 곡선의 기본
- 수요곡선은 소비자의 행동을 조사한 곡선이고 공급곡선은 공급자의 행동을 조사한 곡선이다.
- 수요량의 변화: 시장의 기초는 가격을 기준으로 수량을 조사하고 하여 수요곡선 내부이동은 가격에 의해 q가 변할때 발생한다.
- 수요의 변화: 가격이외의 요인의 변화를 의미하는데 가격을 고정하고 상황이 바뀌었을때 더 살지 안살지로 판단하면 된다.
- 대체관계는 유사하여 둘 중 선택하는 관계이고 보완관계는 수요가 같이 변하는 관계이다.
- LTV와 DTI가 하락하면 대출을 많이 해주지 않는다는 뜻이다.

### 2) 균형변화의 4가지 규칙
- 수요, 공급의 증감과 균형 거래량은 비례한다.
- 균형가격의 변화만 외우면 되는데 수증기(상승), 수감(ㅠㅠ), 공증(나한테 공증을?ㅠㅠ), 공감(상승) 이렇게 외우면 된다.
- 만약 힘의 크기가 주어진 경우는 큰 쪽이 시장을 결정한다.
