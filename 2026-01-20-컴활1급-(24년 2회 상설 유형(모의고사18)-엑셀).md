## 1-1. “▲1,000천원”으로 표시

> [빨강][>=1000000]"▲"#,###,"천원";[녹청][<0]"▼"#,###,"천원";##0,"천원" 에서 문자들 큰따옴표 치는 것과 기본적으로 #으로 표시하고 필수로 0이 나와야하는 자리에만 0으로 채워주는 센스가 필요하다.

## 1-3. worksheetfunction.counta
> worksheetfunction.counta(이름) 을 통해 vba에서도 실제 워크시트에서 이름이라는 변수를 넣어서 counta함수를 적용하고 이 값을 반환할 수 있다. 또한 for을 사용하여 인원수-일차순위만큼의 원을 나타낼 수 있다. 이는 원이 많을 수록 순위가 높다는 그래프를 의미하게 된다.

```
Public Function fn그래프(이름, 일차순위)

인원수 = WorksheetFunction.CountA(이름)

answer = 인원수 - 일차순위
fn그래프 = ""

For t = 1 To answer
fn그래프 = fn그래프 & "●"
Next t

End Function
```

## 1-7. 시나리오
