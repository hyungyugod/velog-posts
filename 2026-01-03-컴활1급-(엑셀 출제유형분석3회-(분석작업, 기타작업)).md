# 📌 1. 엑셀 출제유형분석3회-(분석작업, 기타작업)
## 1-1. 피벗 테이블 보고서 작성하기
- 그룹이 자동으로 작성되어 있는 경우 셀을 하나 클릭하여 그룹 메뉴에서 원하는 그룹만 남겨두어야 한다.
- 열레이블, 행레이블 처럼 필드 명이 나와있으면 이는 압축형식이다.
- 계산필드를 삽입하는 것은 피벗테이블 분석에서 계산 메뉴에서 계산필드를 삽입 할 수 있다. 이는 삭제할때도 이 메뉴에서 삭제해야한다.
- 데이터 영역에 쉼표나 백분율 스타일을 적용하는 것은 홈탭에 표시형식에서 설정할 수 있다. 셀서식에서 설정하는 것이 아니다.
- 문제에서 정렬하라 함은 그 열만 정렬하는 것이 아니라 행 단위로 정렬하라는 것을 의미한다.
- 피벗테이블 옵션에서 레이블이 있는 셀 병합 및 가운데 맞춤을 할 수 있다.
- 디자인 -> 레이아웃에서 총합계나 부분합에 대한 옵션을 걸 수 있다.
- 피벗테이블 옵션 -> 요약및 필터에서 행 총합계를 제거할 수 있다.
- 축의 표시 단위는 축 서식의 표시단위를 변경해주면 된다.

## 1-2. 데이터 통합
- 채울 표에 블록설정한 후 데이터탭, 통합을 클릭하여 통합을 진행한다.
- 함수를 설정하고 통합할 데이터들을 설정한다.
- 첫행, 왼쪽열을 체크하는 것을 잊으면 안된다.
- 셀 내부 내용만 지우는 것은 delete로 하면 된다.
- 합치는 함수를 설정하는 것을 잊으면 안된다. 이걸 헷갈려서 틀렸다.

## 1-3. 데이터 도구 활용하여 중복값 삭제
- 전체 대상 영역을 선택하고 데이터탭의 중복값 제거 메뉴에서 기준 필드를 선택하고 진행한다.

## 1-4. 차트 작성하기
- 범위를 나눠서 설정할때 먼저 블록을 선택하고 나머지 떨어진 블록들은 ctrl을 누르고 선택한다.
- 누적세로 막대와 100% 누적새로막대형의 차이는 막대가 끝까지 채워져있는지 여부이다.
- 차트위치를 바꾸려면 바로가기 메뉴의 차트 이동에서 진행해야 한다.
- 데이터 테이블은 차트요소추가에서 추가할 수 있다.
- 데이터 레이블 자체를 변경하려면 데이터 레이블 자체를 누르고 서식에서 종류를 추가할 수 있다.
- 문제와 무관하면 그림과 달라도 그냥 두어야한다.

## 1-5. 메크로 작성 + 데이터 표
- 데이터표에서는 계산식과 열방향, 행방향 중 어떤 방향으로 계산하는지 입력해주어야한다. 즉 계산식이 열의 값이 원래 계산식이 속한 곳으로 들어가서 계산식이 변경되는 작용이다.
- 그러니까 계산 표를 미리 만들어두고 계산식으로 가져다둔다음 그곳에서 값을 하나씩 계산표에 넣어서 진행하는 셈이다.
- 도형을 만든 후 메크로를 연결한다.
- 데이터 표를 지정할때 최대한 알차게 범위를 선택해야한다 즉 필요없는 정보는 범위에 포함시켜서는 안된다.
- 스페이스바 잘못 누르는거 조심해야한다.

## 1-6. 메크로 작성 + 사용자지정셀서식
- [조건]나올값;[조건]나올값; 이렇게 진행하면 된다.

## 1-7. 프로시저
- 폼을 여는 명령은 폼이름.show이다.
- 개체명.Rowsource를 통해 범위의 값을 집어넣는다.
- columncount는 보여줄 열 개수를 의미하며 속성에서 바꿔도 되나 코딩에서 columncount 필드를 직접 변경해줘도 된다. 
- 입력행 = range("필드위치").currentregion.rows.count + a
- 이후 cells(입력행,열)에 값을 대입한다.
- 목록상자에서 사용자가 선택한 값의 행을 반환하는 함수는 listindex이다. 이때 목록상자의 인덱스는 0부터 시작한다. 보통 이는 참조행이라는 변수에 넣어둔다.
- 이후 대입할때는 참조표.list(참조행, 열) 이렇게 해서 누른 행의 값에서 특정 열 값만 따로 가져올 수 있다. 이때 가져온 값은 숫자여서 val을 굳이 넣지 않아도 된다.\
- 목록상자에서 1번째 항목이 선택되어있게 하려면 목록상자.listindex = 1이고 아무것도 선택이 안되어있게 하려면 목록상자.listindex = 0 이렇게 하면 된다.
- range는 지정된 셀 변경, target은 커서가 있는 곳(해당셀) 변경, cells는 시트 전체 변경이다.
- font.name, font.size, font.bold, font.italic 속성을 코드 내에서 직접 변경할 수 있다.

```vba

Private Sub C등록_Click()


입력행 = Range("b3").CurrentRegion.Rows.Count + 3
참조행 = Lst접수코드.ListIndex

Cells(입력행, 2) = t접수자
Cells(입력행, 3) = t수강개월
Cells(입력행, 4) = Lst접수코드.List(참조행, 0)
Cells(입력행, 5) = Lst접수코드.List(참조행, 1)
Cells(입력행, 6) = Lst접수코드.List(참조행, 3)
Cells(입력행, 7) = Val(t수강개월) * Lst접수코드.List(참조행, 3)

t접수자 = ""
t수강개월 = ""
Lst접수코드.ListIndex = 0

End Sub
```
