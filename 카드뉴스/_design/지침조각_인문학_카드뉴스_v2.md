[프로세스 — 2단계 필수]
1단계: 개요 먼저 제공 (바로 HTML 생성 금지)
  커버 (1장) | 질문형 훅 — "전문가의 정리"가 아니라 "학습자의 질문" (예: "우리는 왜 타인의 시선에 흔들릴까?")
  본문 (3~8장) | 소제목 · 핵심 3줄 요약 + 장별 레이아웃 유형 지정
  요약 (1장) | 핵심 문장 / 해시태그 방향
  레이아웃 | 5종 중 선택: 정의형 / 비교형 / 절차형 / 숫자형 / 사례형 (에세이·인용은 사례형·정의형 우선)
  → 개요 제공 후 사용자 확정("만들어줘") 또는 수정 요청 대기
2단계: 확정 후 HTML 생성 + present_files. 파일명: 지식산책_주제명_v1.html (수정 시 v2, v3)

- 캔버스: 1080×1350 (4:5) 기본 / 5~10장. 핵심 요소는 중앙 3:4 안에 배치(그리드 크롭 대응).
- 테마 humanities (themes/humanities.css): 종이 배경 #FDFAF5 · 브라운 잉크 #5C3D1E · 갈색 획 #8B6343 · 크림 강조 #F5EDD8

- 레이어: 배경 → 콘텐츠(레이아웃 5종 중 1) → chrome 불변(워드마크·페이지번호·푸터)
