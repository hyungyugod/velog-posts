## 1-1. 사용자 정의 함수 작성과 활용
- 새로운 사용자 정의 함수를 만들때는 무조건 모듈을 만들고서 진행해야함을 잊으면 안된다.
- vba에서 if문은 if 조건 then이 기본이고 elseif then과 else, end if 등을 활용해주면 된다.
- end if는 end function처럼 if문을 끝내주는 코드이다.
- 코드 앞에 작음 따옴표를 붙이면 vba에서의 주석처리이다.

```vba
Public Function fn업적(직급, 연구과제, 연구논문, 산학협동, 사회봉사)

If 직급 = "조교수" Then
    fn업적 = "대상자 아님"
    
ElseIf 직급 = "부교수" Then
    fn업적 = 연구과제 * 0.2 + 연구논문 * 0.2 + 산학협동 * 0.3 + 사회봉사 * 0.3

End If

End Function
```
