# 📌 1. vocal preset 뜯어보기
## 1-1. 이펙터 체인
- 기본적으로 믹서의 위에서 아래로 소리가 흐르면서 이펙터가 걸린다.
- 따라서 기본적으로는 보컬 소리 믹서를 위에 배치하고 딜레이와 리버브 같은 공간계 이펙터를 후순위로 배치한다.
```
Auto-Tune Pro ← 1번 (가장 먼저) 
Fruity Limiter (COMP) ← 2번 
Fruity Parametric EQ 2 
Fruity Blood Overdrive 
Fruity Limiter (COMP) 
Fruity Delay 3 
Fruity Reeverb 2 ← 마지막
```
- 1. 튠은 원본 보컬에만 걸려야한다. 뒤에 걸리면 컴프가 만든 다이나믹이나 딜레이, 리버브의 잔향 같은 것들에도 같이 걸리고 그러면 소리가 흔들리고 인공적이 되기 때문이다.
- 2. 튠이 끝난 음정은 음 높이가 안정되어있으나 소리의 크기는 여전히 들쭉날쭉하다. 따라서 이를 보정해주는 작업이 이루어져야 한고 하여 컴프레서로 소리를 눌러준다.
- 3. 다음 볼륨사이즈를 평준화해둔 상태에서 eq를 걸어야 한다. 이유는 컴프가 작은 소리를 키워놓았기 때문에 모든 소리에 eq를 공평하게 걸 수 있기 때문이다.
- 4. 이후 overdrive를 활용해 배음을 추가적으로 생성하여 보컬이 비트 위로 튀어나오도록 한다. (배음이란 기본음의 **정수배(2배, 3배, 4배...)**에 해당하는 주파수들이 함께 울리는 현상 (예: 880Hz, 1320Hz...)으로 우리의 귀는 이 '배음'들이 어떻게 섞여 있느냐에 따라 "이건 피아노 소리구나" 혹은 **"이건 바이올린 소리구나"**라고 구분하게 된다. 즉, 배음은 소리의 **'음색(Tone Color)'**을 결정하는 핵심 요소입이다.) 이때 배음을 eq 뒤에 추가하는 이유는 eq로 필요없는 저역을 이미 잘라냈기 때문에 왜곡이 중요한 대역에만 생기고 만약 eq전에 두면 저역까지 같이 더러워지게 된다.
- 5. EQ + Overdrive로 생긴 튀는 부분을 다시 정리하기 위해 2차 컴프레를 사용한다. 이렇게 되면 보컬을 깔끔하게 정리할 수 있다.
- 6. patcher로 동적 디에싱(“보컬이 특정 주파수에서 세질 때만
그 부분을 자동으로 눌러주는 장치”)을 하고 있다. 앞까지 했을때 톤이 밝아지고 배음이 추가된 상태여서 치찰이 생기기 딱 좋다. 하여 patcher를 통해 디에서를 적용한다.
- Patch: 이어 붙이다. -> 연결 신호를 이어붙여서 특정 주파수에서만 디에서가 작동하도록 설계된 patcher를 지금 사용하고 있다.
- 7. 딜레이를 통해 만들어진 소리를 반복시킨다.
- 8. 만들어진 소리를 특정 공간에 가져다둔다.

## 1-2. autotune pro

### 1-2-1. Tracking
- 목소리를 얼마나 집요하게 추적할지를 의미한다. 
- 낮음 → 자연스럽지만 튠이 살짝 놓칠 수 있음
- 높음 → 작은 흔들림도 끝까지 따라가서 인공적
- → 너무 높으면 S, SH 같은 무성음(성대가 안 울리는 소리다로 목이 아니라 입과 이빨과 공기만으로 나는 소리)까지 음정으로 오해함 → 치찰이 튄다

### 1-2-2. Input Type, Key / Scale (D Major)
- inpot type은 소프라노인지 테너인지 기본을 설정하는 것을 의미하고 Key / Scale은 음계지도로서 Auto-Tune이 “여기만 착지해라”라고 지정하는 스케일을 정하는 카테고리이다.

### 1-2-3. Modern / Classic
- Modern → 빠르고 또렷한 요즘 사운드
- Classic → 느리고 부드러운 옛날 보정

| 항목    | Modern | Classic |
| ----- | ------ | ------- |
| 어택    | 즉각적    | 느림      |
| 음 끝   | 칼같음    | 흐림      |
| 체감 높이 | 살짝 높아짐 | 낮아짐     |
| 공기    | 날아감    | 남음      |
| 로봇 느낌 | 있음     | 거의 없음   |


| 장르           | 추천                    |

| 발라드 / 감성 R&B | Classic               |

### 1-2-4. Formant
- 목소리의 ‘몸 크기’를 바꾼다
- 100 = 원래 체형
- 낮추면 → 남성화 / 두꺼워짐
- 높이면 → 여성화 / 얇아짐
