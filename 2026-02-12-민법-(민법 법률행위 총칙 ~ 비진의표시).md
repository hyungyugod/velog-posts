# 📌 1. 민법 법률행위 총칙 ~ 비진의표시
## 1-1. 단독행위와 계약
- 상대방이 없는 단독행위 (행위자 한사람의 의사표시로만 성립) -> 유언, 유증(유언으로 증여), 재단법인 설립행위, 소유권의 포기
- 계약: 청약과 승낙이라는 두개의 의사표시의 합치

## 1-2. 채권행위(의무부담행위)와 물권행위(처분권리)
- 타인권리매매나 타인권리임대차(채권행위-이행문제발생)는 유효이다. (ex 아버지 집)
- 아때 무권리자의 처분행위를 진정한 권리자가 추인(추후에 인정) -> 소급(처음부터)하여 유효

## 1-3. 불요식행위(의사표시에 방식을 요하지 않음)
- 의사표시 -> 묵시적(행동, ex 그게그말인 말)으로도 가능

## 1-4. 성립요건과 효력요건
- **성립요건과 효력요건의 유효무효 기준시 -> 법률행위 당시**
- 취소사유 -> 제한능력자, 착오, 사기, 강박
- 취소 -> 일단 유효, 취소권자가 취소하면 소급하여 무효

## 1-5. 특별효력요건
- 농지취득자격증명(농지를 취득할 자격이 있는지를 증명) -> 농지매매 효력 발생조건 아님

## 1-6. 행위능력
- **제한 능력자 단독 법률행위는 취소할 수 있다. -> 제한능력자 보호**
- 제한능력자는 이들을 보호하는 제도이다.

## 1-7. 후발적 불능
- 계약했는데 집(목적)이 불탐
- 후발적 불능 -> 법률행위 유효

## 1-8. 강행규정
- 강행규정은 보호해야하는 것들에 대한 배제할 수 없는 규정이다. <-> 임의규정
- 일반적으로 강행규정이라함은 효력규정이며 강행규정에는 효력규정(효력없음)과 단속규정(효력은 있으나 다른 제제)이 있다.
- **강행규정 위반 -> 싹다무효**
- 강행규정 위반에 의해 무효인 경우에도 전환규정이면 -> 유효
- 중간생략등기와 전매 (등기가 두번 이전될때 조세포탈목적)

## 1-9. 중개보수약정
- 초과한 중개보수를 금지하는 것 -> 강행법규(강행규정)
- 위의 경우 초과범위에서 무효이다. -> 초과분 반환청구가능
- 자격없는자가 우연히 1회 거래행위 중개시 -> 유효

## 1-10. 사회적 타당성
- 반사회질서와 불법원인급여는 같은 의미이다. -> 이들은 강행법규와 달리 **반환청구가 불가하다.**
- **반사회질서는 싹다 무효이다. -> 질서보호**
- 첩계약, 해제조건(앞으로 안할거다) 증여계약 -> 무효
- 부첩관계 단절 -> 유효
- 불법동기의 표시, 알려짐 -> 무효
- 강박 -> 반사회적 x (강박은 따로 법률이 있다.)
- 양도소득세 회피, 조세포탈은 기준이 모호함 -> 반사회적 x
- 강제집행 면할 목적으로 허위 -> 반사회적 x
- 허위진술, 과도한 돈을 밥고 증언 -> 무효
- 형사사건 성공보수 약정 -> 무효, 민사사건 성공보수약정 -> 유효
- 비자금 은닉을 위해 임치(맡김) -> 반사회적 x

## 1-11. 부동산이중매매의 법리
- 병이 갑의 배임행위에 '적극가담'(알고도 매도요청) -> 반사회질서행위로 싹다무효
- 을은 갑을 '대위'하여 소유권이전등기 말소를 청구가능
- 을은 직접 말소청구 x, 채권자 취소권도 x
- 싹다 무효이므로 제 3자가 몰랐어도 소유권 취득 x

## 1-12. 불공정한 법률행위 (폭리행위) 요건
- 불공정한 법률행위는 사회질서 위반으로 싹다 무효이다.
- 무경험은 '거래일반'에 대한 전반적 무경험을 의미한다.
- 궁박은 본인을 기준으로 판단한다.
- 폭리의사(피해자 궁박의 사정을 알고 이를 이용함)가 존재해야한다.
- 불공정한 법률행위로의 무효주장은 주장자가 모든 것을 증명해야하므로 현저한 불균형 있어도 궁박같은 사안이 추정(일단 인정)되지 않는다.

## 1-12. 불공정한 법률행위 (폭리행위) 적용범위와 효과
- 증여계약=무상행위 -> 불공정한 법률행위 x
- 경매 -> 불공정한 법률행위 x
- 싹다무효이므로 제 3자에게도 무효주장가능
- 무효인 경우 추인을 해도 유효가 안됨
- 전환을 통해서는 유효로 가능

## 1-13. 비정상적 의사표시규정 공통점
- 의사와 표시의 불일치: 비진의표시(나만 아는 구라), 통정허위표시(같이 아는 구라), 착오(몰랐어)
- 의사와 표시의 일치: 사기, 강박
- 공법상 행위나 소송행위에는 적용되지 않는다.
- 무효나 취소의 효력이 선의의 제 3자에 대항(주장)하지 못한다.

## 1-14. 비진의표시 
- 구라(비진의표시)를 쳐도 효력이 발생하는 것이 원칙
- 비진의표시는 상대방이 알았거나(악의) 알 수 있었으면(과실) 무효인데 상대방이냐 제 3자이냐에 따라 초점이 달라진다.
- 상대방: 선의 이후에 과실(알 수 있었음), 무과실(알라해도 알 수 없었음) -> 무효
- 제 3자: 선의, 악의까지만 따짐 -> 선의였으면 계약이 유효함
- 공무원의 사직의사는 효력이 발생함 -> 공법행위에는 준용x
- 진의는 특정한 내용의 의사표시를 하고자하는 표의자의 생각
- 강박도 주고자 하는 마음은 존재
- 대리권 남용은 겉과 속이 다르다는 점에서 비진의표시에 관한 규정과 같다.
- 명의를 대여하여 대출약정 맺은 경우 즉 대신 대출을 받아준 경우는 비진의 표시에 해당 x
