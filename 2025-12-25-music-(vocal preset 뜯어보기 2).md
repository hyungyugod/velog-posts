# 📌 0. 연 믹싱 중 엔지니어링 문제 해결
## 0-1. 피아노 소리 구조
- 망치가 현을 치는 순간 → 고역이 튄다
- 몸통 공명 → 중저역이 뒤따라온다
- 즉 하이 대역은 파형의 앞부분, 어택을 담당하며 그래서 하이가 커질수록 귀는 “시간 단위가 더 잘게 쪼개졌다고” 인식한다.
- 그래서 피아노 반주의 고음역대가 강조되자 박자가 빠르게 느껴졌던 것이다.
- 그래서 발라드 인트로에서 피아노 하이를 일부러 낮춰두고 후렴에서만 하이를 열어 “속도가 살아난 느낌”을 만드는 경우가 많다.

## 0-2. 시간 기준점(temporal anchor)
- 시간 기준점은 “이번 소리 덩어리의 시작점은 어디지?” 라고 뇌가 판단하는 지점이다.
- 앞의 소리와 뒤의 소리가 오버랩 되었을 때 왜 뒷 소리가 급하게 진입하는 것처럼 느껴질까
- 앞선 소리 덩어리가 끝나지 않았는데 뒤의 소리가 시작되면 앞선 곡에 치고 들어온 것처럼 여겨지게 되는 것이다.
- 또 피아노 소리 구조와 시간 기준점을 엮어서 생각하면 보컬의 하이랑 피아노의 하이가 맞물려서 박자가 엉키는 것처럼 느껴지고 급해져보일 수 있다. 이는 시간 기준점이 빠르게 바뀌는 효과가 있어서 그렇다.

## 0-3. 왜 “하이”가 세면 보컬이 붕 뜰까
- “붕 뜬다”는 것은 보컬이 선명하긴 한데 몸통이 약하고 얇아서 공중에 떠 있는 느낌을 의미한다. 이때 보컬이 공간상 앞에 떠있거나 위에 떠있는 느낌을 받으며 특히 자음(S, T 같은)과 하이(에어)가 튀면서 말만 따로 놀아 “따로 녹음한 것 같다” 느낌을 받게 된다.
- 이때 또 보컬 튠을 세게 걸면 무성음(공기 소리, S/T/SH) 이 더 또렷해지고 피치가 딱 붙으면서 질감이 얇아지고 밝아진다. 결과적으로 보컬의 하이 윤곽선이 더 날카로워진다.
- 이를 해결하기 위해 보컬 튠을 한뒤에 포먼트(Formant)를 살짝 내려서 목소리를 두껍게 할 수 있다.
- 발라드는 보컬이 박자의 기준이 되어야하므로 상대적으로 피아노의 고음대역이 높지 않는게 오히려 더 조화롭다.

## 1-1. autotune pro

### 1-1-1. Tracking
- 목소리를 얼마나 집요하게 추적할지를 의미한다. 
- 낮음 → 자연스럽지만 튠이 살짝 놓칠 수 있음
- 높음 → 작은 흔들림도 끝까지 따라가서 인공적
- → 너무 높으면 S, SH 같은 무성음(성대가 안 울리는 소리다로 목이 아니라 입과 이빨과 공기만으로 나는 소리)까지 음정으로 오해함 → 치찰이 튄다

### 1-1-2. Input Type, Key / Scale (D Major)
- inpot type은 소프라노인지 테너인지 기본을 설정하는 것을 의미하고 Key / Scale은 음계지도로서 Auto-Tune이 “여기만 착지해라”라고 지정하는 스케일을 정하는 카테고리이다.

### 1-1-3. Modern / Classic
- Modern → 빠르고 또렷한 요즘 사운드
- Classic → 느리고 부드러운 옛날 보정

| 항목    | Modern | Classic |
| ----- | ------ | ------- |
| 어택    | 즉각적    | 느림      |
| 음 끝   | 칼같음    | 흐림      |
| 체감 높이 | 살짝 높아짐 | 낮아짐     |
| 공기    | 날아감    | 남음      |
| 로봇 느낌 | 있음     | 거의 없음   |


| 장르           | 추천                    |
| ------------ | --------------------- |
| 랩 / 하이톤 팝    | Modern                |
| 발라드 / 감성 R&B | Classic               |
| 인디 / 로파이     | Classic               |
| K-pop 메인보컬   | Modern + Formant 살짝 ↓ |

### 1-1-4. Formant
- 목소리의 ‘몸 크기’를 바꾼다
- 100 = 원래 체형
- 낮추면 → 남성화 / 두꺼워짐
- 높이면 → 여성화 / 얇아짐

### 1-1-5. Retune Speed (20)
- 음을 얼마나 빨리 붙잡아 끌어올지
- 0~10 → T-Pain 로봇
- 15~30 → 현대 팝/힙합 보컬
- 40 이상 → 자연스럽게

### 1-1-6. Flex-Tune
- “완벽하게 맞으면 그냥 둬라”
- 높을수록 이미 맞는 음은 안 건드림

### 1-1-7. Natural Vibrato
- 원래 떨림 복원량
- 0 → 떨림 제거 (힙합용)
- 10~20 → 발라드
- 30 이상 → 클래식 / R&B

### 1-1-8. Humanize (31)
- 긴 음은 자연스럽게, 짧은 음은 빨리
- → 발음이 살아난다
- → 치찰도 같이 자연스러워진다
