# 📌 0. 가설검정의 두 가지 풀이 (신뢰구간 방법, 검정통계량 방법)

| 방법              | 필요한 값                                                          | 계산식                       | 판단 기준                                       | 특징                            |                 |
| --------------- | -------------------------------------------------------------- | ------------------------- | ------------------------------------------- | ----------------------------- | --------------- |
| **신뢰구간 (CI)**   | - 표본평균 (x̄)<br>- 표준오차 (SE = σ/√n 또는 s/√n)<br>- 임계값 (z 또는 t)    | 신뢰구간 = x̄ ± (임계값 × SE)    | μ₀가 신뢰구간 안 → 기각하지 않음<br>μ₀가 신뢰구간 밖 → 기각     | 구간 단위로 제시 → 직관적, 검정통계량 계산 불필요 |                 |
| **검정통계량 (Z/t)** | - 표본평균 (x̄)<br>- 귀무가설 평균 (μ₀)<br>- 표준오차 (SE)<br>- 임계값 (z 또는 t) | 검정통계량 = (x̄ - μ₀) / SE    | abs(Z) > 임계값 → 기각<br>abs(Z) ≤ 임계값 → 기각하지 않음 | 표본평균과 μ₀ 차이가 SE의 몇 배인지 직접 확인  |                 |
| **p값 방식**       | - 검정통계량 (Z 또는 t)<br>- 분포의 누적확률 값                               | p = P(관측된 검정통계량 이상이 나올 확률 | H₀ 참일 때)                                    | p < α → 기각<br>p ≥ α → 기각하지 않음 | 결과를 “확률”로 해석 가능 |

# 📌 1. 변수 형태별 분석 기법 & 시각화 

| 종속변수(Y)     | 독립변수(X)             | 분석 기법 예시                    | 시각화 예시             |
| ----------- | ------------------- | --------------------------- | ------------------ |
| **연속형**     | X 1개 (연속형)          | 단순 회귀 (Simple Regression)   | 산점도 + 회귀선          |
| **연속형**     | X 여러 개 (연속형/범주형 혼합) | 다중 회귀 (Multiple Regression) | 산점도 행렬, 회귀계수 시각화   |
| **연속형**     | X 1개 (범주형, 2집단)     | t-검정 (t-test)               | 박스플롯, 막대그래프(평균+오차) |
| **연속형**     | X 1개 (범주형, 3집단 이상)  | 분산분석 (ANOVA)                | 박스플롯, 평균 막대그래프     |
| **범주형(이항)** | X 1개 이상 (연속/범주)     | 로지스틱 회귀, 의사결정나무, SVM        | 로지스틱 곡선, 분류 트리     |
| **범주형(다항)** | X 1개 이상             | 다항 로지스틱 회귀, 다중 분류 모델        | 다중 분류 ROC, 혼동행렬    |
| **범주형**     | X 1개 (범주형)          | 카이제곱 검정 (Chi-square Test)   | 교차표, 모자이크 플롯       |
| **범주형**     | X 여러 개 (범주형)        | 교차분석, 로지스틱 회귀               | 누적 막대그래프, 히트맵      |

# 📌 2. 데이터 분석 방법론 절차

| 단계                         | 핵심 내용                                                                                                  | 키워드 (암기 포인트) |
| -------------------------- | ------------------------------------------------------------------------------------------------------ | ------------ |
| **1. 분석 기획 (Planning)**    | - 비즈니스 이해, 범위·목표·KPI 설정<br>- 프로젝트 정의·계획 수립<br>- 위험요인 사전 관리                                             | 목적·범위·계획·위험  |
| **2. 데이터 준비 (Preparing)**  | - 필요한 데이터 정의 (정형·비정형)<br>- 데이터 스토어 설계 (DB, Hadoop, NoSQL)<br>- 데이터 수집·정합성 점검 (ETL, 품질관리)               | 확보·저장·품질     |
| **3. 데이터 분석 (Analyzing)**  | - 분석용 데이터 준비·가공<br>- 텍스트 분석 (사전, 시각화)<br>- 탐색적 분석 (EDA)<br>- 모델링 (분류, 예측, 군집 등)<br>- 모델 평가·검증 (성능, 튜닝) | EDA·모델링·검증   |
| **4. 시스템 구현 (Developing)** | - 시스템·아키텍처 설계 및 구현<br>- BI 도구, 프로그래밍 적용<br>- 시스템 테스트·운영, 사용자 교육                                        | 설계·구현·테스트    |
| **5. 평가 및 전개 (Deploying)** | - 모델 발전계획 (주기적 평가, 유지보수)<br>- 프로젝트 성과 평가 (정량·정성)<br>- 산출물 자산화 및 보고                                     | 유지·평가·보고     |

# 📌 3. 지도학습과 비지도 학습

| 구분                                | 주요 기법                                                                 | 설명/특징                                                                                                                                                                          |
| --------------------------------- | --------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **지도학습 (Supervised Learning)**    | **회귀(Regression)**                                                    | - 선형회귀 (Linear Regression)<br>- 다항회귀 (Polynomial Regression)<br>- 로지스틱 회귀 (Logistic Regression → 분류)<br>- 리지/라쏘/엘라스틱넷 회귀                                                       |
|                                   | **분류(Classification)**                                                | - 의사결정나무 (Decision Tree)<br>- 랜덤포레스트 (Random Forest)<br>- 서포트 벡터 머신 (SVM)<br>- k-최근접이웃 (k-NN)<br>- 나이브 베이즈 (Naïve Bayes)<br>- 그래디언트 부스팅 (GBM)<br>- XGBoost, LightGBM, CatBoost |
|                                   | **신경망(Neural Networks)**                                              | - 퍼셉트론, 다층퍼셉트론 (MLP)<br>- 합성곱 신경망 (CNN, 이미지)<br>- 순환 신경망 (RNN, LSTM, 시계열/텍스트)                                                                                                  |
|                                   | **기타**                                                                | - 시계열 예측(AR, ARIMA → 때로 지도학습으로 다룸)<br>- 회귀/분류 혼합 모델                                                                                                                            |
| **비지도학습 (Unsupervised Learning)** | **군집화(Clustering)**                                                   | - k-평균 (k-means)<br>- 계층적 군집 (Hierarchical Clustering)<br>- DBSCAN<br>- Gaussian Mixture Model (GMM)                                                                           |
|                                   | **차원 축소(Dimensionality Reduction)**                                   | - 주성분분석 (PCA)<br>- 독립성분분석 (ICA)<br>- t-SNE<br>- UMAP<br>- 비음수 행렬 분해 (NMF)                                                                                                      |
|                                   | **연관 규칙 학습 (Association Rule Learning)**                              | - Apriori<br>- FP-Growth                                                                                                                                                       |
|                                   | **특징 학습/잠재 변수 모델**                                                    | - 오토인코더 (Autoencoder)<br>- GAN(생성적 적대 신경망, 일부 반지도학습에도 분류됨)<br>- Self-Organizing Map (SOM)                                                                                      |
| **반지도학습 (Semi-supervised)**       | - 일부 라벨만 있는 데이터를 활용: Label Propagation, Pseudo-labeling               | 지도+비지도 혼합                                                                                                                                                                      |
| **강화학습 (Reinforcement Learning)** | - Q-learning<br>- SARSA<br>- 정책경사 (Policy Gradient)<br>- Actor-Critic | 보상 기반 학습                                                                                                                                                                       |
