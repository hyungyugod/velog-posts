# 📌 1. 엑셀 출제유형분석1회 - 기타작업
## 1-1. 차트 만들기
- 데이터 종류를 추가하거나 삭제할때는 차트영역(차트의 빈공간)의 바로가기 메뉴에서 데이터 선택 항목을 선택하여 사용한다.
- 데이터 선택에서 가로축 데이터 혹은 범례 항목(계열)을 삭제 할때 가로축 레이블에는 추가와 삭제버튼이 없다 하여 이때 이를 처리하기 위해서는 행열 전환을 통해 위치를 바꿔주어야 한다.
- 계열은 “추가·삭제 가능한 데이터 객체”이고, 가로축 레이블은 “차트 전체가 공유하는 좌표 기준”이기 때문에 추가·삭제 버튼이 존재하지 않는다.
- 아니면 차트 데이터 선택에서 컨트롤을 누르고 필요한 범위만 선택하여 다시 차트를 설정하는 방법도 있다.
- 차트의 종류를 변경할때는 차트영역 바로가기에서 차트 종류 변경 메뉴를 활용한다.
- 어떤 텍스트를 다른 텍스트와 연동하려면 해당 텍스트에 수식으로 참조를 하면된다.
- 데이터의 레이블을 추가하는 것은 차트의 특정 레이블을 선택한 뒤에 바로가기 메뉴에서 데이터 레이블 서식 메뉴에서 사용할 수 있다.
- 어떤 영역의 디자인을 바꾸는 것은 해당 영역에서 바로가기메뉴로 들어갔을때 해당 영역 서식 메뉴에서 모두 편집할 수 있다.

## 1-2. 메크로작업
- 시작할때 셀 포인터를 빈 공간에 두고 시작해야한다.
- 셀 포인터를 빈 공간에 둔다는 건: “이 위치는 데이터가 아니다. 여기서는 구조만 잡을 것이다.” 라고 엑셀에게 명확히 알려주는 신호이기 때문이다.
- 끝낼때도 빈 공간을 눌러주면 좋다.
- 메크로 범위는 생성할때 열려있는 통합문서로 한다.
- 양식 컨트롤에서 단추를 만들때 alt를 누르고 그리고 데이터에 딱 맞게 그릴 수 있다.
- 양식 컨트롤: “셀 값을 바꾸기 위한 간단한 스위치”
- ActiveX: “이벤트 기반 미니 프로그램” -> 외부에서 들어온 컴포넌트 기술
- 조건부 서식 블럭설정을 위에서 아래로 해야하는 이유는 조건부 서식은 “범위마다 계산”이 아니라 “기준 셀 하나를 잡고 나머지를 상대 복제” 하며 누적 계산하기 때문이다.
- 조건부 서식 관련 메크로는 조건부 서식이 ‘덮어쓰기 대상’이 아니라 ‘누적 규칙 집합’이기 때문에 누를때마다 규칙이 생긴다. -> 규칙관리자에서 하나만 남겨두어야 한다.

## 1-3. 프로시저 작업
- 단추와 폼의 각 요소들에 특정 코드를 걸어서 작동을 설계하는 작업이다.
- 성적입력.show를 통해 특정 폼을 연다
- alt+f11로 vba로 들어간다.
- 엑셀에서 연속의 정의는 '중간에 완전히 빈 행이나 빈 열이 없을 것'이다.
- CurrentRegion은 이렇게 동작한다.
- B5에서 시작
- 위로 이동 → 빈 행을 만나기 전까지 확장
- 아래로 이동 → 빈 행을 만나기 전까지 확장
- 왼쪽으로 이동 → 빈 열을 만나기 전까지 확장 (개체와 개체 사이에 빈 열이 하나라도 있으면 다른 표로 인식한다.)
- 오른쪽으로 이동 → 빈 열을 만나기 전까지 확장
- 아래는 값이 있는 행의 개수를 세는 코드이다.
- 엑셀의 테두리는 데이터가 아니다.
- 단추같은 것들은 떠있는 객체이기 때문에 더더욱 데이터가 아니다.

```vba
Range("B5").CurrentRegion.Rows.Count
```

- 폼에서 특정 요소를 클릭하고 f4를 누르면 개체 이름을 알 수 있다.
- 셀에 입력할때는 아래 코드를 활용하면 된다.

```vba
Cells(행,열) = 입력받는 개체 이름
```

- 입력받는 값은 문자열이므로 수식으로서 계산을 하기 위해서는 val 함수를 이용해서 정수로 바꾼다음 계산해야한다.
- 종료 시켜주는 명령은 unload me이다.

- 아래는 등록 버튼의 전체 프로시저이다. 현재 표에서 채워져있는 행 + 시작행 offset으로 행을 지정하고 열은 고정된 값이므로 지정해서 cells에서 데이터를 입력해 넣는다.

```vba
Private Sub cmd등록_Click()

input_rownum = Range("b5").CurrentRegion.Rows.Count + 5

Cells(input_rownum, 2) = txt이름
Cells(input_rownum, 3) = txt자료구조
Cells(input_rownum, 4) = txt운영체제
Cells(input_rownum, 5) = (Val(txt자료구조) + Val(txt운영체제)) / 2


End Sub
```
