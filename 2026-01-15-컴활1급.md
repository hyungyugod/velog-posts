## 1-7. count는 숫자만 세는 함수이다.
> =COUNT(IF((YEAR($E$29:$E$46)=I$28)*(LEFT($C$29:$C$46,1)=$H29),1)) 이건 되고 =COUNT(IF((YEAR($E$29:$E$46)=I$28)*(LEFT($C$29:$C$46,1)=$H29),$B$29:$B$46))이건 안되는 이유는 count는 숫자만 세는 함수인데 $B$29:$B$46가 숫자가 아니라 텍스트이기 때문에 개수가 0개가 나오는 것이다.

## 1-8. select문에서 or사용불가
```
Public Function fn비고(소속, 평가점수)

If 평가점수 >= 80 Then
    Select Case Left(소속, 2)
        Case "대기"
            fn비고 = "1차"
        Case "환경"
            fn비고 = "1차"
        Case "녹지"
            fn비고 = "1차"
        
        Case "자연"
            fn비고 = "2차"
        Case "도심"
            fn비고 = "2차"
        
        Case Else
            fn비고 = "3차"
    End Select
Else
    fn비고 = ""
End If

End Function
```
> select문에서 or이 사용불가하므로 그냥 위처럼 복사, 붙여넣기를 통해 식을 전부 따로 적어줘야 한다.

## 1-10. lookup 배열상수 활용하기
> 원래 배열상수는 참조표가 있어야 한다. 즉 조건표와 결과표를 제공해주고 판단할 값이 조건표에 있으면 결과표의 값을 반환해주는 역할을 하는 것이 lookup이다. 만약 참조표가 없다면 직접 내부에 배열을 {"국외세액";"지방세액"} 처럼 입력해줘야한다.
> 이때 조건표 즉 분류표는 오름차순으로 입력되어야한다.
> 이때 두 배열상수를 ;로 구분하면 모두 ;를 구분, 그리고 ,로 구분했으면 ,로 같이 구분해줘야한다.
