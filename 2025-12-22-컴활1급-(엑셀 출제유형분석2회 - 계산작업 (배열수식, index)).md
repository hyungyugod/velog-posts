# 📌 1. 엑셀 출제유형분석2회 - 계산작업 (배열수식, index)
## 1-1. index, match, 계산함수를 사용하는 문제 유형
- vlookup이 없고 index랑 match만 있으면 열들에 대한 조건을 지정하기가 애매하다. 따라서 찾는 배열의 범위를 절대화를 풀고 옆으로 옮기면서 채우기 핸들을 활용하여 채워준다. 이때 답을 채울 표의 필드와 원본 필드의 순서가 동일해야한다.
- f4번을 여러번 누르면서 특정 행이나 열만 고정하도록 절대화 위치를 바꿀 수 있다.
- 배열수식에서 match를 사용할때는 찾을 배열에서도 조건을 걸어줘야한다. -> 특정 조건에 맞는 열만 남기기
- 배열 수식을 마무리할때는 마법사를 사용하더라도 ctrl + shift + enter를 사용해준다.

```vba
{=INDEX($D$3:$D$18,MATCH(MAX(($B$3:$B$18=$B41)*(L$3:L$18)),($B$3:$B$18=$B41)*(L$3:L$18),0),1)}
```
